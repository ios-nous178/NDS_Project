목업 검증이 "전용 입력 컴포넌트 대신 맨 입력칸을 쓰는 실수"를 이제 차단으로 잡아드려요.

- **날짜를 맨 텍스트칸으로** 받기 → DatePicker 사용
- **주소를 맨 텍스트칸으로** 조립 → AddressPicker 사용
- **금액을 일반 입력칸으로** 받기 / 입력칸 자리에 정적 숫자만 표시 → AmountInput 사용
- **약관 동의를 맨 체크박스로** 조립 → 체크박스 그룹 사용
- **어드민 사이드바 로고를 텍스트/수동 이미지로** 박기 → 로고 컴포넌트 사용

모두 잘못된 입력·일관성 깨짐을 부르는 실수라 시안 단계에서 막아드립니다. 이메일·URL 주소, 대시보드 통계 숫자, 날짜 선택 컴포넌트 사용 등 정상 케이스는 그대로 통과해요.
