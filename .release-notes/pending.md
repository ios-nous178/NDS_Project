🚨 *업데이트 이후 Claude Desktop 종료 후 다시 실행* 🚨

새 MCP 가이드/카탈로그가 적용되려면 Claude Desktop 을 완전히 종료(⌘Q)했다가 다시 켜 주세요. 그냥 창만 닫으면 옛 버전이 그대로 떠 있어요.



*이번 업데이트 요약*

· DS 패키지와 MCPB 버전을 *0.0.3* 으로 갱신.
· QuickMenu, ValidationChip, Badge pill, Avatar shape, Toast, Tooltip 등 최근 컴포넌트 정합 작업 반영.
· 캐시워크 for Business 어드민 폼·모달·인증·필터·페이지네이션 규칙과 검수 가드 강화.

*주요 변경사항*

*1. 새 컴포넌트와 폼 검증 패턴 추가*
· PC 우측 고정 *QuickMenu* 추가 — 자주 쓰는 전역 액션과 TOP 이동을 한 자리에서 제공.
· *ValidationChip* 추가 — 비밀번호 조건처럼 입력 요건 충족 여부를 실시간으로 보여주는 작은 상태 표시 표준화.
· 폼 검증 패턴 가이드 추가 — 실시간 검증, 인라인 버튼 입력, 비밀번호 보기 토글의 조합 기준 정리.

*2. Avatar·Asset 크기 체계 정합*
· Avatar 에 원형/라운드/사각 shape 선택 추가.
· Avatar 와 Asset 의 크기 스케일 정렬 — md 48, lg 64, xl 96 기준으로 통일.
· 사람 프로필은 Avatar, 일반 이미지·아이콘·썸네일은 Asset 으로 역할 구분 강화.

*3. Toast·Tooltip 디자인가이드 정렬*
· Toast 를 단일 다크 메시지로 정리 — PC 상단 pill, 모바일 하단 rounded 형태로 표준화.
· Toast 는 기본 1개만 노출되도록 정리 — 오래 쌓이는 알림으로 화면이 복잡해지는 문제 완화.
· Tooltip 배경, 패딩, 꼬리, z-index 를 Figma 규격에 맞춰 조정.

*4. 캐시워크 for Business 어드민 CTA 규칙 안정화*
· 모달·팝업·확인 말풍선의 확정 버튼을 검정 CTA 기준으로 통일.
· 노랑 버튼이 모달 확인 액션에 다시 섞이는 문제를 검수 단계에서 차단.
· 모달 단일 버튼은 128px 우측 정렬, Confirm + Slot 구조는 안내 영역과 입력 영역을 분리하도록 정리.

*5. 본인인증과 약관동의 화면 품질 개선*
· 인증번호 입력에서 타이머가 placeholder나 입력값 위로 겹치던 문제 수정.
· 인증번호 줄에 버튼 없이 타이머만 둘 수 있게 정리 — 별도 재전송 버튼이 있는 캐포비 플로우 지원.
· 약관동의의 [필수] 표시는 자동으로 강조 — 필수 항목이 회색으로 남는 반복 누락 방지.

*6. 입력 필드와 드롭다운 시각 정합*
· 모든 텍스트 입력의 placeholder/helper 색을 한 기준으로 통일.
· 돈 입력 중간 자리 수정 시 커서가 끝으로 튀는 문제 수정.
· MultiSelect 패널 검색창, 전체선택 행, 옵션 간격, 푸터 버튼, 패널 폭을 캐포비 어드민 Figma 기준에 맞춰 정리.

*7. 페이지·리스트 운영 규칙 보강*
· Container/Section 레이아웃 가이드 추가 — PC/태블릿/모바일 폭과 섹션 간격 기준 정리.
· Pagination 과 PageSizeSelect 의 URL 동작, 노출 개수 변경, 8페이지 이상 생략 표시, 우하단 배치 규칙 정리.
· 캐포비 페이지네이션의 radius, 활성 글자 굵기, disabled 상태를 디자인가이드에 맞춰 조정.

*8. MCP 가이드와 검수 가드 강화*
· 캐포비는 Toast 대신 Snackbar 만 쓰도록 검수 룰 추가.
· 탭과 필터의 역할 구분을 Tabs/FilterBar 컴포넌트 가이드까지 확대.
· 브랜드별 금지 버튼색, 알림 방식, 모달 정책을 브랜드 프로필 기준으로 읽도록 정리 — 새 브랜드 정책 확장 준비.

*기대효과*

· 어드민 폼·인증·필터 화면의 반복 회귀가 줄어듭니다.
· 브랜드별 버튼색과 알림 방식이 제작 단계에서 더 일관되게 걸러집니다.
· 외부 프로젝트에서 MCP 가이드를 새로 받아 쓰면 최신 컴포넌트 선택 기준과 검수 규칙을 바로 적용할 수 있습니다.
