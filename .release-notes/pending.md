*브랜드 헤더 / 푸터를 한 줄로*

각 브랜드의 헤더·푸터·바텀네비를 컴포넌트 한 줄만 부르면 그대로 박힙니다. PC ↔ 모바일, 라이트 ↔ 다크, 5탭/3탭 같은 분기는 DS 가 알아서 — *로고 · 메뉴 항목 · 회사 정보* 같은 내용만 넘기면 끝.

⚠️ 기존에 옛 컴포넌트 이름이나 CSS 변수로 직접 override 하던 곳은 새 이름으로 한 번 교체 필요해요.

*4번째 브랜드 "캐포비(Cashpobi)" 합류*

캐시워크 for Business admin 디자인이 정식 브랜드로 들어왔어요. *노란색 + 검정 텍스트* 시그니처, 폼·모달·버튼·헤더 전부 캐포비 가이드 그대로 호출 가능. *웹 전용* 이라 앱 헤더/바텀네비는 없어요.

*신규 컴포넌트 5종 — 바로 갖다 쓰세요*

• *PopularPosts* — "이번 주 가장 많이 본 글" 류 랭킹 리스트
• *FloatingCtaBanner* — 페이지 하단에 떠 있는 고정 CTA ("지금 상담 신청하기")
• *ImageUpload* — 프로필/증빙 사진 업로드. 미리보기 + 업로드 + X 제거 + 에러 상태까지 한 컴포넌트
• *ActionChip* — 어드민 폼의 작은 인라인 액션 칩
• *ProductCard* — 할인율·통화·취소선 가격 슬롯 분리

*어드민 폼이 한 묶음으로*

캐포비 어드민 폼 표준이 그대로 들어왔어요. *좌측 라벨 + 컴팩트 입력(40px) + 한 줄에 여러 입력 묶기* 까지 호출 한 번으로 해결. 라벨 옆에 description 길어지면 위로 자동 줄바꿈.

*캐포비 어드민 Modal 4 패턴*

가장 많이 쓰는 모달 4가지 — *단일 확인 / 취소+확인 / 닫기 X / 안내+커스텀 슬롯* — 480 폭 표준으로 스토리북에 그대로. 캐포비 화면에선 따로 손볼 거 없이 그대로 호출.

*Button 다듬기*

• 캐포비 같은 노란 브랜드에서 *파랑 부조화 없이 자연스러운 톤* 으로 정리
• *Pill 모양* 옵션 추가 (둥근 캡슐형 버튼)
• 비활성 색 / hover 시 색 안 보이던 버그 / 아이콘 색 오버라이드 모두 정리

*Input 타이핑이 브랜드 톤대로*

타이핑할 때 보더 색이 *브랜드 가이드 톤* 으로 바뀝니다. 캐포비는 검정 outline, 다른 브랜드는 기존 그대로.

*브랜드 화면이 Figma 디자인 그대로*

• *Geniet PC 새 헤더* — 검색 헤더 + 메뉴 헤더 2단 구조, 인기검색어 NEW 칩, 캐시리뷰/친구초대 CTA pill
• *Geniet 모바일* — 로고 + 포인트 잔액 칩 + 햄버거 검색 2단
• *NudgeEAP GNB 6탭* — 상담하기 · 심리검사 · 심리치료 · 주간레터 · 소식 · 마이페이지
• *NudgeEAP 바텀네비 5탭* — 홈 · 챌린지 · 상담 · 멘탈케어 · 내 공간 (3탭에서 정정)
• *NudgeEAP 웹 푸터* — 약관 + 앱다운로드 + 회사정보 + ISO 인증 + DAIN 로고
• *NudgeEAP 로고 6종 PNG* — Symbol · KO · KO+EN · EN · EN Dark · DAIN

*AI 목업이 똑똑해졌어요 — MCP 업데이트*

Claude/Cursor 한테 "이 화면 DS 로 짜줘" 시킬 때 *목업 품질이 한 단계 올라갔어요*.

• *시각 레퍼런스 없이 시작 못 하게* — Figma 노드나 스크린샷 첨부 없이 목업 만들기 시작하면 먼저 확인 받아요. 추측으로 만들다가 다 갈아엎는 사고 차단.
• *리포트가 항상 페어로* — "DS@0.1.12 · 23개 컴포넌트 (87%)" 식으로 *버전 + DS 사용 비율* 이 한 줄에 같이. 어느 버전 기준인지, DS 위에서 얼마나 만들어졌는지 한눈에.
• *헤더/푸터 손수 조립 차단* — `<div>` 로 헤더 다시 짜는 패턴을 가이드가 막고 브랜드 컴포넌트 한 줄로 안내.
• *React 없이도 OK* — 순수 HTML 한 파일로 목업 떨어집니다. 외주/디자이너 핸드오프 편해요.
• *Geniet 레퍼런스 이미지 내장* — Geniet 웹 홈/리뷰 모달/앱 리뷰 상세를 MCP 가 직접 들고 있어서, "Geniet 톤으로" 한 마디로 톤이 잡혀요.

*Geniet 헤더 미세 조정*

• *캐시리뷰 / 친구초대 pill* — 토스 블루 톤 (#1677ff · #0093f1) 으로 정렬, 보더·배경 디자인 스펙 그대로
• *2열 위/아래 라인* — 컨텐츠 max-width 안쪽이 아니라 *화면 가로폭 전체* 로 그어집니다 (rail 분리)

*브랜드 로고가 어디서든 안 깨져요*

Geniet/Trost/NudgeEAP/Cashpobi 헤더·푸터 로고가 *DS 안에 인라인 내장* 됐어요. 호스트 앱이 `public/brand-logos/...` 같은 자산 폴더를 따로 안 깔아도, npm 으로 받은 그 자리에서 바로 렌더됩니다. 자체 자산을 쓰고 싶으면 `logo` prop 으로 그대로 override.

*기타*

• *DatePicker* — 캘린더 아이콘 브랜드별 자동 교체, 캐포비 `YYYY-MM-DD` ISO 포맷
• *StarRating* — 사이즈를 sm/md/lg/xl 또는 px 숫자(8~48) 자유롭게
