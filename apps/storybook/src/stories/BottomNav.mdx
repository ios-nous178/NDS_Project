# BottomNav (하단 탭바)

모바일 앱 하단 내비게이션 탭바 모음. 푸터(Footer)와 분리된 별도 내비게이션 컴포넌트입니다.

각 데모는 탭을 클릭하면 해당 탭이 활성화됩니다.
