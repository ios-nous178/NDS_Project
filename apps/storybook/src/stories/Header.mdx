# Header

브랜드별 상단 헤더 모음. 모든 브랜드가 **Desktop / Mobile / Webview** 축으로 구성됩니다.
브랜드 색·로고는 토큰(`--semantic-*`)과 `@nudge-design/assets` 가 자동 적용되며, 콘텐츠(메뉴·로고·액션)만 props 로 주입합니다.

| 브랜드 | Desktop | Mobile | Webview | 비고 |
| --- | --- | --- | --- | --- |
| **NudgeEAP** | ✓ | ✓ | ✓ | 데스크톱 = 풀 WebHeader |
| **Geniet** | ✓ (Search/Menu 2단) | ✓ | ✓ | — |
| **Trost** | ✓ (배너+유틸+탭 3단) | ✓ | ✓ (단단 · 홈 2단) | — |
| **CashwalkBiz** | ✓ | ✓ | — | 웹 전용 |
| **Runmile** | ✓ | ✓ (title-icon · logo · menu-title) | — | 앱 모바일 3 variant |

> 기본(base) `<Header>` 컴포넌트의 슬롯/variant 데모는 `Components/Header/Base/*` 를 참고하세요.
