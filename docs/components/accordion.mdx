# Accordion

접이식 콘텐츠 패널입니다. `type="single"`은 한 번에 하나만, `type="multiple"`은 여러 개를 동시에 열 수 있으며 controlled / uncontrolled 두 패턴을 모두 지원합니다. Compound API(`Accordion.Item / Trigger / Content`)와 Flat 컴포넌트(`AccordionItem / AccordionTrigger / AccordionContent`)를 모두 노출합니다.

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- FAQ, 약관/이용 정책 등 **길지만 평소엔 접혀 있어야 하는 콘텐츠**를 묶을 때
- 같은 형식의 항목이 반복되어 **세로 스캐닝을 줄여야 할 때**
- 한 화면에 정보가 많아 **선택적으로 펼쳐 보게 하고 싶을 때**

### 이럴 때 사용하지 마세요

- 약관 묶음에 동의 체크박스가 필요한 경우 → `CheckboxGroup` data mode로 조립하고, 자체 펼침 패턴은 중첩하지 마세요
- 콘텐츠 영역을 같은 페이지에서 전환 → [Tabs](/components/tabs)를 사용하세요
- 하나의 긴 본문을 줄여 보여주는 용도 → `ExpandableText`를 사용하세요

## 기본 사용

### Compound API (권장)

```tsx
<Accordion type="single" defaultValue="terms">
  <Accordion.Item value="terms">
    <Accordion.Trigger>이용 약관</Accordion.Trigger>
    <Accordion.Content>약관 본문</Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value="privacy">
    <Accordion.Trigger>개인정보 처리방침</Accordion.Trigger>
    <Accordion.Content>약관 본문</Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### 여러 개 동시 펼침

```tsx
<Accordion type="multiple" defaultValue={["a", "b"]}>
  <Accordion.Item value="a">
    <Accordion.Trigger>섹션 A</Accordion.Trigger>
    <Accordion.Content>내용 A</Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value="b">
    <Accordion.Trigger>섹션 B</Accordion.Trigger>
    <Accordion.Content>내용 B</Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### Controlled

```tsx
const [open, setOpen] = useState<string>("terms");

<Accordion type="single" value={open} onValueChange={(v) => setOpen(v as string)}>
  {/* ... */}
</Accordion>;
```

### Accordion.Item

| Prop       | 타입        | 설명                              |
| ---------- | ----------- | --------------------------------- |
| `value`    | `string`    | 아이템 고유 값 (Accordion 식별자) |
| `children` | `ReactNode` | `Trigger` + `Content`             |

### Accordion.Trigger

`<button>` HTML 속성을 모두 지원합니다. 클릭 시 자동으로 해당 아이템을 토글합니다.

### Accordion.Content

`<div>` HTML 속성을 지원합니다. 닫힌 상태에서는 DOM에 렌더되지 않습니다(언마운트).

## data-slot 구조

```
div [data-slot="root"]
└── div [data-slot="item"] [data-state="open" | "closed"]
    ├── button [data-slot="trigger"] [data-state]  (aria-expanded, aria-controls)
    │   └── svg.nds-accordion__chevron
    └── div [data-slot="content"] [data-state="open"]  (role="region", 닫힘 시 미렌더)
```

## CSS 변수

이 컴포넌트는 별도 CSS 변수를 노출하지 않습니다. 색상 / 간격은 `@nudge-design/tokens`의 시멘틱 토큰(`cv.border.light`, `cv.bg.coolGray`, `spacing.*`)에 직접 바인딩됩니다.

---

## 접근성

- `<button>` 트리거에 `aria-expanded`, `aria-controls`를 자동 부여
- 콘텐츠 영역은 `role="region"` + `aria-labelledby`로 트리거와 연결
- 닫힌 패널은 DOM에서 제거되어 스크린 리더가 닫힌 콘텐츠를 읽지 않음
- Trigger는 네이티브 `<button>`이라 Enter / Space로 토글, Tab 포커스 자연 지원
