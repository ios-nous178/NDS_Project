# Accordion

접이식 콘텐츠 패널. `type="single"`은 한 번에 하나만, `type="multiple"`은 여러 개를 동시에 열 수 있습니다.

## 사용법

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@nudge-eap/react";

<Accordion type="single" defaultValue="terms">
  <AccordionItem value="terms">
    <AccordionTrigger>이용 약관</AccordionTrigger>
    <AccordionContent>약관 본문</AccordionContent>
  </AccordionItem>
  <AccordionItem value="privacy">
    <AccordionTrigger>개인정보 처리방침</AccordionTrigger>
    <AccordionContent>약관 본문</AccordionContent>
  </AccordionItem>
</Accordion>
```


> ConsentChecklist(약관 묶음)는 자체 펼침을 가짐 — 중첩 금지.
