# ActivityTimeline

상담·검사 이력을 시간 순으로 표시하는 세로 타임라인입니다. 각 항목에 상태(`completed`/`ongoing`/`warning`/`error`/`default`) 색을 매핑할 수 있습니다.

## 사용법

```tsx
import { ActivityTimeline } from "@nudge-design/react";

<ActivityTimeline
  items={[
    {
      key: "1",
      date: "2026.04.18",
      title: "PHQ-9 우울 검사",
      description: "총점 7점 (정상)",
      status: "completed",
      statusLabel: "완료",
    },
    {
      key: "2",
      date: "2026.05.04 14:00",
      title: "다음 상담",
      description: "박민지 상담사 · 50분",
      status: "ongoing",
      statusLabel: "진행 중",
    },
    {
      key: "3",
      date: "2026.05.18",
      title: "PHQ-9 재검사 예정",
      status: "default",
    },
  ]}
/>
```

## Props

| Prop    | 타입             | 기본값 | 설명     |
| ------- | ---------------- | ------ | -------- |
| `items` | `TimelineItem[]` | —      | 항목들   |

`TimelineItem = { key, date, title, description?, status?, statusLabel?, icon? }`

`TimelineStatus = "default" | "completed" | "ongoing" | "warning" | "error"`

## 단계별 색상

| Status      | Dot 색상       | 배지 색상       |
| ----------- | -------------- | --------------- |
| `default`   | coolGray       | coolGrayLighter |
| `completed` | success.main   | success.bg      |
| `ongoing`   | primary + ring | primary.bgLighter |
| `warning`   | caution.main   | caution.bg      |
| `error`     | error.main     | error.bg        |

## 가이드

- 마지막 항목의 line은 자동 비표시 (`:last-child`).
- `status="ongoing"`은 ring 효과로 시각 강조 — 한 화면에 1개만 사용.
- `statusLabel` 없으면 우측 배지 미표시. 둘 다 필요.
- 아이템 20+ 넘으면 페이지네이션 또는 가상화 권장.
