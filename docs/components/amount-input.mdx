# AmountInput

송금/결제/후원/챌린지 목표 같은 큰 숫자 입력을 위한 컴포넌트입니다. 자동 천 단위 콤마와 빠른 입력 버튼(presets)을 지원합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 송금/결제/충전 금액
- 후원/응원 금액
- 챌린지 목표 (걸음수, 시간 등 숫자)

### 이럴 때 사용하지 마세요
- 작은 정수 조절 (1~99) → `NumberStepper`
- 일반 텍스트 → `Input`

### 자주 하는 실수
- `value`는 `number | null`. 빈 입력은 `null`로 처리 (0과 다름)
- `presets`의 `set: true`는 값 설정, 미지정/false는 누적
- max/min 자동 클램프 — 외부 검증 별도 X

## 사용법

```tsx
import { AmountInput } from "@nudge-design/react";

<AmountInput
  label="송금 금액"
  value={amount}
  onValueChange={setAmount}
  presets={[
    { label: "+1만", amount: 10000 },
    { label: "+5만", amount: 50000 },
    { label: "전액", amount: balance, set: true },
  ]}
  max={balance}
/>

// 달러
<AmountInput value={v} onValueChange={setV} prefix="$" unit="" />
```

## Props

| Prop            | 타입                              | 기본값  | 설명                          |
| --------------- | --------------------------------- | ------- | ----------------------------- |
| `value`         | `number \| null`                  | —       | 숫자 (null = 빈 입력)         |
| `onValueChange` | `(v: number \| null) => void`     | —       | 변경 콜백                     |
| `prefix`        | `string`                          | —       | 통화 prefix (예: `$`)         |
| `unit`          | `string`                          | `"원"`  | 통화 suffix                   |
| `label`         | `ReactNode`                       | —       | 라벨                          |
| `helperText`    | `ReactNode`                       | —       | 헬퍼 / 에러                   |
| `error`         | `boolean`                         | `false` | 에러                          |
| `fullWidth`     | `boolean`                         | `false` | 가로 가득                     |
| `presets`       | `AmountPreset[]`                  | —       | 빠른 입력 버튼                |
| `max`           | `number`                          | —       | 최댓값 (자동 클램프)          |
| `min`           | `number`                          | —       | 최솟값                        |

### `AmountPreset`

| Prop     | 타입      | 설명                          |
| -------- | --------- | ----------------------------- |
| `label`  | `string`  | 버튼 라벨                     |
| `amount` | `number`  | 더할 값 (또는 set 모드 값)    |
| `set`    | `boolean` | true면 값 설정, 미지정 누적   |
