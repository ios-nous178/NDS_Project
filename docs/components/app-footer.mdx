# Footer

`Footer` 는 프로젝트-무관 base compound (`.Info` / `.TabBar` / `.Web`) 입니다. 공개 react 패키지의
프로젝트 chrome 정리로 **프로젝트별 푸터 컴포넌트(`NudgeEAPFooter` / `TrostFooter` / `GenietFooter` /
`CashwalkBizFooter` …)는 제거**됐습니다.

- **목업(html)**: 프로젝트 완성형 푸터는 `<nds-project-footer project="..." surface="web|app">` 한 줄.
- **공개 primitive(react)**: 커스텀 푸터는 base `Footer` compound 로 조립 + 프로젝트 토큰으로 색 분기.

> **Breaking 이력**: 옛 `AppFooter` / `{Project}AppFooter` / `{Project}WebFooter` / `TrostDesktopFooter` / `{Project}Footer` named export 는 제거됐습니다. `nds-app-footer` className 도 `nds-footer` 로 통일.

## 프로젝트 완성형 푸터 (목업 전용 `nds-project-footer`)

| Project       | 지원 surface          | 핵심 slot                                                          |
| ----------- | --------------------- | ----------------------------------------------------------------- |
| NudgeEAP    | `web` / `app`         | web: 약관·앱다운로드·ISO·DAIN·poweredBy / app: 회사정보·약관·로고  |
| Trost       | `app` (dark)          | dark PC/앱 푸터 · 약관·SNS·앱다운로드                              |
| CashwalkBiz | `web` (app 없음)      | light 톤                                                          |
| Geniet      | `app` (web 없음)      | 커머스 고지 · 통신판매중개자 안내                                  |
| Runmile     | `app`                 | light 톤 · gray700 워드마크                                       |

```html
<nds-project-footer project="nudge-eap" surface="web"></nds-project-footer>
<nds-project-footer project="trost" surface="app"></nds-project-footer>
<nds-project-footer project="cashwalk-biz"></nds-project-footer>
```

## Base Footer compound

프로젝트 wrapper 가 부족하거나 커스텀 컴포지션이 필요한 경우 base 를 직접 사용합니다.

## Footer.TabBar — 하단 탭 네비게이션

```tsx
import { Footer } from "@nudge-design/react";

// NudgeEAP WebView (3탭)
const nudgeTabs = [
  { key: "home", label: "홈", icon: <HomeIcon />, activeIcon: <HomeFilledIcon />, href: "/" },
  {
    key: "shop",
    label: "심리샵",
    icon: <ShopIcon />,
    activeIcon: <ShopFilledIcon />,
    href: "/shop",
  },
  { key: "my", label: "마이", icon: <MyIcon />, activeIcon: <MyFilledIcon />, href: "/my" },
];

<Footer.TabBar
  tabs={nudgeTabs}
  activeTab="home"
  onTabClick={(tab, e) => {
    e.preventDefault();
    router.push(tab.href);
  }}
/>;
```

### Next.js Link 연동

```tsx
import Link from "next/link";

<Footer.TabBar
  tabs={tabs}
  activeTab={currentTab}
  renderTab={(tab, isActive, children) => (
    <Link
      key={tab.key}
      href={tab.href}
      className="nds-footer__nav-item"
      data-active={isActive || undefined}
    >
      {children}
    </Link>
  )}
/>;
```

## AI 컨텍스트

- **Info**: 홈페이지 하단 공통. 약관 링크 + 회사 정보(주소/사업자번호/연락처) 배치. NudgeEAP/지니어트는 밝은 배경(`#FAFAFA`), 트로스트는 다크 배경(`#333`). 지니어트는 통신판매중개자 고지와 고객문의 시간 안내가 추가됨.
- **TabBar**: 앱 하단 고정 네비. 아이콘+라벨 구조, active 상태에서 filled 아이콘 전환. 지니어트는 `border-top` 대신 상단 그림자 사용, 웹뷰(`WEBVIEW_CASHWALK`/`WEBVIEW_GENIET`)에서 숨김.
- **Web**: PC 풍부 푸터의 공통 골격 (container + max-width + divider + row helper). 프로젝트 wrapper 가 행 구성을 책임짐.
- 전 프로젝트 공통: 탭바 높이 56px, 라벨 11px, 아이콘 24px.
