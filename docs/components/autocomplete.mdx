# Autocomplete

입력 도중 추천 옵션을 드롭다운으로 보여주는 컴포넌트입니다.
키보드 네비게이션(Up/Down/Enter/Escape), 매칭 텍스트 하이라이트, 로딩/빈 상태를 지원합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 정해진 사전(약, 시설, 카테고리)에서 빠르게 찾을 때
- 사용자가 정확한 이름을 모를 가능성이 높을 때

### 이럴 때 사용하지 마세요
- 자유 검색만 필요하고 추천이 없을 때 → `SearchInput`
- 옵션 수가 작고 모두 보여줄 수 있을 때 → `Select`

### 자주 하는 실수
- `options`는 외부에서 필터링해 전달해야 함 — 컴포넌트가 자동 필터링하지 않음 (서버 검색 케이스 고려)
- `onSelect` 후에는 자동으로 `onValueChange(option.label)`이 호출됨

## 사용법

```tsx
import { Autocomplete } from "@nudge-design/react";

const [query, setQuery] = useState("");
const filtered = useMemo(
  () => ALL.filter((o) => o.label.includes(query)),
  [query]
);

<Autocomplete
  label="복용 약"
  placeholder="약 이름 검색"
  value={query}
  onValueChange={setQuery}
  options={filtered}
  onSelect={(opt) => setSelectedId(opt.value)}
/>

// 서버 비동기 검색
<Autocomplete
  value={q}
  onValueChange={(v) => { setQ(v); fetchSuggestions(v); }}
  options={results}
  loading={isFetching}
  minQueryLength={2}
/>
```

| 키          | 동작                       |

| `Enter`     | 활성 옵션 선택             |
| `Escape`    | 드롭다운 닫기              |
