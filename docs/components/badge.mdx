# Badge

상태, 카테고리, 또는 라벨을 표시하는 인라인 컴포넌트입니다. 6가지 색상 변형과 2가지 사이즈를 제공합니다.

```tsx
import { Badge } from "@nudge-eap/react";
```

## Variant (색상)

6가지 의미 기반 색상을 제공합니다.

<ComponentPreview title="Variants">
  <Badge variant="primary">프라이머리</Badge>
  <Badge variant="secondary">세컨더리</Badge>
  <Badge variant="success">성공</Badge>
  <Badge variant="caution">주의</Badge>
  <Badge variant="error">에러</Badge>
  <Badge variant="neutral">기본</Badge>
</ComponentPreview>

```tsx
<Badge variant="primary">프라이머리</Badge>
<Badge variant="secondary">세컨더리</Badge>
<Badge variant="success">성공</Badge>
<Badge variant="caution">주의</Badge>
<Badge variant="error">에러</Badge>
<Badge variant="neutral">기본</Badge>
```

| Variant | 배경색 | 텍스트 색상 | 사용 시점 |
|---|---|---|---|
| `primary` | Blue 100 | Blue 600 | 기본 강조, 카테고리 |
| `secondary` | Magenta 100 | Magenta 600 | 보조 구분 |
| `success` | Green 50 | Green 500 | 성공, 완료 상태 |
| `caution` | Yellow 50 | Yellow 600 | 경고, 주의 |
| `error` | Red 50 | Red 500 | 에러, 실패 |
| `neutral` | Neutral 100 | Neutral 700 | 일반 정보 (기본값) |

## Props

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `variant` | `BadgeVariant` | `"neutral"` | 색상 변형 |
| `size` | `"sm"` \| `"md"` | `"md"` | 사이즈 |
| `labelClassName` | `string` | — | 라벨 span에 추가할 className |
| `slotProps` | `BadgeSlotProps` | — | 내부 슬롯별 props 전달 |
| `className` | `string` | — | 루트 요소 className |
| `style` | `CSSProperties` | — | 루트 요소 인라인 스타일 |

**BadgeVariant**: `"primary"` | `"secondary"` | `"success"` | `"caution"` | `"error"` | `"neutral"`

`<span>` HTML 속성을 모두 지원합니다.

---

## SlotProps

```tsx
interface BadgeSlotProps {
```
