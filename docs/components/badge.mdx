# Badge

> ✅ **Figma 가이드 정합 완료** · 검증일 2026-05-13 · [Figma 171:10856](https://www.figma.com/design/MqR7O3uvBvH5tVngwzbqGH/?node-id=171-10856)

상태, 카테고리, 또는 라벨을 표시하는 인라인 컴포넌트입니다. 3가지 스타일(`fill` / `ghost` / `line`) × 6가지 색상 × 3가지 크기 매트릭스를 제공합니다.

```tsx
import { Badge } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 상태(성공, 에러, 주의)나 카테고리를 **읽기 전용 라벨**로 표시할 때
- 목록 항목에 짧은 메타 정보를 덧붙일 때 (신규, 완료, 대기 중 등)
- 숫자 카운트를 강조할 때 (알림 뱃지 등)

### 이럴 때 사용하지 마세요

- 사용자가 클릭하여 선택/해제하거나 삭제할 수 있는 태그 → [Chip](/components/chip)을 사용하세요
- 액션을 트리거하는 용도 → [Button](/components/button)을 사용하세요
- 긴 설명 텍스트를 담을 때 → 일반 텍스트나 인라인 메시지를 사용하세요

### 자주 하는 실수

- Badge에 `onClick`을 붙여 인터랙티브하게 사용 — 읽기 전용이어야 합니다. 인터랙션이 필요하면 Chip을 사용하세요
- 동일 화면에서 3가지 이상의 Color를 혼용 — 의미 체계가 흐려집니다
- 모든 카드에 "추천", "안내", "중요" 같은 Badge를 반복 — 실제 상태나 분류가 아니면 사용하지 마세요

## Variant — Style

Figma `Style` 축. 시각적 우선순위는 `fill` → `ghost` → `line` 순으로 낮아집니다.

<ComponentPreview title="Style">
  <Badge variant="fill" color="project">Fill</Badge>
  <Badge variant="ghost" color="project">Ghost</Badge>
  <Badge variant="line" color="project">Line</Badge>
</ComponentPreview>

| Variant | 사용 비중 | 설명             |
| ------- | --------- | ---------------- |
| `fill`  | 높음      | 강조             |
| `ghost` | 중간      | 보조 (옅은 톤)   |
| `line`  | 낮음      | 최소 (테두리만) |

    <Badge variant="fill" color="info">정보</Badge>
  </div>

| Color     | 의미                  | 사용 예              |
| --------- | --------------------- | -------------------- |
| `project`   | 주요 액션, 프로젝트 강조 | 카테고리, 신규       |
| `neutral` | 일반 카테고리         | 기본 레이블          |
| `success` | 완료, 성공, 진행 중   | 참여중, 완료         |
| `error`   | 오류, 실패, 위험      | 취소됨, 만료         |
| `caution` | 주의, 경고, 선착순    | 선착순, 주의         |
| `info`    | 정보, 안내, 공지      | 공지, 안내           |

## DO / Don't

- ✅ 상태 의미에 맞는 Color 사용 (Success=완료, Error=오류)
- ✅ Fill → Ghost → Line 순 우선순위 적용
- ✅ Semantic Color Token으로 바인딩
- ❌ 동일 화면에서 3가지 이상 Color 혼용
- ❌ Badge와 Chip의 혼용 사용 (역할 구분 필요)
- ❌ 고정 색상(하드코딩) 사용
