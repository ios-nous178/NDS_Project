# Banner

프로모션, 앱 다운로드 유도, 공지사항 등을 표시하는 배너 컴포넌트입니다.

## 사용법

```tsx
import { Banner } from "@nudge-design/react";

// 기본 배너
<Banner
  title="새로운 기능이 출시되었습니다"
  description="지금 바로 확인해 보세요."
  actionLabel="자세히 보기"
  onAction={() => router.push("/new")}
/>

// 이미지 포함
<Banner
  title="앱에서 더 많은 기능을"
  actionLabel="앱 다운로드"
  actionHref="https://..."
  imageSrc="/images/app.webp"
  imageWidth={106}
  imageHeight={130}
/>

// 전체 이미지 배너 (모바일 다운로드)
<Banner
  variant="image"
  fullImageSrc="/images/download-banner.webp"
  href="https://..."
/>

// 닫기 가능
<Banner title="공지사항" closable onClose={() => setVisible(false)} />
```

## Props

| Prop           | 타입                                | 기본값     | 설명                      |
| -------------- | ----------------------------------- | ---------- | ------------------------- |
| `variant`      | `"filled" \| "outlined" \| "image"` | `"filled"` | 스타일 변형               |
| `title`        | `ReactNode`                         | —          | 제목                      |
| `description`  | `ReactNode`                         | —          | 설명                      |
| `actionLabel`  | `string`                            | —          | 액션 버튼 텍스트          |
| `onAction`     | `() => void`                        | —          | 액션 클릭 콜백            |
| `actionHref`   | `string`                            | —          | 액션 링크 (onAction 대신) |
| `imageSrc`     | `string`                            | —          | 우측 이미지               |
| `href`         | `string`                            | —          | 전체 배너 클릭 링크       |
| `closable`     | `boolean`                           | `false`    | 닫기 버튼 표시            |
| `onClose`      | `() => void`                        | —          | 닫기 콜백                 |
| `fullImageSrc` | `string`                            | —          | image variant 전체 이미지 |

| 변수                           | 기본값         | 설명      |
