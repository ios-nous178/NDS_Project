# Brand Chrome

브랜드 화면의 반복 chrome(헤더·푸터·바텀네비)입니다. 구조·아이콘 매핑·브랜드 색은 DS 가
`nds-brand-chrome` 한 곳에서 관리하고, 화면에서는 **태그 한 줄**로 꺼내 씁니다.

:::info Breaking — 공개 react 브랜드 chrome 제거
공개 react 패키지에서 브랜드별 chrome 컴포넌트(`GenietAppBar` · `TrostFooter` ·
`NudgeEAPWebHeader` · `RunmileBottomNav` · `CashwalkBizWebHeader` …)는 **제거됐습니다**.
브랜드 chrome 은 이제 목업 전용 `@nudge-design/html` 의 `nds-brand-*` 웹컴포넌트로만 제공됩니다.
공개 primitive 가 필요하면 `Header` · `BottomNav` · [`Footer`](/components/app-footer) +
브랜드 토큰을 쓰세요. (로고는 `NudgeEAPLogo` / `BrandLogo` 유지.)
:::

## HTML 목업 — `nds-brand-*`

`@nudge-design/html` 이 5개 브랜드의 완성형 header/footer/bottom-nav 를 제공합니다. 로고 asset 은
기본적으로 `/assets` 아래 brand-first taxonomy 에서 찾고, 공유용 HTML처럼 경로가 다르면
`asset-base-url` 만 바꿉니다.

```html
<nds-brand-header brand="nudge-eap" surface="web"></nds-brand-header>
<!-- ...page content... -->
<nds-brand-bottom-nav brand="nudge-eap" active-key="home"></nds-brand-bottom-nav>
<nds-brand-footer brand="nudge-eap"></nds-brand-footer>

<nds-trost-header></nds-trost-header>
<nds-trost-bottom-nav active-key="counsel"></nds-trost-bottom-nav>
<nds-trost-footer></nds-trost-footer>

<nds-brand-header
  brand="geniet"
  asset-base-url="https://cdn.example.com/nds/assets"
></nds-brand-header>
```

## 브랜드 × 표면 커버리지

| Brand       | Header (`surface`)         | Footer (`surface`) | BottomNav         |
| ----------- | -------------------------- | ------------------ | ----------------- |
| Geniet      | web / mobile / webview     | app                | ✅ 5탭            |
| Trost       | web / mobile / webview     | app (dark)         | ✅ 5탭            |
| NudgeEAP    | web / mobile / webview     | web / app          | ✅ 5탭            |
| Runmile     | web / mobile / webview     | app                | ✅ 5탭            |
| CashwalkBiz | web / mobile / webview     | web                | ➖ 웹 전용(없음)  |

## 호출 기준

- 헤더: `<nds-brand-header brand="..." surface="web|mobile|webview">`. surface 미지정 시 브랜드 기본값.
- 푸터: `<nds-brand-footer brand="..." surface="web|app">`.
- 바텀네비: `<nds-brand-bottom-nav brand="..." active-key="...">` 한 줄로 브랜드별 5탭(라벨/아이콘/색)이
  자동 렌더됩니다. 제네릭 `nds-footer-tab-bar` 에 아이콘 SVG 를 직접 슬롯으로 주입하지 마세요.
- CashwalkBiz 는 웹 전용이라 bottom-nav 가 없습니다(빈 렌더).
- 브랜드 alias: `nds-{nudge-eap|trost|geniet|runmile|cashwalk-biz}-{header|footer|bottom-nav}`.

## 관련 base 컴포넌트 (공개 primitive)

- `Header`: 브랜드-무관 글로벌 헤더 (variant app/webview/web).
- `BottomNav`: 브랜드-무관 모바일 하단 탭 (compound + 슬롯).
- [`Footer`](/components/app-footer): 브랜드-무관 푸터 base compound (Info/TabBar/Web).
