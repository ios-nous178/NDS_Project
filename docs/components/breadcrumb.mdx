# Breadcrumb

현재 페이지의 계층 구조를 표시하는 경로 탐색 컴포넌트입니다.

## 사용법

```tsx
import { Breadcrumb } from "@nudge-design/react";

<Breadcrumb
  items={[
    { label: "홈", href: "/" },
    { label: "심리검사", href: "/test" },
    { label: "스트레스 검사" },
  ]}
/>

// SPA 라우터 연동
<Breadcrumb
  items={items}
  onItemClick={(item, idx, e) => {
    e.preventDefault();
    router.push(item.href!);
  }}
/>

// 커스텀 구분자
<Breadcrumb items={items} separator="/" />
```
