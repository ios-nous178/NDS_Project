# Breadcrumb

현재 페이지의 계층 구조를 표시하는 경로 탐색 컴포넌트입니다.

## 사용법

```tsx
import { Breadcrumb } from "@nudge-eap/react";

<Breadcrumb
  items={[
    { label: "홈", href: "/" },
    { label: "심리검사", href: "/test" },
    { label: "스트레스 검사" },
  ]}
/>

// SPA 라우터 연동
<Breadcrumb
  items={items}
  onItemClick={(item, idx, e) => {
    e.preventDefault();
    router.push(item.href!);
  }}
/>

// 커스텀 구분자
<Breadcrumb items={items} separator="/" />
```

## Props

| Prop          | 타입                       | 기본값        | 설명                    |
| ------------- | -------------------------- | ------------- | ----------------------- |
| `items`       | `BreadcrumbItem[]`         | —             | `{ label, href? }` 목록 |
| `separator`   | `ReactNode`                | `›` (chevron) | 구분자                  |
| `onItemClick` | `(item, index, e) => void` | —             | 아이템 클릭 핸들러      |
