# BreathingGuide

원형 애니메이션으로 호흡을 안내하는 컴포넌트입니다. 박스 호흡(4-4-4-4), 4-7-8 호흡 등 어떤 사이클이든 `phases`로 정의할 수 있습니다.

## 사용 가이드

### 이럴 때 사용하세요
- 명상/이완 콘텐츠 메인 화면
- 검사 직후 잠시 호흡을 유도하는 페이지
- 위기 상황 (CrisisCallout 후 진정 단계)

### 자주 하는 실수
- `cycles`를 안 주면 무한 반복. 명상 종료를 의도한다면 `cycles + onComplete` 페어 사용.
- `playing` prop을 외부에서 제어할 때 `onPlayingChange`로 양방향 동기화 잊지 말 것.

## 사용법

```tsx
import { BreathingGuide } from "@nudge-design/react";

// 박스 호흡 (기본)
<BreathingGuide />

// 4-7-8 (수면 유도)
<BreathingGuide
  phases={[
    { kind: "inhale", seconds: 4 },
    { kind: "hold", seconds: 7 },
    { kind: "exhale", seconds: 8 },
  ]}
/>

// 3 사이클 후 종료
<BreathingGuide cycles={3} autoStart onComplete={handleDone} />

// 외부 제어
<BreathingGuide playing={running} onPlayingChange={setRunning} />
```

## Props

| Prop              | 타입                          | 기본값         | 설명                            |
| ----------------- | ----------------------------- | -------------- | ------------------------------- |
| `phases`          | `BreathingPhase[]`            | 4-4-4-4        | 호흡 사이클 단계                |
| `autoStart`       | `boolean`                     | `false`        | 마운트 즉시 시작                |
| `cycles`          | `number`                      | —              | 사이클 수. 미지정 = 무한        |
| `onComplete`      | `() => void`                  | —              | cycles 도달 시 콜백             |
| `playing`         | `boolean`                     | —              | 외부 제어 (controlled)          |
| `onPlayingChange` | `(playing: boolean) => void`  | —              | 재생 변경 콜백                  |
| `showCount`       | `boolean`                     | `true`         | 카운트다운 숫자 표시            |
| `hideControls`    | `boolean`                     | `false`        | 컨트롤 버튼 숨김                |

### `BreathingPhase`

| Prop      | 타입                                              | 설명                  |
| --------- | ------------------------------------------------- | --------------------- |
| `kind`    | `"inhale" \| "hold" \| "exhale" \| "rest"`        | 단계 종류             |
| `seconds` | `number`                                          | 지속 시간             |
| `label`   | `string`                                          | 라벨 오버라이드 (선택) |
