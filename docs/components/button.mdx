# Button

사용자의 클릭 액션을 처리하는 버튼 컴포넌트입니다. 4가지 변형(variant)과 6가지 사이즈를 제공하며, 아이콘 슬롯을 통해 좌우에 아이콘을 배치할 수 있습니다.

```tsx
import { Button } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요
- 폼 제출, 다이얼로그 확인/취소 등 **사용자 액션을 트리거**할 때
- 상태를 변경하는 동작이 필요할 때 (저장, 삭제, 전송 등)
- 워크플로우의 다음 단계로 진행할 때 (다음, 완료, 시작하기)

### 이럴 때 사용하지 마세요
- 다른 페이지로 이동만 할 때 → `<a>` 태그 또는 라우터의 Link를 사용하세요
- 읽기 전용 상태 라벨을 표시할 때 → [Badge](/docs/components/badge)를 사용하세요
- 선택/해제가 가능한 필터 태그일 때 → [Chip](/docs/components/chip)을 사용하세요

### 자주 하는 실수
- `onClick`에서 페이지 이동만 수행하는 Button — 스크린 리더가 링크로 인식하지 못합니다. 네비게이션에는 `<a>`를 사용하세요
- `disabled` 상태에서 비활성 이유를 안내하지 않음 — Tooltip이나 helperText로 이유를 함께 표시하세요
- `variant` 선택 기준 없이 사용 — 한 화면에서 `solid`는 1개(주요 CTA)만, 나머지는 `outlined` 이하로 위계를 나누세요

## Variant

4가지 스타일 변형을 제공합니다.

<ComponentPreview title="Variants">
  <Button variant="solid">Solid</Button>
  <Button variant="outlined">Outlined</Button>
  <Button variant="soft">Soft</Button>
  <Button variant="outlined-sub">Outlined Sub</Button>
</ComponentPreview>

```tsx
<Button variant="solid">Solid</Button>
<Button variant="outlined">Outlined</Button>
<Button variant="soft">Soft</Button>
<Button variant="outlined-sub">Outlined Sub</Button>
```

| Variant | 설명 | 사용 시점 |
|---|---|---|
| `solid` | 배경 채움 (기본값) | 주요 CTA, 강한 강조 |
| `outlined` | 테두리만 표시 | 보조 액션, solid와 함께 사용 |
| `soft` | 연한 배경 + 컬러 텍스트 | 부드러운 강조 |
| `outlined-sub` | 회색 테두리 + neutral 텍스트 | 가장 약한 강조, 보조 버튼 |

  <Button size="sm">SM (42px)</Button>

```tsx
<Button size="xl">XL (56px)</Button>
<Button size="lg">LG (48px)</Button>
<Button size="md">MD (44px)</Button>
<Button size="sm">SM (42px)</Button>
<Button size="xs">XS (38px)</Button>
<Button size="field">Field (48px)</Button>
```

| Size | 높이 | 폰트 | 패딩 (좌우) | 사용 시점 |
|---|---|---|---|---|
| `xl` | 56px | 18px | 16px | 하단 CTA 등 큰 버튼 |
| `lg` | 48px | 16px | 16px | 기본 버튼 (기본값) |
| `md` | 44px | 15px | 16px | 카드 내부 등 중간 크기 |
| `sm` | 42px | 14px | 12px | 리스트 액션 등 작은 버튼 |
| `xs` | 38px | 13px | 12px | 인라인, 태그 옆 등 |
| `field` | 48px | 15px | 16px | Input 옆 연동 버튼 |

## 아이콘

`leftIcon`, `rightIcon` prop으로 아이콘을 배치합니다.

```tsx
import { SearchIcon, ChevronRightIcon } from "@nudge-eap/icons";

<Button leftIcon={<SearchIcon />}>검색</Button>
<Button rightIcon={<ChevronRightIcon />}>다음</Button>
<Button leftIcon={<SearchIcon />} rightIcon={<ChevronRightIcon />}>검색</Button>
```

## Props

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `variant` | `ButtonVariant` | `"solid"` | 버튼 스타일 변형 |
| `size` | `ButtonSize` | `"lg"` | 버튼 크기 |
| `color` | `ButtonColor` | `"primary"` | 색상 테마 |
| `fullWidth` | `boolean` | `false` | 부모 너비 100% 채움 |
| `leftIcon` | `ReactNode` | — | 왼쪽 아이콘 |
| `rightIcon` | `ReactNode` | — | 오른쪽 아이콘 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `labelClassName` | `string` | — | 라벨 span className |
| `slotProps` | `ButtonSlotProps` | — | 슬롯별 props 전달 |
| `className` | `string` | — | 루트 className |
| `style` | `CSSProperties` | — | 루트 인라인 스타일 |

**ButtonVariant**: `"solid"` | `"outlined"` | `"soft"` | `"outlined-sub"`

**ButtonSize**: `"xl"` | `"lg"` | `"md"` | `"sm"` | `"xs"` | `"field"`

**ButtonColor**: `"primary"` | `"secondary"`

`<button>` HTML 속성을 모두 지원합니다 (`onClick`, `type`, `form` 등).

## SlotProps

```tsx
interface ButtonSlotProps {
  label?: React.HTMLAttributes<HTMLSpanElement>;
  leftIcon?: React.HTMLAttributes<HTMLSpanElement>;
  rightIcon?: React.HTMLAttributes<HTMLSpanElement>;
}
```

```tsx
<Button
  slotProps={{
    label: { className: "tracking-wide" },
    leftIcon: { style: { marginRight: 4 } },
  }}
>
  슬롯 커스텀
</Button>
```

## data-slot 구조

```
button [data-slot="root"] [data-variant] [data-size] [data-color]
├── span [data-slot="left-icon"]    (leftIcon이 있을 때)
├── span [data-slot="label"]
└── span [data-slot="right-icon"]   (rightIcon이 있을 때)
```

styled-components에서 타겟팅:

```tsx
const StyledButton = styled(Button)`
  [data-slot="label"] {
    letter-spacing: 0.05em;
  }
`;
```

## 접근성

- `type="button"` 기본값으로 form 의도치 않은 submit 방지
- `disabled` 시 `cursor: default` 및 색상 변경
- `ref` 전달 지원 (`React.forwardRef`)
