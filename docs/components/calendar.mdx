# Calendar

DatePicker는 popover로 열리는 입력 컨트롤이지만, Calendar는 **화면에 인라인으로 박혀 있는 월간 그리드**입니다.
상담 예약 화면, 감정 캘린더, 복약 캘린더처럼 "캘린더 자체가 콘텐츠인" 화면에서 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 한 화면 안에 캘린더가 항상 펼쳐져 있어야 할 때
- 날짜별로 마커(일정/감정/기록)가 보여야 할 때
- 예약 가능 / 불가능 날짜를 시각적으로 분리해야 할 때

### 이럴 때 사용하지 마세요
- 폼 안에서 날짜 한 개만 입력받을 때 → `DatePicker`
- 시작·종료 두 날짜를 받을 때 → `DateRangePicker`

## 사용법

```tsx
import { Calendar } from "@nudge-design/react";

// 기본
<Calendar value={value} onChange={setValue} />

// 일정 마커
<Calendar
  value={value}
  onChange={setValue}
  markers={[
    { date: "2026-05-12", color: "var(--semantic-primary-main)" },
    { date: "2026-05-19", color: "var(--semantic-caution-main)" },
  ]}
/>

// 비활성 날짜 (과거, 일요일)
<Calendar
  isDateDisabled={(iso) => iso < todayIso || new Date(iso).getDay() === 0}
/>

// 외부에서 월을 제어
<Calendar month="2026-05" hideHeader onMonthChange={setYm} />
```

## Props

| Prop             | 타입                                | 기본값  | 설명                                |
| ---------------- | ----------------------------------- | ------- | ----------------------------------- |
| `month`          | `string \| Date`                    | today   | `YYYY-MM` 또는 Date. 미지정 시 controlled 아님 |
| `onMonthChange`  | `(yearMonth: string) => void`       | —       | 월 변경 콜백                        |
| `value`          | `string`                            | —       | 선택된 날짜 (`YYYY-MM-DD`)          |
| `onChange`       | `(date: string) => void`            | —       | 날짜 클릭 콜백                      |
| `isDateDisabled` | `(date: string) => boolean`         | —       | 비활성 날짜 판별                    |
| `markers`        | `CalendarMarker[]`                  | —       | 날짜별 점 마커. `{ date, color? }`  |
| `hideHeader`     | `boolean`                           | `false` | 헤더(월/네비) 숨김                  |
| `weekStartsOn`   | `0 \| 1`                            | `0`     | 0=일요일, 1=월요일                  |
