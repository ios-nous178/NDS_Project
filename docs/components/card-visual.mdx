# CardVisual

결제 수단 등록·관리 화면에서 카드를 시각적으로 표현하는 컴포넌트입니다. 8가지 브랜드 톤이 내장되어 있고 카드 번호 마지막 4자리만 표시합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 카드 등록 화면
- 결제 수단 관리
- 결제 직전 선택된 카드 표시

### 자주 하는 실수
- `number`는 마지막 4자리만 표시되도록 자동 마스킹 — 전체 번호 넣어도 됨
- 만료된 카드는 `disabled` + `label="만료됨"` 패턴

## 사용법

```tsx
import { CardVisual } from "@nudge-design/react";

<CardVisual
  brand="shinhan"
  number="1234"
  holder="MIN JI KIM"
  expiry="12/27"
/>

// 카카오뱅크 + 별명
<CardVisual brand="kakao" number="5678" label="용돈 카드" />

// 만료
<CardVisual brand="visa" disabled label="만료됨" />
```

## Props

| Prop       | 타입               | 기본값      | 설명                              |
| ---------- | ------------------ | ----------- | --------------------------------- |
| `brand`    | `CardVisualBrand`  | `"generic"` | 브랜드 톤                         |
| `number`   | `string`           | —           | 카드 번호 (마지막 4자리만 표시)   |
| `holder`   | `string`           | —           | 소유자 이름                       |
| `expiry`   | `string`           | —           | 만료일 (MM/YY)                    |
| `label`    | `ReactNode`        | —           | 라벨/별명                         |
| `showChip` | `boolean`          | `true`      | 카드 칩 표시                      |
| `disabled` | `boolean`          | `false`     | 만료/정지                         |
