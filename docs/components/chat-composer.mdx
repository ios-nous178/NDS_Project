# ChatComposer

채팅/상담 화면 하단에 두는 입력바입니다. `ChatBubble`(메시지 표시)과 짝을 이룹니다.

## 사용 가이드

### 이럴 때 사용하세요
- 1:1 채팅 상담 화면
- 챗봇 대화 화면 (quickReplies와 함께)
- AI 상담 입력

### 자주 하는 실수
- `value`/`onValueChange`/`onSubmit` 모두 controlled. 내부에서 입력값 보관 안 함
- `submitOnEnter=true` (기본)에서는 Enter=전송, Shift+Enter=줄바꿈. 모바일 키보드에선 ‘다음 줄’ 키로 줄바꿈
- 빠른 응답을 클릭하면 보통 onClick에서 즉시 onSubmit을 호출하거나, 입력값으로 채워 넣음 — 컴포넌트는 그 정책을 강제하지 않음

## 사용법

```tsx
import { ChatComposer } from "@nudge-design/react";

<ChatComposer
  value={text}
  onValueChange={setText}
  onSubmit={(t) => {
    sendMessage(t);
    setText("");
  }}
  onAttach={openAttach}
  onMic={openMic}
  maxLength={1000}
  quickReplies={[
    { label: "🙂 괜찮아요", onClick: () => sendMessage("괜찮아요") },
    { label: "😣 힘들어요", onClick: () => sendMessage("힘들어요") },
  ]}
/>
```

## Props

| Prop            | 타입                              | 기본값                | 설명                           |
| --------------- | --------------------------------- | --------------------- | ------------------------------ |
| `value`         | `string`                          | —                     | 입력값 (controlled)            |
| `onValueChange` | `(v: string) => void`             | —                     | 변경 콜백                      |
| `onSubmit`      | `(v: string) => void`             | —                     | 전송 콜백                      |
| `placeholder`   | `string`                          | `"메시지를 입력하세요"` | 자리표시자                  |
| `maxLength`     | `number`                          | —                     | 최대 글자 수 (지정 시 카운터)  |
| `onAttach`      | `() => void`                      | —                     | 첨부 콜백 (지정 시 버튼 노출)  |
| `onMic`         | `() => void`                      | —                     | 음성 콜백 (지정 시 버튼 노출)  |
| `quickReplies`  | `QuickReply[]`                    | —                     | 입력 영역 위 빠른 응답 칩      |
| `disabled`      | `boolean`                         | `false`               | 비활성화                       |
| `submitOnEnter` | `boolean`                         | `true`                | Enter=전송 (Shift+Enter 줄바꿈) |
| `maxHeight`     | `number`                          | `120`                 | textarea 최대 높이 px          |
