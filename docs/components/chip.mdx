# Chip

선택 가능한 태그/필터 컴포넌트입니다. 읽기 전용 `Badge`와 달리 interactive한 선택/해제, 삭제 동작을 지원합니다.

```tsx
import { Chip } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 사용자가 **선택/해제/삭제**할 수 있는 인터랙티브 태그일 때 (필터, 관심사, 태그 선택 등)
- 선택된 항목을 시각적으로 표시하고 제거할 수 있게 할 때
- 카테고리 필터를 토글 형태로 제공할 때

### 이럴 때 사용하지 마세요

- 읽기 전용 상태 라벨 → [Badge](/docs/components/badge)를 사용하세요
- 주요 액션 트리거 → [Button](/docs/components/button)을 사용하세요
- 폼에서 명확한 복수 선택 → [Checkbox](/docs/components/checkbox-radio)를 사용하세요

### 자주 하는 실수

- 읽기 전용 라벨에 Chip을 사용 — 사용자가 클릭 가능하다고 오해합니다. 읽기 전용이면 Badge를 사용하세요
- 너무 긴 텍스트를 Chip에 넣기 — Chip은 짧은 키워드에 적합합니다

---

## 기본 사용법

```tsx
<Chip label="심리상담" />
```

### 토글 선택

```tsx
<Chip label="스트레스" variant="filled" selected={isSelected} onClick={() => toggle()} />
```

### 삭제 가능

```tsx
<Chip label="서울" variant="outlined" selected onRemove={() => removeFilter("서울")} />
```

### 아이콘 포함

```tsx
<Chip label="새 상담" icon={<PlusIcon />} variant="soft" onClick={handleAdd} />
```

## Props

| Prop       | 타입                               | 기본값       | 설명                              |
| ---------- | ---------------------------------- | ------------ | --------------------------------- |
| `label`    | `string`                           | —            | 라벨 텍스트                       |
| `variant`  | `"outlined" \| "filled" \| "soft"` | `"outlined"` | 스타일 변형                       |
| `size`     | `"sm" \| "md" \| "lg"`             | `"md"`       | 크기                              |
| `selected` | `boolean`                          | `false`      | 선택 상태                         |
| `disabled` | `boolean`                          | `false`      | 비활성화                          |
| `onClick`  | `() => void`                       | —            | 클릭 콜백 (있으면 interactive)    |
| `onRemove` | `() => void`                       | —            | 삭제 콜백 (있으면 삭제 버튼 표시) |
| `icon`     | `ReactNode`                        | —            | 좌측 아이콘                       |

---

## 변형별 스타일

| 변형       | 기본                     | 선택                     |
| ---------- | ------------------------ | ------------------------ |
| `outlined` | 회색 테두리, 흰 배경     | 파란 테두리, 파란 텍스트 |
| `filled`   | 회색 배경                | 파란 배경, 흰 텍스트     |
| `soft`     | 연파랑 배경, 파란 텍스트 | —                        |

## 크기별 높이

| Size | 높이 | 폰트            |
| ---- | ---- | --------------- |
| `sm` | 28px | caption2 (12px) |
| `md` | 36px | body3 (14px)    |
| `lg` | 44px | body2 (15px)    |

## Badge와의 차이

|          | Badge                 | Chip                 |
| -------- | --------------------- | -------------------- |
| 용도     | 상태 표시 (읽기 전용) | 필터/선택 (인터랙션) |
| 클릭     | 불가                  | 가능                 |
| 삭제     | 불가                  | 가능                 |
| selected | 없음                  | 있음                 |

## data-slot 구조

```
div [data-slot="root"] [data-variant] [data-size] [data-selected] [data-interactive] [data-disabled]
├── span [data-slot="icon"]
├── span [data-slot="label"]
└── button [data-slot="remove"]
```

## 마이그레이션

```tsx
// Before (WebView FilterChip)
<FilterChip label="서울" handleRemove={() => remove("서울")} />

// After
<Chip label="서울" variant="outlined" selected onRemove={() => remove("서울")} />

// Before (HomePage CounselorSubjectChip)
<CounselorSubjectChip subject="심리상담" isSelected={true} onClick={toggle} />

// After
<Chip label="심리상담" variant="filled" size="lg" selected={true} onClick={toggle} />
```

## 접근성

- interactive일 때 `role="button"`, `tabIndex=0`
- 토글형일 때 `aria-pressed` 적용
- 삭제 버튼에 `aria-label="{label} 삭제"` 내장
- Enter / Space 키보드 지원
