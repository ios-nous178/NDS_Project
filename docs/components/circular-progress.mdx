# CircularProgress

값/최댓값 비율을 원형 링으로 보여주는 컴포넌트입니다. CircularProgress는 단계 구분 없는 **순수 진행도** 표현입니다.

## 사용 가이드

### 이럴 때 사용하세요
- 일일 목표 달성률 (걸음수, 명상 시간)
- 챌린지 완료 비율
- 업로드/저장 진행도

### 이럴 때 사용하지 마세요
- 가로 막대로 충분한 단순 진행도 → `ProgressBar`

## 사용법

```tsx
import { CircularProgress } from "@nudge-design/react";

<CircularProgress value={65} />

// 분 단위 라벨 커스텀
<CircularProgress value={28} max={60} label="28분" caption="목표 60분" />

// semantic 색상
<CircularProgress
  value={95}
  color="var(--semantic-success-main)"
  caption="달성"
/>

// 더 크고 두꺼운 링
<CircularProgress value={70} size={160} thickness={14} />
```
