# CircularProgress

값/최댓값 비율을 원형 링으로 보여주는 컴포넌트입니다. ScoreGauge가 단계(normal/mild/moderate/severe)를 가진 결과 표시 위주라면, CircularProgress는 **순수 진행도** 표현입니다.

## 사용 가이드

### 이럴 때 사용하세요
- 일일 목표 달성률 (걸음수, 명상 시간)
- 챌린지 완료 비율
- 업로드/저장 진행도

### 이럴 때 사용하지 마세요
- 심리검사 결과 같이 단계별 분류가 중요한 경우 → `ScoreGauge`
- 가로 막대로 충분한 단순 진행도 → `ProgressBar`

## 사용법

```tsx
import { CircularProgress } from "@nudge-design/react";

<CircularProgress value={65} />

// 분 단위 라벨 커스텀
<CircularProgress value={28} max={60} label="28분" caption="목표 60분" />

// semantic 색상
<CircularProgress
  value={95}
  color="var(--semantic-success-main)"
  caption="달성"
/>

// 더 크고 두꺼운 링
<CircularProgress value={70} size={160} thickness={14} />
```

## Props

| Prop          | 타입         | 기본값       | 설명                                |
| ------------- | ------------ | ------------ | ----------------------------------- |
| `value`       | `number`     | —            | 현재 값                             |
| `max`         | `number`     | `100`        | 최댓값                              |
| `size`        | `number`     | `80`         | 정사각 px                           |
| `thickness`   | `number`     | 자동 (`size/12`) | 링 두께                         |
| `color`       | `string`     | `primary`    | 채움 색                             |
| `trackColor`  | `string`     | `bg.coolGray`| 트랙 색                             |
| `label`       | `ReactNode`  | `"{percent}%"` | 가운데 라벨                       |
| `caption`     | `ReactNode`  | —            | 라벨 아래 보조 텍스트               |
| `hideLabel`   | `boolean`    | `false`      | 라벨 영역 숨김 (작은 인디케이터용)  |
| `aria-label`  | `string`     | `"진행도"`    | 접근성 라벨                         |
