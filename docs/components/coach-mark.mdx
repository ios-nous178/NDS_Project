# CoachMark

화면 전체에 어두운 오버레이를 깔고 특정 영역만 구멍을 뚫어 보여주면서 단계별 안내를 띄우는 컴포넌트입니다. Tooltip(가벼운 호버 안내)과 분리됩니다.

## 사용 가이드

### 이럴 때 사용하세요
- 첫 진입 시 핵심 기능 3~5단계 안내
- 새 기능 출시 후 위치 알림
- 사용자가 명시적으로 "도움말" 요청 시

### 이럴 때 사용하지 마세요
- 단순 hover 설명 → `Tooltip`
- 매번 노출되어야 하는 안내 → 영구 UI 요소로 처리

### 자주 하는 실수
- `target`은 selector 또는 ref-getter — 컴포넌트 마운트 시점에 DOM에 존재해야 함
- 스크롤이 필요한 위치를 가리킬 때는 사전에 `scrollIntoView` 직접 호출 (CoachMark가 자동 스크롤하지 않음)

## 사용법

```tsx
import { CoachMark } from "@nudge-design/react";

<CoachMark
  open={onboardingOpen}
  onClose={() => setOnboardingOpen(false)}
  steps={[
    {
      target: "#nav-record",
      title: "기록을 시작해보세요",
      description: "여기서 오늘의 기분을 남길 수 있어요.",
      placement: "bottom",
    },
    {
      target: () => myRef.current,
      title: "다음 단계",
      placement: "right",
    },
  ]}
/>
```

## Props

| Prop           | 타입                                              | 기본값        | 설명                                  |
| -------------- | ------------------------------------------------- | ------------- | ------------------------------------- |
| `open`         | `boolean`                                         | —             | 활성 여부                             |
| `steps`        | `CoachMarkStep[]`                                 | —             | 단계 배열                             |
| `step`         | `number`                                          | —             | 현재 인덱스 (controlled, 선택)        |
| `onStepChange` | `(i: number) => void`                             | —             | 단계 변경 콜백                        |
| `onClose`      | `() => void`                                      | —             | 닫기 (skip 또는 마지막 완료)          |
| `finishLabel`  | `string`                                          | `"완료"`      | 마지막 단계 버튼                      |
| `nextLabel`    | `string`                                          | `"다음"`      | 다음 단계 버튼                        |
| `skipLabel`    | `string`                                          | `"건너뛰기"`  | Skip 버튼                             |
| `hideSkip`     | `boolean`                                         | `false`       | Skip 버튼 숨김                        |

### `CoachMarkStep`

| Prop          | 타입                                            | 설명                              |
| ------------- | ----------------------------------------------- | --------------------------------- |
| `target`      | `string \| () => HTMLElement \| null`           | selector 또는 element-getter      |
| `title`       | `ReactNode`                                     | 단계 제목                         |
| `description` | `ReactNode`                                     | 설명 (선택)                       |
| `placement`   | `"top" \| "bottom" \| "left" \| "right"`        | 툴팁 위치 (기본 `"bottom"`)       |
| `padding`     | `number`                                        | 하이라이트 영역 패딩 (기본 8)     |
