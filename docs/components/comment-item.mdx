# CommentItem

소셜/콘텐츠/그룹 화면의 댓글 한 줄을 표현하는 컴포넌트입니다. 답글(replies)을 내부에 중첩 가능하고 좋아요/답글 액션 슬롯을 제공합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 콘텐츠/일기/Q&A 댓글
- 그룹 채팅에 부착된 코멘트
- 후기 페이지의 댓글 영역

### 자주 하는 실수
- 댓글 본문에 줄바꿈 보존이 필요하면 `text`에 `\n` 그대로 넣으면 됨 (자동 `white-space: pre-wrap`)
- 답글에는 `isReply`로 들여쓰기 시각 강조 — 부모 안에 자식 CommentItem 묶을 때 `replies`에 넣고 자식에 `isReply`

## 사용법

```tsx
import { CommentItem, Avatar, Badge, LikeButton } from "@nudge-eap/react";

<CommentItem
  avatar={<Avatar name="김민지" />}
  author="김민지"
  authorBadge={<Badge variant="fill" color="brand" size="sm">작성자</Badge>}
  time="3분 전"
  text="좋은 글 감사해요 :)"
  likeAction={<LikeButton liked={liked} count={count} onChange={...} />}
  onReply={openReplyComposer}
  replies={
    <>
      <CommentItem isReply ... />
      <CommentItem isReply ... />
    </>
  }
/>
```

## Props

| Prop          | 타입         | 기본값  | 설명                              |
| ------------- | ------------ | ------- | --------------------------------- |
| `avatar`      | `ReactNode`  | —       | 작성자 아바타                     |
| `author`      | `ReactNode`  | —       | 작성자 이름                       |
| `authorBadge` | `ReactNode`  | —       | 작성자 뱃지 (작성자/상담사 등)    |
| `time`        | `ReactNode`  | —       | 시간 라벨                         |
| `text`        | `ReactNode`  | —       | 본문                              |
| `likeAction`  | `ReactNode`  | —       | 좋아요 슬롯 (LikeButton 등)       |
| `onReply`     | `() => void` | —       | 답글 클릭 (지정 시 답글 버튼 노출) |
| `replyLabel`  | `string`     | `"답글"` | 답글 버튼 라벨                  |
| `more`        | `ReactNode`  | —       | 우상단 더보기 메뉴 슬롯           |
| `replies`     | `ReactNode`  | —       | 답글 영역 (자식 CommentItem)      |
| `isReply`     | `boolean`    | `false` | 답글 들여쓰기                     |
