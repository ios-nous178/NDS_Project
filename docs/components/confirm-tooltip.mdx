# ConfirmTooltip

트리거 요소 옆에 흰 말풍선을 띄워 제목/본문과 1~2개 액션 버튼으로 가벼운 확인을 받는 popconfirm 컴포넌트입니다. 방향 tail 로 어떤 요소에 대한 확인인지 가리킵니다.

## 사용 가이드

### 이럴 때 사용하세요
- 연결 해제·삭제 등 인라인 트리거 옆에서 가벼운 확인을 받을 때
- 사용자의 응답/결정이 필요하지만 화면을 막을 필요는 없을 때

### 이럴 때 사용하지 마세요
- 차단형·긴 본문의 확인 → `Modal` / `Popup`
- 다크 배경의 단순 hover 안내 → `Tooltip`
- 온보딩 단계별 dim 안내 → `CoachMark`

### 자주 하는 실수
- `open` 은 controlled — 직접 상태로 열고 닫아야 함 (내부 토글 없음)
- `children` 은 트리거 요소 — 이 요소를 기준으로 말풍선이 anchor 됨
- 너비 조절은 `bodyWidth` (기본 280) — number 면 px 로 적용

## 사용법

```tsx
import { ConfirmTooltip } from "@nudge-design/react";

<ConfirmTooltip
  open={open}
  title="연결을 해제하시겠습니까?"
  description="해제하면 다시 연결할 때 인증이 필요합니다."
  placement="top"
  confirmLabel="해제"
  cancelLabel="취소"
  onConfirm={() => {
    disconnect();
    setOpen(false);
  }}
  onCancel={() => setOpen(false)}
>
  <button onClick={() => setOpen(true)}>연결 해제</button>
</ConfirmTooltip>
```

## Props

| Prop           | 타입                                       | 기본값   | 설명                                                  |
| -------------- | ------------------------------------------ | -------- | ----------------------------------------------------- |
| `title`        | `ReactNode`                                | —        | 확인 질문 제목                                        |
| `description`  | `ReactNode`                                | —        | 보조 설명 본문 (선택)                                 |
| `open`         | `boolean`                                  | `false`  | 말풍선 표시 여부 (controlled)                         |
| `placement`    | `"top" \| "bottom" \| "left" \| "right"`   | `"top"`  | 말풍선 위치 (anchor 기준)                             |
| `actions`      | `"dual" \| "single"`                       | `"dual"` | 버튼 구성 — dual(취소+확인) / single(확인만)          |
| `bodyWidth`    | `number \| string`                         | `280`    | 본문/액션 영역 너비 (number → px)                     |
| `confirmLabel` | `string`                                   | `"확인"` | 확인 버튼 라벨 (검정 CTA)                             |
| `cancelLabel`  | `string`                                   | `"취소"` | 취소 버튼 라벨 (`actions="dual"` 일 때만)             |
| `onConfirm`    | `() => void`                               | —        | 확인 버튼 클릭                                        |
| `onCancel`     | `() => void`                               | —        | 취소 버튼 클릭                                        |
| `children`     | `ReactNode`                                | —        | 트리거 요소 — 말풍선이 anchor 되는 기준               |
