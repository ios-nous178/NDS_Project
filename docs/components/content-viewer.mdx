# ContentViewer

검사 결과 해설·명상 가이드·정신건강 콘텐츠 등 **HTML 본문을 안전하게 렌더링**합니다.

## 사용법

```tsx
import { ContentViewer } from "@nudge-design/react";

<ContentViewer
  html={post.content}
  // 기본값으로 다음 자동 처리:
  // - script/iframe/on*=/javascript: 위험 패턴 제거
  // - <img>에 loading="lazy" + decoding="async"
  // - 외부 링크(http/https)에 target="_blank" + rel="noopener noreferrer"
/>
```

## Props

| Prop                | 타입         | 기본값  | 설명                                                |
| ------------------- | ------------ | ------- | --------------------------------------------------- |
| `html`              | `string`     | —       | 본문 HTML (가능한 한 호출부에서 sanitize한 것 권장) |
| `sanitize`          | `boolean`    | `true`  | 컴포넌트 내장 위험 태그 제거                        |
| `imageLazy`         | `boolean`    | `true`  | 이미지에 loading="lazy" 자동                        |
| `externalLinkBlank` | `boolean`    | `true`  | 외부 링크에 target="_blank" + rel 자동              |

## 자동 적용되는 본문 스타일

`h1~h4`, `p`, `ul/ol`, `blockquote`, `code`, `pre`, `img`, `hr`, `table` 등 흔한 HTML 요소에 일관된 토큰 기반 스타일이 자동 적용됩니다.

## 가이드

- **사용자 입력 HTML은 반드시 서버 또는 DOMPurify로 1차 sanitize 후 넘길 것** — 내장 sanitize는 알려진 attack vector(`<script>`, `on*=`, `javascript:`, `<iframe>` 등)만 커버하는 보완책.
- 내부 라우팅 링크는 그대로 유지 — `http(s)://`로 시작하는 외부 링크에만 `target="_blank"` 자동.
- 본문 안 `<table>`, `<pre>`, `<blockquote>`까지 표준 스타일 적용. 검사 해설처럼 구조 있는 본문에 적합.
- 마크다운 입력이면 호출부에서 `marked` 등으로 HTML로 변환 후 넘기기.
