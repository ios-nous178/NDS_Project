# CouponCard

좌측에 할인율(또는 금액), 우측에 제목·조건·만료일·사용 버튼이 있는 쿠폰 카드입니다. 좌우 사이에 점선 + 반원 컷아웃이 자동.

## 사용 가이드

### 이럴 때 사용하세요
- 할인 쿠폰함
- EAP 무료 코드
- 챌린지 보상권

### 자주 하는 실수
- `discount`와 `discountSuffix` 분리 — "30%할인"이 아니라 "30%" + "할인" 따로
- `disabled=true`면 버튼 자동으로 `disabledLabel("사용 완료")`로 변경

## 사용법

```tsx
import { CouponCard } from "@nudge-design/react";

<CouponCard
  discount="30%"
  title="첫 구매 30% 할인"
  description="3만원 이상 구매 시"
  expiry="2026.06.30 까지"
  onAction={useCoupon}
/>

// 금액 할인
<CouponCard discount="5,000" discountSuffix="원" ... />

// 사용 완료
<CouponCard ... disabled />
```

## Props

| Prop             | 타입         | 기본값         | 설명                              |
| ---------------- | ------------ | -------------- | --------------------------------- |
| `discount`       | `ReactNode`  | —              | 할인율/금액 (큰 숫자)             |
| `discountSuffix` | `ReactNode`  | `"할인"`       | 보조 단위 (할인/원/OFF)           |
| `title`          | `ReactNode`  | —              | 쿠폰 제목                         |
| `description`    | `ReactNode`  | —              | 조건/설명                         |
| `expiry`         | `ReactNode`  | —              | 만료일                            |
| `actionLabel`    | `string`     | `"사용하기"`   | 활성 버튼 라벨                    |
| `onAction`       | `() => void` | —              | 사용 콜백                         |
| `disabled`       | `boolean`    | `false`        | 사용 완료/만료                    |
| `disabledLabel`  | `string`     | `"사용 완료"`  | 비활성 버튼 라벨                  |
