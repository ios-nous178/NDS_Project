# CrisisCallout

자살·자해 사고 등 위기 신호 발생 시 즉시 연결 가능한 핫라인을 노출하는 강조 박스입니다. 일반 `Banner`와 달리 **닫기가 불가능**합니다 — EAP 플랫폼의 안전 책임 영역이기 때문입니다.

## 사용법

```tsx
import { CrisisCallout } from "@nudge-eap/react";

// 위기 (자살예방·응급)
<CrisisCallout
  tone="danger"
  title="혼자 견디지 마세요"
  description="지금 너무 힘드시다면 24시간 상담 받을 수 있어요."
  actions={[
    { label: "1393 자살예방상담", phoneNumber: "1393" },
    { label: "119 응급", phoneNumber: "119", variant: "outlined" },
  ]}
/>

// 주의 (가벼운 톤)
<CrisisCallout
  tone="caution"
  title="잠시 숨을 고를 시간이에요"
  description="스트레스가 평소보다 높아 보여요."
  actions={[{ label: "명상 시작하기", onClick: () => {} }]}
/>
```

## Props

| Prop          | 타입                    | 기본값     | 설명                              |
| ------------- | ----------------------- | ---------- | --------------------------------- |
| `tone`        | `"danger" \| "caution"` | `"danger"` | 톤 (danger=빨강, caution=주황)    |
| `title`       | `ReactNode`             | —          | 제목                              |
| `description` | `ReactNode`             | —          | 설명                              |
| `actions`     | `CrisisAction[]`        | —          | 액션 (전화/링크), 보통 1~2개      |
| `icon`        | `ReactNode`             | 기본 ⚠️    | 좌측 아이콘 커스터마이즈          |

`CrisisAction = { label, onClick?, phoneNumber?, variant?: "solid" | "outlined", withPhoneIcon? }`

## 가이드

- **dismiss 불가능**합니다. closable 같은 prop이 없습니다.
- 자살 사고 등 실제 위기는 무조건 `tone="danger"`. `caution`은 "잠시 휴식이 필요해요" 수준에만.
- `phoneNumber` 제공 시 자동으로 `tel:` 링크 — 모바일에서 바로 통화.
- `AssessmentResultCard`의 `severe` 결과 옆이나 채팅 화면 상단에 배치하는 게 권장 패턴.
