# DataTable

데이터 행을 표로 표시합니다. **모바일에서는 카드로 자동 변환**하는 옵션이 있어 사용자 앱과 운영 화면 모두에서 사용 가능합니다.

> 💡 CMS/어드민에서는 antd `Table`이 우선입니다 (CLAUDE.md 분기 룰). DataTable은 사용자 앱 흐름이나 antd를 쓰지 않는 곳에서.

## 사용법

```tsx
import { DataTable, type DataTableColumn } from "@nudge-design/react";

interface Med {
  id: string;
  name: string;
  dosage: string;
  taken: number;
  total: number;
}

const columns: DataTableColumn<Med>[] = [
  { key: "name", title: "약 이름", sortable: true },
  { key: "dosage", title: "용량/주기" },
  {
    key: "taken",
    title: "복용",
    align: "right",
    sortable: true,
    render: (row) => `${row.taken} / ${row.total}`,
  },
];

<DataTable
  columns={columns}
  data={meds}
  rowKey={(r) => r.id}
  responsive="cards"   // 모바일에서 카드로 변환
  onRowClick={(row) => router.push(`/meds/${row.id}`)}
/>
```

`DataTableColumn<T> = { key, title, width?, align?, sortable?, render?, cardLabel?, hideOnCard? }`

## 가이드

- **정렬은 controlled** — `sortKey`/`sortDirection`/`onSort`를 부모에서 관리. 컴포넌트가 데이터를 자체 정렬하지 않음.
- `rowKey`는 안정적인 식별자 함수. index를 그대로 쓰면 reorder 시 버그.
- 행이 길면 외부에 `Pagination` 컴포넌트와 조합.
- `responsive="cards"` 시 `cardLabel`/`hideOnCard`로 카드 모드 라벨/숨김 조정.
