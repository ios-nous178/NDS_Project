# DatePicker

단일 날짜 선택 popover. 트리거 필드를 클릭하면 캘린더가 펼쳐집니다. `Calendar`(인라인)와 분리된 입력 형태.

## 사용법

```tsx
import { DatePicker } from "@nudge-eap/react";

const [date, setDate] = useState<Date | undefined>();

<DatePicker
  value={date}
  onChange={setDate}
  minDate={new Date()}
  placeholder="날짜를 선택하세요"
/>
```


> Calendar(인라인 캘린더)와 분리. 입력 필드 형태로 폼에 들어갈 때 사용.
