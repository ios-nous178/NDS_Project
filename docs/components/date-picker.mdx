# DatePicker

단일 날짜를 선택하는 popover 컴포넌트입니다. 트리거 버튼을 누르면 캘린더 패널이 포털로 펼쳐지고, `minDate` / `maxDate`로 선택 가능한 범위를 제한할 수 있습니다. 인라인 캘린더가 필요한 경우 `Calendar`를 사용하세요.

```tsx
import { DatePicker } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 폼에서 **단일 날짜를 입력**받아야 할 때 (생년월일, 예약일 등)
- 트리거 + 드롭다운 형태로 **공간을 적게 차지**해야 할 때
- `min` / `max`로 **선택 가능한 날짜 범위를 제한**해야 할 때

### 이럴 때 사용하지 마세요

- 캘린더가 항상 보여야 할 때 → `Calendar`(인라인)를 사용하세요
- 기간(시작 ~ 종료)을 선택할 때 → `DateRangePicker`를 사용하세요
- 시간 포함 선택 → 별도 Time 입력과 조합하거나 도메인 컴포넌트를 사용하세요

## 기본 사용

```tsx
const [date, setDate] = useState<Date | undefined>();

<DatePicker
  value={date}
  onChange={setDate}
  minDate={new Date()}
  placeholder="날짜를 선택하세요"
/>;
```

### 포맷 커스터마이즈

```tsx
<DatePicker
  value={date}
  onChange={setDate}
  formatValue={(d) => `${d.getFullYear()}년 ${d.getMonth() + 1}월 ${d.getDate()}일`}
/>
```

### 에러 상태 + 전체 너비

```tsx
<DatePicker value={date} onChange={setDate} error fullWidth />
```

## Props

| Prop              | 타입                       | 기본값       | 설명                                       |
| ----------------- | -------------------------- | ------------ | ------------------------------------------ |
| `value`           | `Date`                     | —            | 선택된 날짜                                |
| `onChange`        | `(value: Date) => void`    | —            | 변경 콜백 (필수)                           |
| `minDate`         | `Date`                     | —            | 선택 가능한 최소 날짜 (포함)               |
| `maxDate`         | `Date`                     | —            | 선택 가능한 최대 날짜 (포함)               |
| `placeholder`     | `string`                   | `"날짜 선택"` | 미선택 상태의 트리거 텍스트                |
| `disabled`        | `boolean`                  | `false`      | 비활성화                                   |
| `error`           | `boolean`                  | `false`      | 에러 상태 (테두리 색 변경)                 |
| `fullWidth`       | `boolean`                  | `false`      | 트리거 너비를 100%로                       |
| `className`       | `string`                   | —            | 루트 className                             |
| `portalContainer` | `HTMLElement \| null`      | —            | 포털 컨테이너 (미지정 시 `document.body`)  |
| `formatValue`     | `(d: Date) => string`      | `YYYY.MM.DD` | 트리거 표시용 포맷 함수                    |

## data-slot 구조

```
div [data-slot="root"] [data-fullwidth]
├── button [data-slot="trigger"] [data-open] [data-error]   (aria-haspopup="dialog")
│   ├── span [data-slot="trigger-text"] [data-placeholder]
│   └── span.nds-date-picker__icon  (캘린더 아이콘)
└── (Portal)
    └── div [data-slot="panel"]  (role="dialog", aria-labelledby)
        ├── div [data-slot="header"]
        │   ├── button (aria-label="이전 달")
        │   ├── span [data-slot="title"]
        │   └── button (aria-label="다음 달")
        ├── div [data-slot="dow"]
        │   └── span [data-slot="dow-cell"] [data-day="0..6"]
        └── div [data-slot="grid"]  (role="grid")
            └── button [data-slot="day"] [data-outside] [data-selected] [data-today] [data-disabled] [data-day]  (role="gridcell")
```

## CSS 변수

| 변수                       | 기본값  | 설명                       |
| -------------------------- | ------- | -------------------------- |
| `--nds-date-picker-width`  | `auto`  | 루트 너비 (fullWidth=true면 `100%`) |

색상·간격·shadow는 시멘틱 토큰(`cv.bg.white`, `cv.border.default`, `cv.primary.main`, `shadow["2"]` 등)에 바인딩됩니다.

## 접근성

- 트리거: `aria-haspopup="dialog"`, `aria-expanded`로 패널 상태 노출
- 패널: `role="dialog"` + `aria-labelledby`로 월/년 헤더와 연결
- 셀: `role="grid"` + `aria-selected`로 선택 상태 표시, 일요일 셀에 `data-day="0"`로 빨간색
- 키보드: ESC로 닫고 트리거에 포커스 복귀
- 외부 클릭/터치 시 자동 닫힘 (내장 dismissable layer)
