# DateRangePicker

시작/끝 날짜를 한 쌍으로 선택합니다. `DatePicker` 두 개를 묶어 시작·끝 일관성 검증과 빠른 프리셋(최근 7일/30일/이번 달)을 추가했습니다.

## 사용법

```tsx
import { DateRangePicker, defaultRangePresets, type DateRange } from "@nudge-design/react";

const [range, setRange] = useState<DateRange>({});

<DateRangePicker
  value={range}
  onValueChange={setRange}
  presets={defaultRangePresets}
  maxDate={new Date()} // 미래는 못 고르게
/>
```

## 기본 프리셋

`defaultRangePresets` 으로 export — 최근 7일 / 최근 30일 / 이번 달.

## 가이드

- `value` 부분 선택 가능 — 시작만 있고 끝이 없을 수 있음. 폼 검증 시 둘 다 체크.
- 끝일 `DatePicker`는 자동으로 `minDate=시작일` — 시작일을 뒤로 옮기면 끝일이 비워집니다.
- `presets[].range`는 **함수** (호출 시점 기준 계산). 객체 리터럴로 박으면 영원히 같은 날짜.
- 검사·리포트마다 다른 기본 프리셋이 필요하면 직접 정의해서 넘기세요.
