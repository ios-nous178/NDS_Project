# DropdownMenu

버튼 트리거 기반 액션 드롭다운입니다. **명령형 액션 모음(편집/공유/삭제 등)** 에 사용하며, 값을 선택하는 용도는 `Select`를 사용합니다. 아이템에 leading 아이콘 / trailing 보조(단축키·칩) / `danger` 강조 / 비활성 상태를 지원하고, `groups`로 섹션 라벨을 가진 그룹을 구성할 수 있습니다.

```tsx
import { DropdownMenu } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- `IconButton`(더보기 ⋯) 뒤에 숨기는 **컨텍스트 액션 모음**
- 한 화면에 너무 많은 버튼을 노출하지 않도록 **보조 액션을 묶을 때**
- 같은 대상에 대한 여러 **명령형 동작**(편집/공유/삭제)을 한 곳에 모을 때

### 이럴 때 사용하지 마세요

- 미리 정해진 옵션 중 **값을 선택**할 때 → [Select](/components/select)를 사용하세요
- 단일 액션 1개만 필요할 때 → [Button](/components/button) 또는 [IconButton](/components/icon-button)를 직접 사용하세요
- 메뉴 트리/하위 메뉴 → 별도 메뉴 컴포넌트를 검토하세요 (현재 sub-menu 미지원)

## 기본 사용

### 단일 아이템 목록

```tsx
<DropdownMenu
  items={[
    { key: "edit", label: "편집", onSelect: () => {} },
    { key: "share", label: "공유", onSelect: () => {} },
    { key: "delete", label: "삭제", danger: true, onSelect: () => {} },
  ]}
>
  <IconButton aria-label="더보기" icon={<MoreIcon />} />
</DropdownMenu>
```

### 그룹 + leading / trailing 슬롯

```tsx
<DropdownMenu
  groups={[
    {
      key: "edit",
      label: "편집",
      items: [
        { key: "rename", label: "이름 변경", leading: <EditIcon />, trailing: "⌘E" },
        { key: "duplicate", label: "복제", leading: <CopyIcon />, trailing: "⌘D" },
      ],
    },
    {
      key: "danger",
      items: [
        { key: "delete", label: "삭제", leading: <TrashIcon />, danger: true },
      ],
    },
  ]}
  align="end"
  minWidth={200}
>
  <IconButton aria-label="더보기" icon={<MoreIcon />} />
</DropdownMenu>
```

## Props

### DropdownMenu

| Prop              | 타입                       | 기본값     | 설명                                              |
| ----------------- | -------------------------- | ---------- | ------------------------------------------------- |
| `children`        | `ReactElement`             | —          | 트리거 (단일 React 엘리먼트). 내부에서 `cloneElement`로 `onClick` · `aria-*` 주입 |
| `items`           | `DropdownMenuItem[]`       | —          | 단일 아이템 목록 (`groups`와 둘 중 하나)          |
| `groups`          | `DropdownMenuGroup[]`      | —          | 그룹화된 아이템 (라벨·구분선 자동)                |
| `align`           | `"start" \| "end"`         | `"start"`  | 트리거 기준 정렬                                  |
| `minWidth`        | `number`                   | `160`      | 패널 최소 너비 (px)                               |
| `portalContainer` | `HTMLElement \| null`      | —          | 포털 컨테이너 (미지정 시 `document.body`)         |
| `panelClassName`  | `string`                   | —          | 패널 className                                    |

### DropdownMenuItem

| 속성       | 타입            | 설명                                            |
| ---------- | --------------- | ----------------------------------------------- |
| `key`      | `string`        | 아이템 식별자                                   |
| `label`    | `ReactNode`     | 라벨                                            |
| `onSelect` | `() => void`    | 클릭 콜백 (호출 후 메뉴 자동 닫힘)              |
| `leading`  | `ReactNode`     | 좌측 아이콘                                     |
| `trailing` | `ReactNode`     | 우측 보조 (단축키, 칩 등 — 회색 caption 스타일) |
| `disabled` | `boolean`       | 비활성화 (클릭 무시 + 색상 변경)                |
| `danger`   | `boolean`       | 위험 액션 (빨간 텍스트 + 호버 시 빨간 배경)     |

### DropdownMenuGroup

| 속성     | 타입                   | 설명                                          |
| -------- | ---------------------- | --------------------------------------------- |
| `key`    | `string`               | 그룹 식별자                                   |
| `label`  | `ReactNode`            | 그룹 헤더 라벨 (생략 시 헤더 없이 구분선만)   |
| `items`  | `DropdownMenuItem[]`   | 그룹 내 아이템                                |

## data-slot 구조

```
<trigger>  (cloneElement로 onClick / aria-haspopup / aria-expanded / aria-controls 주입)
(Portal)
└── div [data-slot="panel"] [data-align="start" | "end"]   (role="menu")
    └── div [data-slot="group"]   (role="group")
        ├── div [data-slot="group-label"]
        └── button [data-slot="item"] [data-danger] [data-disabled]   (role="menuitem")
            ├── span [data-slot="item-leading"]
            ├── span [data-slot="item-label"]
            └── span [data-slot="item-trailing"]
```

## 접근성

- 패널: `role="menu"`, 아이템: `role="menuitem"`, 그룹: `role="group"`
- 트리거에 `aria-haspopup="menu"`, `aria-expanded`, `aria-controls` 자동 주입
- ESC로 닫고 트리거에 포커스 복귀
- 외부 클릭/터치 자동 닫힘 (`addDismissableLayerListeners`)
- `disabled` 아이템은 native `disabled` 속성 + `data-disabled="true"`
- 트리거는 단일 React 엘리먼트만 허용 — `forwardRef` + `onClick`을 지원해야 합니다
