# EmotionHeatmap

한 달치 감정 기록을 GitHub 잔디처럼 색 강도로 시각화하는 컴포넌트입니다. 매일의 변화를 한눈에 보여주고, 셀을 클릭하면 그 날의 상세로 이동시키는 데 유용합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 감정/기분 월간 리포트
- 스트레스/수면 점수 트렌드
- 챌린지 30일 시각화 (StreakCard로 7일은 부족할 때)

### 자주 하는 실수
- entries에 없는 날짜는 자동으로 빈 셀(점선) — 직접 0 단계 데이터를 넣지 말 것 (treatZeroAsEmpty 기본 true)
- colors는 5개 필수, 옅음→짙음 순서

## 사용법

```tsx
import { EmotionHeatmap } from "@nudge-eap/react";

<EmotionHeatmap
  month="2026-05"
  entries={[
    { date: "2026-05-02", level: 3 },
    { date: "2026-05-05", level: 4 },
    { date: "2026-05-12", level: 1 },
  ]}
  onCellClick={(date) => router.push(`/journal/${date}`)}
/>

// 워밍 톤 (스트레스 강도용)
<EmotionHeatmap
  entries={...}
  colors={["#FFE9C4", "#FFD088", "#FFB44D", "#F08020", "#C25B0E"]}
  legendLabels={{ low: "차분", high: "활기" }}
/>
```

## Props

| Prop               | 타입                                                       | 기본값                | 설명                              |
| ------------------ | ---------------------------------------------------------- | --------------------- | --------------------------------- |
| `month`            | `string`                                                   | 현재 월               | `YYYY-MM`                         |
| `entries`          | `EmotionEntry[]`                                           | —                     | `{ date, level: 0~4 }` 배열       |
| `title`            | `ReactNode`                                                | `"{년}년 {월}월"`     | 카드 제목                         |
| `onCellClick`      | `(date, level) => void`                                    | —                     | 셀 클릭 콜백                      |
| `treatZeroAsEmpty` | `boolean`                                                  | `true`                | 0 단계를 빈 셀로 표시             |
| `colors`           | `[string, string, string, string, string]`                 | 푸른 5단계            | 색상 (옅음→짙음)                  |
| `legendLabels`     | `{ low: string; high: string }`                            | `{low:'낮음', high:'좋음'}` | 범례 라벨                  |
| `weekStartsOn`     | `0 \| 1`                                                   | `0`                   | 0=일요일, 1=월요일                |
