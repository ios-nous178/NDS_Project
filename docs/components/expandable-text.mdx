# ExpandableText

긴 본문을 N줄까지만 보여주고 더보기 버튼을 자동으로 노출하는 컴포넌트입니다. 텍스트가 짧아 클램프가 필요 없으면 토글 버튼이 자동으로 숨겨집니다.

## 사용 가이드

### 이럴 때 사용하세요
- 일기 본문 미리보기
- 콘텐츠 카드 설명
- 약관 / 안내문 미리보기
- 리뷰 / 후기

### 자주 하는 실수
- 줄 수 측정은 `line-height` 기반 — 본문 안에 다른 폰트 사이즈가 섞이면 정확도 떨어짐. 단일 톤 텍스트 위주로 사용
- `hideCollapse=true`면 한 번 펼친 후 다시 못 접음 (약관 한 번 보면 끝나는 케이스용)

## 사용법

```tsx
import { ExpandableText } from "@nudge-eap/react";

<ExpandableText>{longBody}</ExpandableText>

// 5줄까지 표시
<ExpandableText lines={5}>{body}</ExpandableText>

// 라벨 커스텀
<ExpandableText expandLabel="전문 보기" collapseLabel="요약">
  {terms}
</ExpandableText>

// 한 번 펼치면 다시 못 접기
<ExpandableText hideCollapse>{body}</ExpandableText>
```

## Props

| Prop               | 타입                          | 기본값       | 설명                              |
| ------------------ | ----------------------------- | ------------ | --------------------------------- |
| `children`         | `ReactNode`                   | —            | 본문                              |
| `lines`            | `number`                      | `3`          | 클램프 줄 수                      |
| `expandLabel`      | `string`                      | `"더보기"`   | 더보기 라벨                       |
| `collapseLabel`    | `string`                      | `"접기"`     | 접기 라벨                         |
| `hideCollapse`     | `boolean`                     | `false`      | 펼친 후 토글 숨김                 |
| `expanded`         | `boolean`                     | —            | 외부 제어 (controlled, 선택)      |
| `onExpandedChange` | `(v: boolean) => void`        | —            | 펼침 상태 변경 콜백               |
