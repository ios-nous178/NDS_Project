# FAB

화면 우측/좌측/중앙 하단에 떠 있는 **주요 액션 버튼**입니다. 일기 작성, 새 기록, 빠른 만들기처럼 "지금 화면에서 가장 중요한 한 액션"에 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 리스트 화면에서 "새로 작성" 같은 가장 흔한 액션
- 캘린더/감정 화면에서 "오늘 기록 추가"
- 한 화면에 **딱 한 개**만 등장하는 액션

### 이럴 때 사용하지 마세요
- 한 화면에 두 개 이상 → 위계 붕괴. 그럴 땐 일반 Button.
- StickyBottom CTA가 이미 있는 화면 → FAB와 CTA가 겹치면 모바일 안전영역이 무너짐

### 자주 하는 실수
- `position="bottom-right"`로 두면 `position: fixed`가 적용됨 — Storybook 등에서 잘리거나 다른 화면에 떠다닐 수 있음. static으로 두고 부모 div에서 fixed 처리 가능
- 라벨 없이 사용할 때는 `aria-label` 필수

## 사용법

```tsx
import { FAB } from "@nudge-design/react";
import { PlusIcon } from "@nudge-design/icons";

// 기본 (아이콘만, 우측 하단 fixed)
<FAB icon={<PlusIcon />} aria-label="추가" />

// Extended (라벨 포함)
<FAB icon={<EditIcon />} label="새 글 작성" />

// 색상 / 위치
<FAB icon={<PlusIcon />} color="secondary" position="bottom-center" />

// 화면 흐름 안에 (테스트/스토리북에서)
<FAB icon={<PlusIcon />} position="static" />

// 하단바와 겹치지 않도록 offset 조정
<FAB icon={<PlusIcon />} offset={80} />
```
