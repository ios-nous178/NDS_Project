# FieldActionRow

입력 필드 + 액션 버튼 조합 레이아웃 컴포넌트입니다. 휴대폰 인증, 인증번호 입력 등 "입력 → 확인" 패턴에 특화되어 있습니다.

```tsx
import { FieldActionRow } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- **입력 필드 + 버튼**을 한 줄로 배치할 때 (인증번호 발송, 쿠폰 적용, 타이머 등)
- Input과 Button의 높이/정렬을 자동으로 맞춰야 할 때
- 에러 상태, 타이머 등 부가 정보를 입력 줄 아래에 표시할 때

### 이럴 때 사용하지 마세요

- 버튼 없이 입력만 필요할 때 → [Input](/docs/components/input)을 사용하세요
- 검색 기능 → [SearchInput](/docs/components/search-input)을 사용하세요
- 복잡한 폼 레이아웃 (여러 필드 + 여러 버튼) → 개별 Input + Button 조합을 사용하세요

### 자주 하는 실수

- 복잡한 폼 전체를 FieldActionRow로 구성 — 이 컴포넌트는 "입력 1개 + 버튼 1개" 패턴에 최적화되어 있습니다

## Flat API

### 휴대폰 인증

```tsx
<FieldActionRow
  field={
    <input
      type="tel"
      value={phone}
      onChange={(e) => setPhone(e.target.value)}
      placeholder="숫자만 입력해 주세요."
    />
  }
  action={
    <button type="button" disabled={phone.length < 10} onClick={handleSend}>
      인증 번호 받기
    </button>
  }
  actionTone="outline"
/>
```

### 인증번호 입력 + 타이머

```tsx
<FieldActionRow
  field={<input type="number" placeholder="인증번호 6자리" />}
  action={<button type="button">확인</button>}
  actionTone="solid"
  timer="02:58"
  helperText="이메일 또는 문자로 전송된 인증번호를 입력해주세요"
/>
```

### 에러 상태

```tsx
<FieldActionRow
  field={<input type="tel" value="010-0000-0000" />}
  action={<button disabled>인증 번호 받기</button>}
  error
  helperText="휴대폰 번호를 확인해주세요."
/>
```

### 타이머 만료

```tsx
<FieldActionRow
  field={<input type="number" placeholder="인증번호 6자리" />}
  action={<button disabled>확인</button>}
  timer="00:00"
  timerExpired
  error
  helperText="인증 시간이 만료되었습니다."
/>
```

## Compound API

```tsx
<FieldActionRow.Root>
  <FieldActionRow.Row>
    <FieldActionRow.Field error={hasError}>
      <input type="tel" placeholder="숫자만 입력해 주세요." />
      <FieldActionRow.Timer expired={isExpired}>02:58</FieldActionRow.Timer>
    </FieldActionRow.Field>
    <FieldActionRow.Action tone="outline">
      <button type="button">인증 번호 받기</button>
    </FieldActionRow.Action>
  </FieldActionRow.Row>
  <FieldActionRow.Helper error={hasError}>도움 텍스트</FieldActionRow.Helper>
</FieldActionRow.Root>
```

## Flat API Props

| Prop           | 타입                      | 기본값      | 설명                                  |
| -------------- | ------------------------- | ----------- | ------------------------------------- |
| `field`        | `ReactNode`               | —           | 입력 필드 (input 또는 Input 컴포넌트) |
| `action`       | `ReactNode`               | —           | 액션 버튼                             |
| `actionTone`   | `"outline" \| "solid"`    | `"outline"` | 버튼 스타일                           |
| `timer`        | `ReactNode`               | —           | 타이머 표시 (입력 필드 우측)          |
| `timerExpired` | `boolean`                 | `false`     | 타이머 만료 상태                      |
| `error`        | `boolean`                 | `false`     | 에러 상태                             |
| `helperText`   | `string`                  | —           | 도움/에러 텍스트                      |
| `success`      | `boolean`                 | `false`     | 성공 상태                             |
| `slotProps`    | `FieldActionRowSlotProps` | —           | 슬롯 프롭                             |

## Compound 컴포넌트

| 컴포넌트                | 역할                     |
| ----------------------- | ------------------------ |
| `FieldActionRow.Root`   | 루트                     |
| `FieldActionRow.Row`    | 입력 + 버튼 가로 행      |
| `FieldActionRow.Field`  | 입력 필드 래퍼           |
| `FieldActionRow.Timer`  | 타이머 표시 (Field 내부) |
| `FieldActionRow.Action` | 버튼 래퍼 (tone 지원)    |
| `FieldActionRow.Helper` | 도움/에러/성공 텍스트    |
