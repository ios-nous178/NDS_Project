# FileUpload

진단서·프로필 이미지 등의 파일 업로드 컴포넌트입니다. 클릭과 끌어다 놓기를 모두 지원하고, 거부된 파일은 `onReject`로 분리해서 처리합니다.

## 사용법

```tsx
import { FileUpload } from "@nudge-design/react";

const [files, setFiles] = useState<File[]>([]);
const [err, setErr] = useState<string>();

<FileUpload
  value={files}
  onValueChange={(next) => {
    setErr(undefined);
    setFiles(next);
  }}
  accept="image/*"
  maxSize={5 * 1024 * 1024}
  multiple={false}
  onReject={(rejected, reason) => {
    if (reason === "size") setErr(`크기 초과: ${rejected.map((f) => f.name).join(", ")}`);
  }}
  description="JPG, PNG · 최대 5MB"
  errorMessage={err}
/>
```

## 가이드

- 제어 컴포넌트 — 부모에서 `useState`로 관리.
- `maxSize`는 **bytes** 단위. `10 * 1024 * 1024` = 10MB.
- `accept`는 브라우저 힌트일 뿐이라 실제 다른 파일도 들어올 수 있음 — **서버에서 한 번 더 검증** 필요.
- `multiple={false}`에서 두 번째 파일을 드롭하면 첫 번째를 덮어씀.
- 거부 사유는 `errorMessage`로 지속 표시하는 게 Toast보다 명확 (사용자가 다시 시도하기 쉬움).
