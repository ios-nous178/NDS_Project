# 전체 카탈로그

`@nudge-design/react`가 제공하는 모든 컴포넌트를 한 페이지에서 확인할 수 있도록 정리한 페이지입니다.
**카테고리 칩으로 좁히고, 이름·설명·활용 키워드로 검색하고, "Figma Synced"만 필터링**할 수 있습니다.

> 실제 동작과 인터랙션은 Storybook의 [`Foundations / All Components`](/storybook/?path=/docs/foundations-all-components--all-search-filter) 카탈로그에서 라이브 미니 프리뷰로 확인할 수 있어요.
> 컴포넌트 하나의 props와 상태 매트릭스를 깊게 보고 싶다면 카드를 클릭해 개별 문서로 이동하세요.

## 사용 팁

- **빠르게 컴포넌트 이름이 떠오르지 않을 때** — 키워드 검색이 가장 빠릅니다. 예: `차트`, `시간`, `상담`.
- **Figma 정합 상태를 확인할 때** — "Figma Synced만" 토글로 검증 완료된 컴포넌트만 봅니다.
- **카테고리 의미**
  - **일반** — Button, Badge, Card 같은 범용 프리미티브
  - **입력** — Input, Select, Checkbox 등 폼 입력 컴포넌트
  - **오버레이** — Modal, BottomSheet, Popup 등 portal 기반 UI
  - **피드백** — Toast, Snackbar, EmptyState, Progress
  - **내비게이션** — Tabs, FilterBar
  - **레이아웃** — PageHeader
  - **데이터 표시** — DataTable, StatsTable, SummaryCard
  - **데이터 시각화** — Chart, Sparkline
  - **도메인** — ProductCard, ChatBubble, PopularPosts, TrendingKeywords 등 도메인 특화 컴포넌트

> 브랜드별 구현·Figma 정합 현황을 표로 보고 싶다면 [브랜드별 컴포넌트 커버리지](/components/brand-coverage)를 참고하세요.
