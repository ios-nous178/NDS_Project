# GreetingHeader

홈 화면 진입 시 인삿말과 첫 인터랙션(MoodSelector 등)을 자연스럽게 묶는 카드입니다.

## 사용 가이드

### 이럴 때 사용하세요
- 홈 화면 최상단
- 마이페이지 인삿말

### 자주 하는 실수
- `greeting`은 시간대 자동 인식 X — 외부에서 "좋은 아침이에요"/"좋은 오후예요" 등 직접 결정
- 호칭(`name`)은 컴포넌트가 자동으로 `"{name}님"` 처리

## 사용법

```tsx
import { GreetingHeader, MoodSelector } from "@nudge-design/react";

<GreetingHeader
  name="민지"
  greeting="좋은 아침이에요"
  question="오늘 기분은 어때요?"
  trailing={<IconButton aria-label="알림" icon={<BellIcon />} />}
  actions={<MoodSelector value={mood} onValueChange={setMood} />}
/>

// primary 톤
<GreetingHeader name="민지" tone="primary" />
```

## Props

| Prop       | 타입                           | 기본값           | 설명                              |
| ---------- | ------------------------------ | ---------------- | --------------------------------- |
| `name`     | `string`                       | —                | 사용자 호칭 (자동 `"{name}님"`)   |
| `greeting` | `string`                       | `"안녕하세요"`   | 인삿말                            |
| `question` | `ReactNode`                    | —                | 메인 질문/메시지                  |
| `trailing` | `ReactNode`                    | —                | 우측 슬롯 (Avatar/IconButton 등)  |
| `actions`  | `ReactNode`                    | —                | 하단 액션 (MoodSelector 등)       |
| `tone`     | `"default" \| "primary"`       | `"default"`      | 배경 톤                           |
