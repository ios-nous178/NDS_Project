# IconButton

아이콘만 보이는 정사각 버튼입니다. 4가지 사이즈(`x-large` 36 / `large` 32 / `medium` 28 / `small` 24px)를 제공하며, `aria-label`이 필수입니다. 텍스트가 함께 보여야 한다면 `Button` 또는 `TextButton`을 사용하세요.

```tsx
import { IconButton } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 더보기(⋯), 닫기(×), 좋아요(♥) 등 **단일 아이콘만으로 의미가 명확한 액션**
- 툴바·헤더에서 **공간을 적게 차지**해야 할 때
- 같은 패턴이 반복되어 라벨이 오히려 노이즈가 되는 액션

### 이럴 때 사용하지 마세요

- 텍스트가 같이 보여야 의미가 명확할 때 → [Button](/components/button)을 사용하세요
- 토글 상태가 필요한 좋아요/북마크 → `LikeButton` 같은 도메인 컴포넌트를 검토하세요
- 페이지 이동만 수행할 때 → `<a>` 태그 또는 라우터 Link를 사용하세요

### 자주 하는 실수

- `aria-label` 누락 — 스크린 리더가 버튼의 의미를 알 수 없습니다. **필수 prop입니다**
- 모양만 비슷한 아이콘으로 닫기/뒤로가기를 동시에 표현 — 명확한 의미가 다른 아이콘으로 분리하세요

## 기본 사용

```tsx
import { IconButton } from "@nudge-design/react";
import { SearchIcon } from "@nudge-design/icons";

<IconButton aria-label="검색" icon={<SearchIcon size={20} />} size="medium" />;
```

### 사이즈

```tsx
<IconButton aria-label="검색" icon={<SearchIcon />} size="x-large" />
<IconButton aria-label="검색" icon={<SearchIcon />} size="large" />
<IconButton aria-label="검색" icon={<SearchIcon />} size="medium" />
<IconButton aria-label="검색" icon={<SearchIcon />} size="small" />
```

| Size       | Box   | Icon  | 사용 시점                       |
| ---------- | ----- | ----- | ------------------------------- |
| `x-large`  | 36px  | 28px  | 손가락 타깃이 중요한 모바일     |
| `large`    | 32px  | 24px  | 기본값                          |
| `medium`   | 28px  | 20px  | 데스크톱 툴바 / 리스트 라인     |
| `small`    | 24px  | 16px  | 인라인·캡션 옆 등 매우 작은 자리 |

### DropdownMenu 트리거로 쓰기

```tsx
<DropdownMenu items={[...]}>
  <IconButton aria-label="더보기" icon={<MoreIcon />} />
</DropdownMenu>
```

## Props

| Prop         | 타입            | 기본값     | 설명                          |
| ------------ | --------------- | ---------- | ----------------------------- |
| `icon`       | `ReactNode`     | —          | 표시할 아이콘 (필수)          |
| `size`       | `IconButtonSize`| `"large"`  | 버튼 크기                     |
| `aria-label` | `string`        | —          | 접근성 라벨 (필수)            |
| `disabled`   | `boolean`       | `false`    | 비활성화                      |
| `type`       | `"button" \| "submit" \| "reset"` | `"button"` | 네이티브 버튼 타입            |
| `className`  | `string`        | —          | 루트 className                |
| `style`      | `CSSProperties` | —          | 루트 인라인 스타일            |

`<button>` HTML 속성을 모두 지원하며 `forwardRef`로 ref 전달이 가능합니다.

**IconButtonSize**: `"x-large"` | `"large"` | `"medium"` | `"small"`

## data-slot 구조

```
button [data-slot="root"] [data-size="x-large" | "large" | "medium" | "small"]
└── { icon }
```

내부 SVG는 CSS 변수(`--nds-icon-button-icon-size`)로 자동 사이징됩니다 — 아이콘 컴포넌트에서 `size` prop을 지정하지 않아도 박스에 맞게 그려집니다.

---

## CSS 변수

| 변수                                  | 기본값                        | 설명                          |
| ------------------------------------- | ----------------------------- | ----------------------------- |
| `--nds-icon-button-size`              | `32px` (size별 자동 설정)     | 버튼 정사각 크기              |
| `--nds-icon-button-icon-size`         | `24px` (size별 자동 설정)     | 내부 SVG 크기                 |
| `--nds-icon-button-color`             | `cv.icon.default`             | 아이콘 색상                   |
| `--nds-icon-button-hover-bg`          | `neutral[100]` (`#F5F5F5`)    | hover 배경색 (Figma 171:8590) |
| `--nds-icon-button-focus-ring-color`  | `cv.primary.main`             | focus-visible outline 색상    |
| `--nds-icon-button-focus-ring-offset` | `2px`                         | focus-visible outline offset  |

```tsx
// 좋아요 아이콘만 빨갛게
<IconButton
  aria-label="좋아요"
  icon={<HeartIcon />}
  style={{ "--nds-icon-button-color": "#ef4444" } as React.CSSProperties}
/>
```

## 접근성

- `aria-label` 필수 — 미지정 시 TypeScript 단계에서 에러 (`IconButtonProps`가 강제)
- `type="button"` 기본값으로 form 내 의도치 않은 submit 방지
- `:focus`에서는 outline 제거, `:focus-visible`에서만 `2px solid` 링이 표시됨 (키보드 사용자에게만 노출)
- `disabled` 시 `cursor: default` + 색상이 `cv.text.disabled`로 변경
- `forwardRef` 지원 (`React.forwardRef`)
