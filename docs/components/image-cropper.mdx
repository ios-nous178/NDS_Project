# ImageCropper

이미지를 원형/사각형 영역으로 자르는 컴포넌트입니다. 사용자는 드래그로 위치를 조정하고 슬라이더로 줌할 수 있습니다. ref로 `toDataURL()`을 호출해 잘린 PNG를 추출합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 프로필 사진 등록/변경
- 커버 이미지 자르기

### 자주 하는 실수
- 외부 이미지(`https://`)는 CORS 설정이 필요 — `crossOrigin="anonymous"`는 이미 들어가 있지만, 서버가 CORS 헤더를 안 보내면 dataURL이 비어나오거나 타이트닝
- `outputSize`를 너무 작게 잡으면 화질 저하 — 일반적으로 200~400 권장

## 사용법

```tsx
import { ImageCropper, type ImageCropperHandle } from "@nudge-design/react";

const ref = useRef<ImageCropperHandle>(null);

<ImageCropper ref={ref} src={imageUrl} shape="circle" />

<Button
  onClick={() => {
    const url = ref.current?.toDataURL(200);
    if (url) uploadProfileImage(url);
  }}
>
  저장
</Button>
```
