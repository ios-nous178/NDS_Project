# LikertScale

5점 또는 N점 리커트 척도. PHQ-9, GAD-7 같은 정형 임상 척도에 사용합니다. 가벼운 의견 수렴은 `VotePoll` 검토.

## 사용법

```tsx
import { LikertScale } from "@nudge-eap/react";

const [v, setV] = useState<string | number>(3);

<LikertScale
  value={v}
  onValueChange={setV}
  options={[{ value: 1 }, { value: 2 }, { value: 3 }, { value: 4 }, { value: 5 }]}
  startLabel="전혀 아니다"
  endLabel="매우 그렇다"
/>
```


> 가벼운 의견 수렴은 VotePoll. LikertScale은 정형 설문 척도.
