# List

줄 단위 단순 리스트 컨테이너. `plain` · `card` · `divided` 변형을 지원합니다. 복합 카드 그리드는 `Card` 사용.

## 사용법

```tsx
import { List, ListItem } from "@nudge-eap/react";

<List variant="divided">
  <ListItem
    title="알림 설정"
    description="푸시 · 이메일"
    trailing={<ChevronRightIcon />}
    onSelect={() => navigate("/settings/notifications")}
  />
  <ListItem title="결제 수단" trailing={<ChevronRightIcon />} onSelect={() => {}} />
</List>
```


> 복합 카드 그리드는 Card 또는 도메인 카드. List는 줄 단위 단순 리스트.

## 인벤토리

전체 컴포넌트 카탈로그는 [컴포넌트 인벤토리](/docs/components/inventory)에서 확인하세요. Storybook에서는 [Components/List](http://localhost:6006/?path=/docs/components-list--docs)로 진입할 수 있습니다.
