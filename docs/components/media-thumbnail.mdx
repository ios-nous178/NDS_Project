# MediaThumbnail

콘텐츠 썸네일·일반 이미지를 표준화하는 컴포넌트입니다. **`Avatar`(사람 얼굴/이니셜) 와 다르게** 콘텐츠 이미지·검사 결과 이미지·기타 일반 이미지에 사용합니다.

## 사용법

```tsx
import { MediaThumbnail } from "@nudge-eap/react";

// 콘텐츠 카드 썸네일
<MediaThumbnail
  src={post.cover}
  alt={post.title}
  aspectRatio="16/9"
  rounded="md"
/>

// 리스트 썸네일
<MediaThumbnail
  src={item.thumb}
  alt=""
  aspectRatio="1/1"
  width={64}
  rounded="md"
/>

// fallback + placeholder
<MediaThumbnail
  src={profile.image}
  alt={profile.name}
  aspectRatio="1/1"
  width={120}
  rounded="pill"
  fallbackSrc="/images/default-avatar.png"
/>
```

## Props

| Prop          | 타입                                          | 기본값   | 설명                                  |
| ------------- | --------------------------------------------- | -------- | ------------------------------------- |
| `src`         | `string`                                      | —        | 이미지 URL                            |
| `alt`         | `string`                                      | —        | 대체 텍스트 (장식이면 `""`)            |
| `aspectRatio` | `string \| number`                            | —        | 종횡비 (`"16/9"`, `"1/1"`, `1.5` …)   |
| `width`       | `number \| string`                            | —        | 너비 (CSS 값)                         |
| `fit`         | `"cover" \| "contain"`                        | `"cover"` | object-fit                            |
| `rounded`     | `"none" \| "sm" \| "md" \| "lg" \| "pill"`    | `"md"`   | 모서리 둥글기                         |
| `fallbackSrc` | `string`                                      | —        | 로드 실패 시 대체 src 1회 시도        |
| `placeholder` | `ReactNode`                                   | 기본 아이콘 | 로딩/에러 시 자리 표시                |
| `lazy`        | `boolean`                                     | `true`   | loading="lazy"                        |

`<img>`의 다른 prop들 (`onLoad`, `onError`, `srcSet` 등)도 그대로 전달 가능.

## 가이드

- `Avatar`는 사람 얼굴/이니셜 전용 — 콘텐츠/검사 결과/일반 이미지는 모두 `MediaThumbnail`.
- `aspectRatio`를 명시하지 않으면 부모 너비에 따라 의도치 않은 높이가 잡힐 수 있음. 카드/그리드에서는 명시 권장.
- `fallbackSrc`는 `onError` 시 한 번만 시도 → 그래도 실패하면 자체 placeholder 노출 (무한 루프 방지).
- `rounded="pill"`은 정사각형(`aspectRatio="1/1"`)과 함께 써야 자연스러움.
