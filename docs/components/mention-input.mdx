# MentionInput

`@`(또는 커스텀 트리거)을 입력하면 사용자 목록을 보여주고 멘션을 삽입합니다. 단체 상담, 그룹 채팅, 댓글에 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 단체 상담 / 그룹 채팅 댓글
- 노트/일기에 상담사 태그
- 키워드 자동완성 (#로 트리거)

### 자주 하는 실수
- `users`는 전체 목록 — 컴포넌트가 query로 자동 필터. 외부에서 미리 필터링하지 말 것
- 멘션 삽입 후 저장된 텍스트는 plain text (`@김민지`) — 멘션을 사용자 ID로 보존하려면 별도 파싱 필요

## 사용법

```tsx
import { MentionInput } from "@nudge-design/react";

<MentionInput
  label="댓글"
  value={comment}
  onValueChange={setComment}
  users={[
    { key: "1", name: "김민지", description: "상담사", avatar: <Avatar name="김민지" /> },
    ...
  ]}
  onMention={(u) => trackMention(u.key)}
/>

// 해시태그 자동완성
<MentionInput trigger="#" users={tagSuggestions} ... />
```

## Props

| Prop            | 타입                          | 기본값  | 설명                              |
| --------------- | ----------------------------- | ------- | --------------------------------- |
| `value`         | `string`                      | —       | 입력값                            |
| `onValueChange` | `(v: string) => void`         | —       | 변경 콜백                         |
| `users`         | `MentionUser[]`               | —       | 전체 사용자 목록 (자동 필터)      |
| `onMention`     | `(u: MentionUser) => void`    | —       | 멘션 선택 콜백                    |
| `label`         | `ReactNode`                   | —       | 라벨                              |
| `placeholder`   | `string`                      | —       | 자리표시자                        |
| `helperText`    | `ReactNode`                   | —       | 헬퍼/에러                         |
| `error`         | `boolean`                     | `false` | 에러                              |
| `disabled`      | `boolean`                     | `false` | 비활성화                          |
| `trigger`       | `string`                      | `"@"`   | 멘션 트리거 문자                  |

### `MentionUser`

| Prop          | 타입         | 설명                  |
| ------------- | ------------ | --------------------- |
| `key`         | `string`     | 고유 키               |
| `name`        | `string`     | 표시 이름 (`@<name>`) |
| `avatar`      | `ReactNode`  | 아바타 (선택)         |
| `description` | `string`     | 보조 설명 (선택)      |

## 키보드

| 키          | 동작                  |
| ----------- | --------------------- |
| `↓` / `↑`   | 사용자 이동           |
| `Enter`/`Tab` | 멘션 삽입           |
| `Escape`    | 드롭다운 닫기         |
