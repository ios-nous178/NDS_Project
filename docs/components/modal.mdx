# Modal

대화형 모달 다이얼로그 컴포넌트입니다. 포커스 트랩, ESC 닫기, 스크롤 잠금 등 접근성 기능이 내장되어 있습니다. Flat API와 Compound API를 모두 지원합니다.

```tsx
import { Modal } from "@nudge-eap/react";
```

## Flat API

props만으로 빠르게 모달을 구성합니다.

### 확인 모달

```tsx
const [open, setOpen] = useState(false);

<Modal
  open={open}
  onClose={() => setOpen(false)}
  title="알림"
  onConfirm={(close) => {
    handleAction();
    close();
  }}
>
  작업을 진행하시겠습니까?
</Modal>
```

### 확인 + 취소 모달

```tsx
<Modal
  open={open}
  onClose={() => setOpen(false)}
  title="삭제 확인"
  confirmText="삭제"
  closeText="취소"
  onConfirm={(close) => {
    handleDelete();
    close();
  }}
>
  이 항목을 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.
</Modal>
```

### 닫기 버튼이 있는 모달

```tsx
<Modal
  open={open}
  onClose={() => setOpen(false)}
  title="상세 정보"
  closable
  showModalButtonGroup={false}
>
  상세 내용이 여기에 표시됩니다.
</Modal>
```

## Compound API

각 영역을 직접 조합하여 자유로운 레이아웃을 구성합니다.

```tsx
<Modal.Root open={open} onClose={() => setOpen(false)}>
  <Modal.Overlay />
  <Modal.Content maxWidth={400}>
    <Modal.Header title="프로필 편집" closable />
    <Modal.Body>
      <Input label="이름" placeholder="이름을 입력하세요" />
      <Input label="이메일" placeholder="이메일을 입력하세요" />
    </Modal.Body>
    <Modal.Footer
      onConfirm={(close) => {
        handleSave();
        close();
      }}
      confirmText="저장"
      onClose={() => setOpen(false)}
      closeText="취소"
    />
  </Modal.Content>
</Modal.Root>
```

### 커스텀 Footer

`Modal.Footer`에 `children`을 전달하면 자유로운 레이아웃을 구성할 수 있습니다.

```tsx
<Modal.Footer>
  <Button variant="outlined-sub" size="md" onClick={() => setOpen(false)}>
    나중에
  </Button>
  <Button variant="soft" size="md" onClick={handleSave}>
    임시 저장
  </Button>
  <Button variant="solid" size="md" onClick={handleSubmit}>
    제출
  </Button>
</Modal.Footer>
```

## Flat API Props

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `open` | `boolean` | — | 모달 표시 여부 (필수) |
| `onClose` | `() => void` | — | 닫기 콜백 |
| `title` | `string` | — | 헤더 타이틀 |
| `closable` | `boolean` | `false` | X 닫기 버튼 표시 |
| `children` | `ReactNode` | — | 본문 내용 (필수) |
| `mask` | `boolean` | `true` | 오버레이 표시 |
| `isMaskClose` | `boolean` | `true` | 오버레이 클릭으로 닫기 |
| `onConfirm` | `(onClose: () => void) => void` | — | 확인 콜백 |
| `confirmText` | `string` | `"확인"` | 확인 버튼 텍스트 |
| `closeText` | `string` | `"취소"` | 취소 버튼 텍스트 |
| `showModalButtonGroup` | `boolean` | `true` | Footer 버튼 영역 표시 |
| `maxWidth` | `number` | `320` | 모달 최대 너비 (px) |
| `slotProps` | `ModalSlotProps` | — | 내부 슬롯별 props 전달 |

## data-slot 구조

```
div [data-slot="root"]
├── div [data-slot="overlay"]
└── div [data-slot="content"]  role="dialog" aria-modal="true"
    ├── div [data-slot="header"]  [data-has-title]
    │   ├── h3 (title)
    │   └── button [data-slot="close"]
    ├── div [data-slot="body"]
    └── div [data-slot="footer"]  [data-layout] [data-has-both-actions]
        ├── button (cancel)
        └── button (confirm)
```

## 접근성

- `role="dialog"`, `aria-modal="true"` 자동 적용
- `aria-labelledby`가 Header 타이틀과 자동 연결
- `aria-describedby`가 Body와 자동 연결
- ESC 키로 모달 닫기
- Tab 키 포커스 트랩 (모달 내부에서만 순환)
- 모달 열릴 때 첫 번째 포커스 가능 요소에 자동 포커스
- `data-modal-initial-focus="true"` 속성으로 초기 포커스 대상 지정 가능
- 모달 닫힐 때 이전 포커스 요소로 자동 복귀
- body 스크롤 자동 잠금 및 복원
