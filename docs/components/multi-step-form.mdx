# MultiStepForm

회원가입, EAP 가입, 심리검사 같은 단계별 플로우를 관리하는 컨테이너입니다. `Stepper`(단계 인디케이터)와는 분리된 **로직 + UI 셸**을 함께 제공합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 회원가입 / 본인 인증 다단계
- 검사 단계 (intro → 동의 → 검사 → 결과)
- 챌린지/목표 설정 마법사

### 이럴 때 사용하지 마세요
- 단순 진행 표시만 필요 → `Stepper`
- 한 화면 폼 → 일반 form

### 자주 하는 실수
- `canProceed`는 동기 boolean. 비동기 검증이 필요하면 외부 state로 결과를 관리하고 그 결과를 boolean으로 전달
- 마지막 단계의 "다음" 버튼은 자동으로 `submitLabel`로 바뀌고 `onSubmit`을 호출

## 사용법

```tsx
import { MultiStepForm } from "@nudge-eap/react";

const [name, setName] = useState("");
const [phone, setPhone] = useState("");

<MultiStepForm
  steps={[
    {
      key: "name",
      title: "이름을 알려주세요",
      description: "상담 시 호칭으로 사용해요.",
      canProceed: name.length >= 2,
      content: <Input value={name} onValueChange={setName} />,
    },
    {
      key: "phone",
      title: "연락처를 입력해주세요",
      canProceed: /^010\d{8}$/.test(phone),
      content: <PhoneInput ... />,
    },
  ]}
  onSubmit={async () => {
    await api.signUp({ name, phone });
  }}
/>
```

내부 단계에서 `useMultiStepForm()` 훅으로 `next/prev/goTo`에 접근할 수도 있습니다.

```tsx
import { useMultiStepForm } from "@nudge-eap/react";

const { next, current, total } = useMultiStepForm();
```

## Props

| Prop              | 타입                                | 기본값        | 설명                                |
| ----------------- | ----------------------------------- | ------------- | ----------------------------------- |
| `steps`           | `MultiStepFormStep[]`               | —             | 단계 목록                           |
| `current`         | `number`                            | —             | 현재 인덱스 (controlled, 선택)      |
| `onCurrentChange` | `(i: number) => void`               | —             | 단계 변경 콜백                      |
| `onSubmit`        | `() => void \| Promise<void>`       | —             | 마지막 단계 제출                    |
| `indicator`       | `"progress" \| "steps" \| "none"`   | `"progress"`  | 진행 표시 종류                      |
| `nextLabel`       | `string`                            | `"다음"`      | 다음 버튼                           |
| `prevLabel`       | `string`                            | `"이전"`      | 이전 버튼                           |
| `submitLabel`     | `string`                            | `"완료"`      | 마지막 단계 버튼                    |
| `submitting`      | `boolean`                           | `false`       | 제출 중 (다음 버튼 비활성)          |

### `MultiStepFormStep`

| Prop          | 타입         | 설명                              |
| ------------- | ------------ | --------------------------------- |
| `key`         | `string`     | 고유 키                           |
| `title`       | `string`     | 단계 제목                         |
| `description` | `string`     | 단계 설명 (선택)                  |
| `content`     | `ReactNode`  | 단계 콘텐츠                       |
| `canProceed`  | `boolean`    | 다음 단계 진행 가능 여부          |
