# NotificationItem

알림 화면의 한 줄을 표현하는 컴포넌트입니다. 종류(info/success/warning/error/system)에 따라 아이콘 톤이 자동 적용되고, 미읽음 표시 점·시간 라벨·본문 클램프(2줄)가 제공됩니다.

## 사용 가이드

### 이럴 때 사용하세요
- 알림 센터 / 푸시 히스토리
- 시스템 안내 리스트

### 자주 하는 실수
- 단순 toast/스낵바 용도가 아님 — 그건 `Toast`, `Snackbar`
- description은 자동 2줄 클램프. 더 길게 보여줘야 하면 클릭 시 디테일로 진입

## 사용법

```tsx
import { NotificationItem } from "@nudge-design/react";

<NotificationItem
  kind="success"
  title="14일 챌린지 완료!"
  description="감정 기록을 14일 동안 이어왔어요."
  time="1시간 전"
  unread
  onClick={openDetail}
/>

// 종류별 아이콘 자동
<NotificationItem kind="warning" title="복약 알림" time="오후 8:00" />
```
