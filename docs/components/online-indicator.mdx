# OnlineIndicator

상담사/사용자의 온라인 상태를 점으로 표현합니다. online은 펄스 애니메이션이 자동 적용됩니다.

## 사용 가이드

### 이럴 때 사용하세요
- 상담사 리스트 옆 상태 표시
- 아바타 우하단 점
- 그룹 채팅 참가자 상태

### 자주 하는 실수
- "online"은 펄스 애니메이션이 자동 — 별도 강조 효과 추가 X
- 아바타 위에 올릴 때는 부모를 `position: relative`로 두고 점은 `position: absolute`

## 사용법

```tsx
import { OnlineIndicator } from "@nudge-eap/react";

<OnlineIndicator status="online" showLabel />

// 라벨 없이 (아바타 위 등)
<OnlineIndicator status="busy" size={10} />

// 라벨 커스텀
<OnlineIndicator status="busy" showLabel label="상담 중" />
```

## Props

| Prop        | 타입                                              | 기본값  | 설명                              |
| ----------- | ------------------------------------------------- | ------- | --------------------------------- |
| `status`    | `"online" \| "away" \| "busy" \| "offline"`       | —       | 상태                              |
| `showLabel` | `boolean`                                         | `false` | 라벨 표시                         |
| `label`     | `string`                                          | 자동    | 라벨 오버라이드                   |
| `size`      | `number`                                          | `8`     | 점 크기 px                        |
