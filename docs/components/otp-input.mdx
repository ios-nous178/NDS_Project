# OtpInput

SMS·이메일 인증코드를 셀로 분리해서 입력받습니다. 자동 포커스 이동, 붙여넣기 분배, 숫자만 허용 등 OTP 입력의 표준 동작이 모두 들어있습니다.

## 사용법

```tsx
import { OtpInput } from "@nudge-eap/react";

const [code, setCode] = useState("");

<OtpInput
  length={6}
  value={code}
  onValueChange={setCode}
  onComplete={(value) => verifyOtp(value)}
  autoFocus
/>
```

## Props

| Prop            | 타입                          | 기본값  | 설명                            |
| --------------- | ----------------------------- | ------- | ------------------------------- |
| `length`        | `number`                      | `6`     | 자릿수                          |
| `value`         | `string`                      | —       | 현재 값 (숫자 문자열)           |
| `onValueChange` | `(value: string) => void`     | —       | 변경 콜백                       |
| `onComplete`    | `(value: string) => void`     | —       | length 만큼 채워졌을 때         |
| `disabled`      | `boolean`                     | `false` | 비활성화                        |
| `error`         | `boolean`                     | `false` | 에러 상태 (테두리 빨강)         |
| `autoFocus`     | `boolean`                     | `false` | 첫 셀 자동 포커스               |

## 가이드

- 숫자만 허용 — 영문/특수문자는 자동 필터링.
- iOS/Android의 SMS 자동 추출을 위해 첫 셀에만 `autoComplete="one-time-code"`가 적용됨.
- 인증 실패 시 자동 clear는 옵션 — 어떤 자리가 틀렸는지 알 수 없어 통째 clear가 보통 더 안전.
- Backspace 동작: 현재 셀에 값 있으면 비우고, 비어있으면 이전 셀로 이동 + 이전 셀 비움.
