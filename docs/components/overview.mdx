# 컴포넌트

NudgeEAP Design System이 제공하는 React 컴포넌트입니다. 모든 컴포넌트는 Flat API와 Compound API를 모두 지원하며, CSS 변수와 `slotProps`를 통해 스타일을 자유롭게 확장할 수 있습니다.

반복 UI를 기반으로 다음 추출 후보를 검토할 때는 [추출 후보 목록](/docs/components/candidates)과 [컴포넌트 인벤토리](/docs/components/inventory)를 함께 참고해주세요.

전체 설계 기준은 [디자인 원칙](/docs/guide/design-principles)에서, 실제 서비스 적용 시작점은 [페이지 교체 우선순위](/docs/guide/page-migration-priority)에서 함께 볼 수 있습니다.

## 공통 특징

### Hybrid API

모든 컴포넌트는 두 가지 사용 방식을 제공합니다.

```tsx
// Flat — 간단하게
<Input label="이름" placeholder="입력하세요" error errorMessage="필수 항목입니다" />

// Compound — 자유롭게
<Input.Root error>
  <Input.Label>이름</Input.Label>
  <Input.Wrapper>
    <Input.Field placeholder="입력하세요" />
    <Input.ClearButton onClick={handleClear} />
  </Input.Wrapper>
  <Input.Helper error>필수 항목입니다</Input.Helper>
</Input.Root>
```

### CSS 변수 기반 스타일링

모든 컴포넌트의 시각적 속성은 CSS 변수로 노출됩니다. 프로젝트 테마에 맞게 오버라이드할 수 있습니다.

```css
/* 프로젝트 전역 오버라이드 */
:root {
  --nds-button-radius: 12px;
  --nds-input-radius: 12px;
}
```

### data-slot 속성

각 내부 요소는 `data-slot` 속성을 가지며, 이를 통해 styled-components나 CSS에서 타겟팅할 수 있습니다.

```css
[data-slot="label"] {
  font-weight: 600;
}
```

### 접근성

- Modal/Popup: 포커스 트랩, ESC 닫기, `aria-modal`, `aria-labelledby` 내장
- Input: `label`과 `htmlFor` 자동 연결
- Button: `type="button"` 기본값, `disabled` 상태 처리
