# Pagination

콘텐츠 목록의 페이지를 전환하는 네비게이션 컴포넌트입니다.

## 사용법

```tsx
import { Pagination } from "@nudge-design/react";

const [page, setPage] = useState(1);

<Pagination
  page={page}
  totalPages={20}
  onPageChange={setPage}
/>

// 화살표 없이
<Pagination
  page={page}
  totalPages={10}
  onPageChange={setPage}
  showArrows={false}
/>

// 형제 페이지 수 조정
<Pagination
  page={page}
  totalPages={50}
  siblings={2}
  onPageChange={setPage}
/>
```
