# PhoneInput

국가 다이얼 코드 셀렉트 + 번호 입력이 합쳐진 컴포넌트입니다. 가입, 본인 인증, 연락처 변경 화면에서 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 회원가입 / 본인 인증 폼
- 프로필 연락처 수정

### 자주 하는 실수
- 국가 코드는 **ISO code**(KR, US 등)로 관리하고, 다이얼 코드/이름/국기는 `countries` 데이터에서 조회. 직접 `+82` 문자열을 state로 두지 말 것.
- 번호 포맷팅(하이픈 등)은 컴포넌트가 강제하지 않음 — 필요한 경우 onValueChange에서 마스킹

## 사용법

```tsx
import { PhoneInput } from "@nudge-design/react";

const [country, setCountry] = useState("KR");
const [phone, setPhone] = useState("");

<PhoneInput
  label="휴대폰 번호"
  countryCode={country}
  onCountryChange={setCountry}
  value={phone}
  onValueChange={setPhone}
  helperText="인증번호를 받을 번호를 입력해주세요"
/>

// 국가 목록 커스텀
<PhoneInput
  countries={[
    { code: "KR", name: "대한민국", dialCode: "+82", flag: "🇰🇷" },
    { code: "VN", name: "Việt Nam", dialCode: "+84", flag: "🇻🇳" },
  ]}
  ...
/>
```
