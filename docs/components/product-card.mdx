# ProductCard

상품 그리드/리스트의 한 칸을 표현하는 카드입니다. 가격은 `PriceTag` 컴포넌트를 슬롯으로 받습니다.

## 사용 가이드

### 이럴 때 사용하세요
- 굿즈/콘텐츠 패키지 그리드
- 상품 검색 결과
- 추천 상품 캐러셀

### 자주 하는 실수
- 가격은 `price` 슬롯에 `<PriceTag />`를 직접 넘김 — string 가격 넣지 말 것 (포맷 일관성 깨짐)
- 품절은 `soldOut`로 — 외곽 뱃지(`badge`)와 다름 (overlay)

## 사용법

```tsx
import { ProductCard, PriceTag } from "@nudge-design/react";

<ProductCard
  thumbnail="/products/book.jpg"
  description="마음챙김 굿즈"
  title="마음챙김 가이드북"
  price={<PriceTag amount={29000} originalAmount={39000} />}
  footer={<>★ 4.8 (124)</>}
  badge="NEW"
  onClick={openDetail}
/>

// 품절
<ProductCard ... soldOut />
```

## Props

| Prop           | 타입         | 기본값  | 설명                              |
| -------------- | ------------ | ------- | --------------------------------- |
| `thumbnail`    | `string`     | —       | 썸네일 이미지 URL                 |
| `thumbnailAlt` | `string`     | —       | alt                               |
| `badge`        | `ReactNode`  | —       | 좌상단 뱃지 (NEW/BEST/-30%)       |
| `title`        | `ReactNode`  | —       | 상품 제목 (2줄 클램프)            |
| `description`  | `ReactNode`  | —       | 보조 설명 (브랜드/카테고리)       |
| `price`        | `ReactNode`  | —       | 가격 슬롯 (PriceTag 권장)         |
| `footer`       | `ReactNode`  | —       | 푸터 (별점/리뷰수)                |
| `soldOut`      | `boolean`    | `false` | 품절 오버레이                     |
| `onClick`      | `() => void` | —       | 카드 클릭 (디테일 진입)           |
