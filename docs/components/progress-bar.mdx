# ProgressBar

작업 진행 상태를 시각적으로 표시하는 바 컴포넌트입니다.

## 사용법

```tsx
import { ProgressBar } from "@nudge-design/react";

// 기본
<ProgressBar value={65} />

// 크기 변형
<ProgressBar value={40} size="sm" />
<ProgressBar value={80} size="lg" />

// 색상 오버라이드
<ProgressBar value={100} color="#00A07C" />
```
