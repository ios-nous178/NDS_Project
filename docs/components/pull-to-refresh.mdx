# PullToRefresh

화면 상단을 아래로 당기면 새로고침을 트리거하는 패턴입니다. `onRefresh`가 Promise를 반환하면 자동으로 인디케이터가 종료됩니다.

## 사용 가이드

### 이럴 때 사용하세요
- 일기/콘텐츠/상담 리스트 새로고침
- 모바일 우선 화면

### 자주 하는 실수
- 컨테이너의 `scrollTop > 0`이면 트리거하지 않음 (의도적). 항상 최상단에서만 동작
- 데스크톱에선 패턴이 어색 — 모바일 화면 위주로 사용
- threshold가 너무 작으면 일반 스크롤 시작도 새로고침으로 잘못 인식

## 사용법

```tsx
import { PullToRefresh } from "@nudge-design/react";

<PullToRefresh
  onRefresh={async () => {
    await refetchList();
  }}
>
  <List />
</PullToRefresh>
```
