# Radio

단일 선택 라디오 버튼. `Checkbox`와 한 쌍입니다. 카드형 선택지는 `SelectionCard`.

## 사용법

```tsx
import { Radio, RadioGroup } from "@nudge-eap/react";

const [plan, setPlan] = useState("basic");

<RadioGroup name="plan" value={plan} onValueChange={setPlan}>
  <Radio value="basic" label="베이직" />
  <Radio value="pro" label="프로" />
  <Radio value="enterprise" label="엔터프라이즈" />
</RadioGroup>
```


> Checkbox와 한 쌍. 카드형 선택지는 SelectionCard 사용.

## 인벤토리

전체 컴포넌트 카탈로그는 [컴포넌트 인벤토리](/docs/components/inventory)에서 확인하세요. Storybook에서는 [Components/Radio](http://localhost:6006/?path=/docs/components-radio--docs)로 진입할 수 있습니다.
