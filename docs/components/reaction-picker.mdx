# ReactionPicker

게시물·일기·콘텐츠에 이모지 반응을 다는 칩 그룹입니다. 다중 선택(기본) 또는 단일 선택을 지원하고, 카운트 표시도 옵션입니다.

## 사용 가이드

### 이럴 때 사용하세요
- 콘텐츠 반응 (좋아요/공감/응원)
- 일기 반응 (그룹 일기, 가족 공유)
- 챌린지 게시판

### 자주 하는 실수
- value는 **항상 string[]** (single이어도 길이 0~1 배열) — 일관된 형태로 처리
- options에 카운트가 없으면(undefined) 자동 숨김. 0이면 표시됨

## 사용법

```tsx
import { ReactionPicker } from "@nudge-design/react";

<ReactionPicker
  options={[
    { key: "love", emoji: "❤️", label: "공감", count: 12 },
    { key: "support", emoji: "🤗", label: "응원", count: 4 },
  ]}
  value={selected}
  onValueChange={setSelected}
/>

// 단일 선택 (한 번에 하나)
<ReactionPicker options={...} value={v} onValueChange={setV} single hideCount />
```

## Props

| Prop            | 타입                          | 기본값  | 설명                              |
| --------------- | ----------------------------- | ------- | --------------------------------- |
| `options`       | `ReactionOption[]`            | —       | 반응 옵션                         |
| `value`         | `string[]`                    | —       | 선택된 키 배열                    |
| `onValueChange` | `(keys: string[]) => void`    | —       | 변경 콜백                         |
| `single`        | `boolean`                     | `false` | 단일 선택 (배열 길이 0~1)         |
| `hideCount`     | `boolean`                     | `false` | 카운트 표시 숨김                  |
| `disabled`      | `boolean`                     | `false` | 비활성화                          |
