# ScoreGauge

검사 점수를 반원 게이지(needle)로 시각화합니다. 4단계 구간(`normal`/`mild`/`moderate`/`severe`)의 색이 자동으로 매핑됩니다.

## 사용법

```tsx
import { ScoreGauge } from "@nudge-eap/react";

// 기본 (max 비율 0~40/40~60/60~80/80~100)
<ScoreGauge value={45} max={100} />

// PHQ-9 커스텀 구간
<ScoreGauge
  value={14}
  max={27}
  segments={[
    { level: "normal", label: "정상", from: 0, to: 5 },
    { level: "mild", label: "경증", from: 5, to: 10 },
    { level: "moderate", label: "중등도", from: 10, to: 20 },
    { level: "severe", label: "심각", from: 20, to: 28 },
  ]}
  showLegend
/>
```

## Props

| Prop          | 타입             | 기본값 | 설명                                |
| ------------- | ---------------- | ------ | ----------------------------------- |
| `value`       | `number`         | —      | 점수                                |
| `max`         | `number`         | `100`  | 최대 점수                           |
| `segments`    | `GaugeSegment[]` | 자동   | 단계 구간 (각 검사마다 직접 정의)   |
| `showLabel`   | `boolean`        | `true` | 단계 라벨 표시 (예: "정상")         |
| `showLegend`  | `boolean`        | `false` | 4단계 색상 범례 표시               |

`GaugeSegment = { level, label, from, to }` — `from <= value < to` 로 매칭.

## 가이드

- 단계 경계는 검사마다 다름. `segments`로 명시해서 결과 해석을 통일.
- `max`를 점수 합과 동일하게 — "100점 만점"으로 정규화하지 말 것 (해석 매트릭스가 어긋남).
- 모바일 카드 안에서는 `showLegend={false}` 권장.
