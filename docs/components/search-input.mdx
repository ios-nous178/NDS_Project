# SearchInput

검색 입력 필드 컴포넌트입니다. 검색 아이콘, 클리어 버튼, Enter 키 검색을 기본 지원하며, `outlined`와 `filled` 두 가지 스타일 변형을 제공합니다. Flat API와 Compound API를 모두 지원합니다.

```tsx
import { SearchInput } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- **검색 기능**을 제공할 때 (목록 필터링, 전체 검색 등)
- 검색 아이콘 + 입력 + 클리어 버튼 + Enter 트리거 패턴이 필요할 때

### 이럴 때 사용하지 마세요

- 일반 텍스트 입력 (이름, 이메일 등) → [Input](/components/input)을 사용하세요
- 미리 정해진 옵션에서 필터링 → [Select](/components/select) 또는 [Chip](/components/chip)을 사용하세요

### 자주 하는 실수

- 검색이 아닌 일반 입력에 SearchInput 사용 — 검색 아이콘이 사용자에게 잘못된 기대를 줍니다
- 검색 결과가 없을 때 피드백 미제공 — [EmptyState](/components/empty-state)와 함께 사용하세요

```tsx

<SearchInput
  value={keyword}
  onChange={(e) => setKeyword(e.target.value)}
  onClear={() => setKeyword("")}
  onSearch={() => handleSearch(keyword)}
  placeholder="검색"
/>;
```

## Compound API

각 요소를 직접 조합하여 자유로운 레이아웃을 구성합니다.

```tsx
<SearchInput.Root>
  <SearchInput.Wrapper variant="filled">
    <SearchInput.Field
      value={keyword}
      onChange={(e) => setKeyword(e.target.value)}
      placeholder="도움이 필요하세요?"
    />
    {keyword && <SearchInput.ClearButton onClick={() => setKeyword("")} />}
    <SearchInput.SearchButton onClick={handleSearch} />
  </SearchInput.Wrapper>
</SearchInput.Root>
```

## Flat API Props

| Prop               | 타입                     | 기본값       | 설명                               |
| ------------------ | ------------------------ | ------------ | ---------------------------------- |
| `variant`          | `"outlined" \| "filled"` | `"outlined"` | 스타일 변형                        |
| `clearable`        | `boolean`                | `true`       | 클리어 버튼 자동 표시              |
| `onClear`          | `() => void`             | —            | 클리어 콜백                        |
| `showSearchButton` | `boolean`                | `true`       | 검색 버튼 표시                     |
| `onSearch`         | `() => void`             | —            | 검색 실행 콜백 (버튼 클릭 + Enter) |
| `searchIcon`       | `ReactNode`              | —            | 검색 아이콘 커스텀                 |
| `fullWidth`        | `boolean`                | `true`       | 부모 너비 100%                     |
| `slotProps`        | `SearchInputSlotProps`   | —            | 슬롯별 props 전달                  |

`<input>` HTML 속성을 모두 지원합니다 (`value`, `onChange`, `placeholder` 등).

## Compound 컴포넌트

### SearchInput.Root

검색 입력 루트 컨테이너입니다.

| Prop        | 타입      | 기본값 | 설명      |
| ----------- | --------- | ------ | --------- |
| `fullWidth` | `boolean` | `true` | 전체 너비 |

### SearchInput.Wrapper

입력 필드를 감싸는 컨테이너입니다.

| Prop      | 타입                     | 기본값       | 설명        |
| --------- | ------------------------ | ------------ | ----------- |
| `focused` | `boolean`                | `false`      | 포커스 상태 |
| `variant` | `"outlined" \| "filled"` | `"outlined"` | 스타일 변형 |

### SearchInput.Field

`<input type="search">` 요소입니다. `ref` 전달을 지원합니다.

### SearchInput.ClearButton

검색어 삭제 버튼입니다. `aria-label="검색어 지우기"`가 내장되어 있습니다.

### SearchInput.SearchButton

검색 실행 버튼입니다. `aria-label="검색"`이 내장되어 있습니다.

## data-slot 구조

```
div [data-slot="root"]
└── div [data-slot="wrapper"]  [data-focused] [data-variant]
    ├── input [data-slot="field"]  (type="search")
    ├── button [data-slot="clear"]     (값이 있을 때)
    └── button [data-slot="search-button"]
```

---

## 접근성

- 클리어 버튼에 `aria-label="검색어 지우기"` 내장
- 검색 버튼에 `aria-label="검색"` 내장
- `type="search"`로 시맨틱 역할 명시
- `ref` 전달 지원
- Enter 키로 검색 실행 지원
