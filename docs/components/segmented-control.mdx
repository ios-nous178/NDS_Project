# SegmentedControl

2~4개 옵션을 가로로 배치한 단일 선택 컨트롤. 옵션이 5개 이상이거나 스크롤이 필요하면 `Tabs` / `FilterBar`.

## 사용법

```tsx
import { SegmentedControl } from "@nudge-eap/react";

const [view, setView] = useState("week");

<SegmentedControl
  value={view}
  onValueChange={setView}
  options={[
    { value: "day", label: "일" },
    { value: "week", label: "주" },
    { value: "month", label: "월" },
  ]}
/>
```


> 옵션 ≥ 5개거나 스크롤 필요하면 Tabs / FilterBar 검토.
