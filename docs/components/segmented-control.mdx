# SegmentedControl

2~4개 옵션을 가로로 배치한 단일 선택 컨트롤입니다. 짧고 동등한 옵션을 빠르게 토글할 때 사용합니다.

```tsx
import { SegmentedControl } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- "일 / 주 / 월" 같은 **2~4개의 동등한 뷰**를 토글할 때
- 짧은 라벨로 표현 가능한 옵션이며 한 줄에 다 들어갈 때
- 라디오 그룹이지만 시각적으로 **컨트롤 한 덩어리**로 보여야 할 때

### 이럴 때 사용하지 마세요

- 옵션이 5개 이상이거나 가로 스크롤이 필요 → [Tabs](/components/tabs)의 `chip` 변형 또는 FilterBar를 사용하세요
- 페이지 콘텐츠 영역을 전환 → [Tabs](/components/tabs)를 사용하세요

---

## 기본 사용

```tsx
const [view, setView] = useState("week");

<SegmentedControl
  value={view}
  onValueChange={setView}
  options={[
    { value: "day", label: "일" },
    { value: "week", label: "주" },
    { value: "month", label: "월" },
  ]}
/>;
```

---

## Props

| Prop            | 타입                              | 기본값  | 설명                            |
| --------------- | --------------------------------- | ------- | ------------------------------- |
| `options`       | `SegmentedOption<T>[]`            | —       | 옵션 목록                       |
| `value`         | `T`                               | —       | 선택된 값                       |
| `onValueChange` | `(value: T) => void`              | —       | 값 변경 콜백                    |
| `size`          | `"sm" \| "md"`                    | `"sm"`  | 크기 (sm: 32px / md: 36px)      |
| `fullWidth`     | `boolean`                         | `false` | 전체 너비 차지                  |
| `disabled`      | `boolean`                         | `false` | 전체 비활성화                   |
| `className`     | `string`                          | —       | 루트 className                  |

`<div>` HTML 속성을 모두 지원합니다 (`onChange` 제외).

### SegmentedOption

| 속성       | 타입        | 설명               |
| ---------- | ----------- | ------------------ |
| `value`    | `T`         | 옵션 값            |
| `label`    | `ReactNode` | 표시 라벨          |
| `disabled` | `boolean`   | 개별 비활성화 여부 |

---

## data-slot 구조

```
div [data-slot="root"] [data-size] [data-fullwidth]  (role="radiogroup")
└── button [data-slot="item"] [data-active]          (role="radio")
```

---

## CSS 변수

이 컴포넌트는 CSS 변수를 노출하지 않습니다.

---

## 접근성

- `role="radiogroup"` + 각 옵션 `role="radio"` + `aria-checked`
- 활성 옵션 클릭 시 콜백 미발화 (불필요한 재선택 방지)
- 키보드 포커스는 `:focus-visible`로 표시
- `disabled` 옵션은 `<button disabled>`로 포커스 불가
