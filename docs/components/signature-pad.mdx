# SignaturePad

마우스/터치로 서명을 그리고 PNG dataURL로 추출할 수 있는 캔버스 컴포넌트입니다. 동의서, 가입 신청, 의료 동의 등에 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- EAP 가입 동의서
- 상담 시작 전 정보 동의 서명
- 의료/법적 책임 관련 동의

### 자주 하는 실수
- 외부 ref(`SignaturePadHandle`)로 `clear()` / `toDataURL()` / `isEmpty()` 호출 — `onChange`로 바로 dataURL을 받으면 매 stroke마다 큰 base64 문자열이 흘러서 비효율적
- 빈 캔버스에서 `toDataURL()`은 `null` 반환 (의도적). 서명 검증에 사용

## 사용법

```tsx
import { SignaturePad, type SignaturePadHandle } from "@nudge-design/react";

const ref = useRef<SignaturePadHandle>(null);

<SignaturePad ref={ref} label="동의 서명" />

<Button
  onClick={() => {
    if (ref.current?.isEmpty()) return alert("서명해주세요");
    submit(ref.current!.toDataURL());
  }}
>
  제출
</Button>

// onChange로 실시간 추출 (가능하지만 리렌더 잦음)
<SignaturePad onChange={(url) => setSig(url)} />
```

## Props

| Prop           | 타입                              | 기본값              | 설명                          |
| -------------- | --------------------------------- | ------------------- | ----------------------------- |
| `label`        | `ReactNode`                       | —                   | 라벨                          |
| `placeholder`  | `string`                          | `"여기에 서명해주세요"` | 빈 캔버스 텍스트          |
| `height`       | `number`                          | `180`               | 캔버스 높이 px                |
| `penColor`     | `string`                          | `"#1A1A1A"`         | 펜 색상                       |
| `penWidth`     | `number`                          | `2.4`               | 펜 두께                       |
| `onChange`     | `(url: string \| null) => void`   | —                   | 변경 콜백 (PointerUp 시점)    |
| `disabled`     | `boolean`                         | `false`             | 비활성화 (읽기 전용)          |
| `hideControls` | `boolean`                         | `false`             | "지우기" 버튼 숨김            |

## Imperative Handle

`useRef<SignaturePadHandle>`로 다음 메서드를 호출할 수 있습니다.

| 메서드        | 반환값                | 설명                             |
| ------------- | --------------------- | -------------------------------- |
| `clear()`     | `void`                | 캔버스 비우기                    |
| `toDataURL()` | `string \| null`      | 현재 서명을 PNG dataURL로 (없으면 null) |
| `isEmpty()`   | `boolean`             | 비어있는지                       |
