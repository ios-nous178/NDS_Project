# Sparkline

축/레이블 없는 작은 추이 그래프입니다. 메트릭 카드 안에 "최근 7일 추이" 같은 시각적 신호로 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 카드 우측 하단의 "최근 N일 추이"
- 리스트 아이템 옆 작은 트렌드
- 대시보드의 빠른 시각 신호

### 이럴 때 사용하지 마세요
- 정확한 값 비교가 필요할 때 → 본격 차트 라이브러리
- 데이터 포인트가 50개 이상 → 가독성 저하

## 사용법

```tsx
import { Sparkline } from "@nudge-design/react";

<Sparkline data={[3, 4, 5, 6, 5, 4, 6, 7]} kind="line" width={120} height={36} />

// 영역 채우기 + 색상 커스텀
<Sparkline
  data={sleepHours}
  kind="area"
  color="var(--semantic-success-main)"
/>

// 막대 형태
<Sparkline data={steps} kind="bar" />

// 음수가 있는 경우 baseline 표시
<Sparkline data={moodScores} kind="line" showBaseline />
```
