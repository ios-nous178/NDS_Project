# StarRating

별점 표시 및 입력 컴포넌트입니다. `onValueChange`를 전달하면 자동으로 인터랙티브(입력) 모드로 전환됩니다.

```tsx
import { StarRating } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 리뷰/평점 결과를 **시각적으로 표시**할 때 (`onValueChange` 없이)
- 사용자가 만족도를 **1~5점으로 입력**할 때
- ReviewCard 같은 카드 컴포넌트의 평점 슬롯으로 임베드할 때

### 이럴 때 사용하지 마세요

- 5점 척도 이상의 자세한 응답이 필요 → LikertScale을 사용하세요
- 좋아요/북마크 같은 단순 토글 → LikeButton을 사용하세요

---

## 기본 사용

```tsx
// 표시 전용 (onValueChange 없음)
<StarRating value={4} size={20} />

// 입력 (onValueChange 지정 시 인터랙티브 모드)
const [score, setScore] = useState(0);
<StarRating value={score} onValueChange={setScore} size={28} />

// 숫자 값 함께 표시
<StarRating value={4.5} showValue />
```

> 별점은 정수 단위 클릭으로 변경되며, 표시 시 `Math.round(value)` 만큼 채워집니다. `showValue`를 켜면 `value.toFixed(1)` 형식의 텍스트가 우측에 함께 노출됩니다.

---

## Props

| Prop            | 타입                       | 기본값  | 설명                                                 |
| --------------- | -------------------------- | ------- | ---------------------------------------------------- |
| `value`         | `number`                   | —       | 별점 값 (0 ~ max)                                    |
| `onValueChange` | `(value: number) => void`  | —       | 값 변경 콜백 (지정 시 인터랙티브 모드)               |
| `size`          | `number`                   | `16`    | 별 크기 (px)                                         |
| `max`           | `number`                   | `5`     | 최대 별 개수                                         |
| `showValue`     | `boolean`                  | `false` | 우측에 숫자 값(`value.toFixed(1)`) 표시              |
| `disabled`      | `boolean`                  | `false` | 비활성화 (인터랙티브 모드 강제 해제)                 |
| `className`     | `string`                   | —       | 루트 className                                       |

`<div>` HTML 속성을 모두 지원합니다 (`onChange` 제외).

## data-slot 구조

```
div [data-slot="root"] [data-interactive]   (role="radiogroup" or "img")
├── span [data-slot="star"] [data-filled]   (role="radio" — 인터랙티브일 때)
└── span [data-slot="value"]                (showValue일 때)
```

## 접근성

- 인터랙티브 모드: `role="radiogroup"`, 각 별 `role="radio"` + `aria-checked` + `aria-label="N star(s)"`
- 표시 전용: `role="img"` + `aria-label="N out of max stars"`
- 호버 시 1.15배 확대 트랜지션으로 인터랙션 피드백 제공
- `disabled` 또는 `onValueChange` 미지정 시 클릭/호버 비활성
