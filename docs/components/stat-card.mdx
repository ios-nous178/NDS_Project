# StatCard

"이번 주 7.4시간" 같은 **숫자 강조 카드**입니다. 라벨, 큰 숫자, 단위, 변화량(trend), 우측 Sparkline 슬롯, 하단 설명을 한 번에 표현합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 홈/대시보드 메트릭 그리드 (수면 / 스트레스 / 챌린지 등)
- 디테일 화면 상단의 핵심 지표
- 리포트 페이지의 요약 카드

### 자주 하는 실수
- `trend`만 주고 `delta`를 빼면 trend가 의미 없음 — 둘 다 함께 사용
- 그리드에서 카드마다 `value` 자릿수가 크게 차이나면 baseline이 흔들림 — 같은 단위로 통일

## 사용법

```tsx
import { StatCard, Sparkline } from "@nudge-eap/react";

// 기본
<StatCard label="이번 주 수면" value="7.4" unit="시간" delta="+0.4" trend="up" />

// Sparkline 결합
<StatCard
  label="기분 점수"
  value="78"
  unit="점"
  delta="+8"
  trend="up"
  trailing={<Sparkline data={[60, 65, 62, 70, 75, 78]} kind="area" />}
/>

// 클릭 가능
<StatCard label="챌린지" value="12" unit="/ 30일" onClick={openDetail} />
```

## Props

| Prop          | 타입                          | 기본값  | 설명                              |
| ------------- | ----------------------------- | ------- | --------------------------------- |
| `label`       | `ReactNode`                   | —       | 상단 라벨                         |
| `value`       | `ReactNode`                   | —       | 강조 숫자/값                      |
| `unit`        | `ReactNode`                   | —       | 값 옆 단위                        |
| `icon`        | `ReactNode`                   | —       | 라벨 좌측 아이콘                  |
| `delta`       | `ReactNode`                   | —       | 변화량 라벨 (예: "+0.4")          |
| `trend`       | `"up" \| "down" \| "flat"`    | —       | 변화 방향 (delta 색상 결정)       |
| `trailing`    | `ReactNode`                   | —       | 우측 슬롯 (Sparkline 등)          |
| `description` | `ReactNode`                   | —       | 하단 보조 설명                    |
| `onClick`     | `() => void`                  | —       | 카드 전체 클릭                    |
