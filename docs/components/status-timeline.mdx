# StatusTimeline

선형 진행 단계를 보여주는 트래커입니다. 가로(배송 추적형) / 세로(상담 진행형) 둘 다 지원하고, 완료된 단계는 자동으로 색이 채워집니다.

> 시간순 활동 로그가 필요하면 `ActivityTimeline`. 단계가 정해진 진행은 StatusTimeline.

## 사용 가이드

### 이럴 때 사용하세요
- 배송 추적 (접수 → 준비 → 배송 중 → 완료)
- 상담/검사 진행 단계
- 신청서 처리 (접수 → 검토 → 승인)

### 자주 하는 실수
- `current`는 **현재 진행 중인** 인덱스 — 그 인덱스 이전은 완료, 이후는 todo
- 모든 단계 완료를 표시하려면 `current = steps.length` (마지막 인덱스보다 1 큰 값)

## 사용법

```tsx
import { StatusTimeline } from "@nudge-design/react";

<StatusTimeline
  steps={[
    { key: "received", label: "주문 접수", time: "5/8 14:00" },
    { key: "preparing", label: "상품 준비", time: "5/8 16:30" },
    { key: "shipping", label: "배송 중", time: "5/9 08:00" },
    { key: "delivered", label: "배송 완료" },
  ]}
  current={2}
/>

// 세로형 (상담 진행)
<StatusTimeline steps={...} current={1} direction="vertical" />
```

## Props

| Prop        | 타입                            | 기본값         | 설명                              |
| ----------- | ------------------------------- | -------------- | --------------------------------- |
| `steps`     | `StatusTimelineStep[]`          | —              | 단계 목록                         |
| `current`   | `number`                        | —              | 현재 진행 인덱스 (0-based)        |
| `direction` | `"horizontal" \| "vertical"`    | `"horizontal"` | 방향                              |

### `StatusTimelineStep`

| Prop          | 타입         | 설명     |
| ------------- | ------------ | -------- |
| `key`         | `string`     | 고유 키  |
| `label`       | `ReactNode`  | 단계 이름 |
| `description` | `ReactNode`  | 설명 (선택) |
| `time`        | `ReactNode`  | 시간/시점 |
