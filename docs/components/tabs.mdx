# Tabs

탭 내비게이션 컴포넌트입니다. `line`, `chip`, `segment` 세 가지 변형을 지원하고, 사이즈는 `mobile` / `pc`, 톤은 `neutral` / `color`로 조합합니다. `line` 변형은 슬라이딩 인디케이터 애니메이션을 내장하며 Flat API와 Compound API를 모두 지원합니다.

```tsx
import { Tabs } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 같은 맥락 내에서 **뷰를 전환**할 때 (프로필 탭, 설정 카테고리 등)
- 콘텐츠를 논리적 그룹으로 나누되, 같은 페이지에서 전환해야 할 때
- 탭 간 콘텐츠가 **동등한 수준**의 정보일 때

### 이럴 때 사용하지 마세요

- 페이지 간 네비게이션 → 라우터 기반 Navigation을 사용하세요
- 단계별 진행 (1단계 → 2단계 → 3단계) → Stepper 패턴을 사용하세요
- 콘텐츠 접기/펴기 → Accordion 패턴을 사용하세요

### 변형 선택 가이드

| 변형      | 용도                                                    | 사이즈 지원        |
| --------- | ------------------------------------------------------- | ------------------ |
| `line`    | 모바일/PC 공용. 표준 언더라인 탭                        | `mobile` · `pc`    |
| `chip`    | 모바일/PC 공용. 가로 스크롤 가능한 알약(pill) 형태      | `mobile` · `pc`    |
| `segment` | **PC 전용 (CMS)**. 균등 분할 세그먼트 컨트롤            | `pc` 고정          |

### 자주 하는 실수

- 탭이 2개 미만 — 탭 UI의 의미가 없습니다. 1개면 탭 없이 바로 표시하세요
- 탭 내용이 서로 완전히 무관 — 탭은 같은 주제의 다른 관점을 보여줄 때 사용합니다
- `segment` 변형에 `size` / `tone`을 지정 — 무시되고 각각 `pc` / `neutral`로 고정됩니다

## Flat API

### Line 변형 (언더라인)

```tsx
const [activeKey, setActiveKey] = useState("all");

const items = [
  { key: "all", title: "전체", content: <AllContent /> },
  { key: "counsel", title: "상담", content: <CounselContent /> },
  { key: "challenge", title: "챌린지", content: <ChallengeContent /> },
];

<Tabs
  items={items}
  activeKey={activeKey}
  onTabChange={setActiveKey}
  variant="line"
  size="mobile"
/>;
```

### Chip 변형 (알약)

가로 스크롤이 가능합니다. 모바일 가로 카테고리 필터로 적합합니다.

```tsx
<Tabs
  items={items}
  activeKey={activeKey}
  onTabChange={setActiveKey}
  variant="chip"
  size="mobile"
  tone="color"
/>
```

### Segment 변형 (PC / CMS)

PC 전용 세그먼트 컨트롤입니다. `size` · `tone`은 무시됩니다. 트리거에 `icon` 지정 가능.

```tsx
const items = [
  { key: "list", title: "목록", icon: <ListIcon /> },
  { key: "grid", title: "그리드", icon: <GridIcon /> },
];

<Tabs
  items={items}
  activeKey={activeKey}
  onTabChange={setActiveKey}
  variant="segment"
/>;
```

### 탭만 사용 (패널 없이)

`content`를 생략하면 탭 리스트만 렌더링됩니다.

```tsx
const items = [
  { key: "address", title: "주소 검색" },
  { key: "center", title: "센터 검색" },
];

<Tabs items={items} activeKey={activeKey} onTabChange={setActiveKey} />;
```

## Compound API

각 요소를 직접 조합하여 자유로운 레이아웃을 구성합니다.

```tsx
<Tabs.Root
  activeKey={activeKey}
  onTabChange={setActiveKey}
  variant="line"
  size="mobile"
  tone="neutral"
>
  <Tabs.List>
    <Tabs.Trigger tabKey="address">주소 검색</Tabs.Trigger>
    <Tabs.Trigger tabKey="center">센터 검색</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Panel tabKey="address">
    <AddressSearch />
  </Tabs.Panel>
  <Tabs.Panel tabKey="center">
    <CenterSearch />
  </Tabs.Panel>
</Tabs.Root>
```

## Flat API Props

| Prop          | 타입                                | 기본값      | 설명                                  |
| ------------- | ----------------------------------- | ----------- | ------------------------------------- |
| `items`       | `TabItem[]`                         | —           | 탭 아이템 목록                        |
| `activeKey`   | `string`                            | —           | 현재 활성 탭 키                       |
| `onTabChange` | `(key: string) => void`             | —           | 탭 변경 콜백                          |
| `variant`     | `"line" \| "chip" \| "segment"`     | `"line"`    | 스타일 변형                           |
| `size`        | `"mobile" \| "pc"`                  | `"pc"`      | 사이즈. `segment`는 `"pc"`로 고정     |
| `tone`        | `"neutral" \| "color"`              | `"neutral"` | 톤. `segment`는 `"neutral"`로 고정    |
| `fullWidth`   | `boolean`                           | `true`      | 전체 너비                             |
| `className`   | `string`                            | —           | 루트 className                        |
| `style`       | `CSSProperties`                     | —           | 루트 style                            |
| `slotProps`   | `TabsSlotProps`                     | —           | 슬롯별 props 전달                     |

### TabItem

| 속성       | 타입        | 설명                                              |
| ---------- | ----------- | ------------------------------------------------- |
| `key`      | `string`    | 탭 식별자                                         |
| `title`    | `ReactNode` | 탭 라벨                                           |
| `content`  | `ReactNode` | 패널 콘텐츠 (생략 시 패널 미렌더링)               |
| `icon`     | `ReactNode` | Segment 변형에서 텍스트 앞에 노출되는 아이콘 노드 |
| `disabled` | `boolean`   | 비활성화 여부                                     |

### TabsSlotProps

`root` / `list` / `trigger` / `panel` 슬롯에 추가 props를 주입합니다. 각 슬롯은 해당 요소의 `className`, `style` 등을 받습니다.

## Compound 컴포넌트

### Tabs.Root

| Prop          | 타입                            | 기본값      | 설명                  |
| ------------- | ------------------------------- | ----------- | --------------------- |
| `activeKey`   | `string`                        | —           | 활성 탭 키            |
| `onTabChange` | `(key: string) => void`         | —           | 탭 변경 콜백          |
| `variant`     | `"line" \| "chip" \| "segment"` | `"line"`    | 변형                  |
| `size`        | `"mobile" \| "pc"`              | `"pc"`      | 사이즈                |
| `tone`        | `"neutral" \| "color"`          | `"neutral"` | 톤                    |
| `fullWidth`   | `boolean`                       | `true`      | 전체 너비             |

### Tabs.List

탭 버튼을 감싸는 `<ul>`. `role="tablist"`. `line` 변형에서는 활성 탭 위치를 측정해 인디케이터가 슬라이딩합니다.

### Tabs.Trigger

개별 탭 버튼.

| Prop       | 타입        | 설명                                  |
| ---------- | ----------- | ------------------------------------- |
| `tabKey`   | `string`    | 탭 식별자 (activeKey와 비교)          |
| `children` | `ReactNode` | 탭 라벨                               |
| `icon`     | `ReactNode` | Segment 변형에서 라벨 앞 아이콘       |
| `disabled` | `boolean`   | 비활성화 여부                         |

### Tabs.Panel

탭 콘텐츠 패널. 비활성 패널의 children은 렌더되지 않습니다(언마운트).

| Prop     | 타입     | 설명                     |
| -------- | -------- | ------------------------ |
| `tabKey` | `string` | 이 패널에 대응하는 탭 키 |

## data-slot 구조

```
div [data-slot="root"]
├── ul [data-slot="list"] [data-variant] [data-size] [data-tone]  (role="tablist")
│   ├── li [data-slot="trigger"] [data-active] [data-disabled] [data-tab-key]  (role="tab")
│   ├── li [data-slot="trigger"] ...
│   └── span [data-slot="indicator"]  (line variant만)
├── div [data-slot="panel"] [data-hidden]  (role="tabpanel")
└── div [data-slot="panel"] [data-hidden]
```

## 변형 × 사이즈 × 톤 매트릭스

| 변형      | 사이즈 | 톤        | 활성 스타일                                        | 인디케이터          |
| --------- | ------ | --------- | -------------------------------------------------- | ------------------- |
| `line`    | mobile | neutral   | `text.strong` + bold                               | 하단 3px 바 (슬라이딩) |
| `line`    | mobile | color     | `primary.main` + bold                              | 하단 3px primary 바    |
| `line`    | pc     | neutral   | `text.strong` + bold                               | 하단 3px 바            |
| `line`    | pc     | color     | `primary.main` + bold                              | 하단 3px primary 바    |
| `chip`    | mobile | neutral   | `fill.neutral` 배경 + `textRole.inverse`           | —                   |
| `chip`    | mobile | color     | `primary.main` 배경 + `primary.fg`                 | —                   |
| `chip`    | pc     | neutral   | `fill.neutral` 배경 + `textRole.inverse`           | —                   |
| `chip`    | pc     | color     | `primary.main` 배경 + `primary.fg`                 | —                   |
| `segment` | pc     | neutral   | `fill.neutral` 배경 + `textRole.inverse` (PC 전용) | —                   |

> `chip`은 `overflow-x: auto`로 가로 스크롤이 활성화되어 있습니다 (스크롤바 숨김).

## QA 체크포인트

Storybook에서 아래 시나리오를 별도 story로 유지합니다.

- 변형 × 사이즈 × 톤 매트릭스 (line/chip/segment, mobile/pc, neutral/color)
- Flat API와 Compound API parity
- 패널 없이 탭 리스트만 쓰는 경우
- 긴 라벨과 overflow (chip 가로 스크롤)
- `disabled` 트리거 동작
- Segment + icon

키보드 화살표 이동, `aria-controls` / `aria-labelledby` 연결, active tab의 `tabIndex` 규칙 같은 항목은 컴포넌트 테스트에서 고정합니다.

## 접근성

- `role="tablist"`, `role="tab"`, `role="tabpanel"` 시맨틱 구조
- `aria-selected`로 활성 탭 표시, `aria-controls` / `aria-labelledby`로 탭-패널 연결
- 키보드: Enter / Space로 탭 전환, ←/→로 인접 탭 이동, Home / End로 처음/마지막 이동
- `tabIndex` 관리: 활성 탭만 0, 나머지 -1 (roving tabindex)
- `disabled` 트리거는 `aria-disabled="true"` + 포커스 불가
