# Textarea

여러 줄 텍스트를 입력받는 필드입니다. 라벨, 헬퍼/에러 메시지, 글자수 카운터를 한 컴포넌트로 묶어 제공합니다.

```tsx
import { Textarea } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 일기, 후기, 상담 메모처럼 **여러 줄의 자유 입력**이 필요할 때
- 글자수 제한이 있는 입력에 카운터를 함께 보여주고 싶을 때
- 라벨·헬퍼·에러를 일관된 폼 필드로 묶고 싶을 때

### 이럴 때 사용하지 마세요

- 한 줄 텍스트(이름, 이메일 등) → [Input](/docs/components/input)을 사용하세요
- 마크다운·리치텍스트 편집 → 별도 에디터를 사용하세요

---

## 기본 사용

```tsx
<Textarea
  label="오늘의 일기"
  placeholder="오늘 어떤 일이 있었나요?"
  minHeight={120}
  maxLength={500}
  helperText="500자 이내로 작성해 주세요"
/>
```

`maxLength`를 전달하면 우측 하단에 `현재/최대` 카운터가 자동 표시되며, 초과 시 빨간색으로 강조됩니다.

### 에러 상태

```tsx
<Textarea
  label="후기"
  value={value}
  onChange={(e) => setValue(e.target.value)}
  error
  helperText="최소 10자 이상 입력해 주세요"
/>
```

## Props

| Prop          | 타입                                              | 기본값  | 설명                                  |
| ------------- | ------------------------------------------------- | ------- | ------------------------------------- |
| `label`       | `ReactNode`                                       | —       | 상단 라벨                             |
| `error`       | `boolean`                                         | `false` | 에러 상태 (border + helper 색 변경)   |
| `helperText`  | `ReactNode`                                       | —       | 하단 헬퍼 / 에러 메시지               |
| `maxLength`   | `number`                                          | —       | 최대 글자수 (지정 시 카운터 노출)     |
| `minHeight`   | `number`                                          | `80`    | 최소 높이 (px)                        |
| `resize`      | `"none" \| "vertical" \| "horizontal" \| "both"`  | `"vertical"` | 리사이즈 모드                  |
| `disabled`    | `boolean`                                         | `false` | 비활성화                              |
| `readOnly`    | `boolean`                                         | `false` | 읽기 전용                             |
| `className`   | `string`                                          | —       | 루트 className                        |

`<textarea>` HTML 속성을 모두 지원합니다 (`value`, `defaultValue`, `onChange`, `onFocus`, `onBlur`, `placeholder` 등).

## data-slot 구조

```
div [data-slot="root"]
├── label [data-slot="label"]                                       (label이 있을 때)
├── div [data-slot="wrapper"] [data-focused] [data-error] [data-disabled] [data-readonly]
│   ├── textarea [data-slot="field"]
│   └── div [data-slot="count"] [data-over]                         (maxLength가 있을 때)
└── span [data-slot="helper"] [data-error]                          (helperText가 있을 때)
```

## CSS 변수

| 변수                          | 기본값            | 설명                |
| ----------------------------- | ----------------- | ------------------- |
| `--nds-textarea-width`        | `100%`            | 루트 너비           |
| `--nds-textarea-min-height`   | `80px`            | 입력 최소 높이      |
| `--nds-textarea-resize`       | `vertical`        | 리사이즈 모드       |
| `--nds-textarea-border-color` | `border.default`  | 테두리 색상         |
| `--nds-textarea-radius`       | `8px`             | 모서리 둥글기       |
| `--nds-textarea-background`   | `bg.white`        | 배경색              |
| `--nds-textarea-helper-color` | `text.subtle`     | 헬퍼 텍스트 색상    |

`minHeight` / `resize` prop을 전달하면 위 CSS 변수가 인라인으로 설정됩니다.

## 접근성

- `<label htmlFor={id}>`로 라벨-필드 연결 (id 미지정 시 `useId()`로 자동 생성)
- `error` 시 `aria-invalid="true"`
- `helperText`가 있으면 `aria-describedby`로 연결
- `disabled` / `readOnly` 시 배경 변경 + 커서 default
