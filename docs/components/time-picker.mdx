# TimePicker

날짜 없이 시간(HH:mm)만 선택하는 컴포넌트입니다. 알림 시각, 복약 시간 등 "하루 중 시각" 입력에 사용하세요.

## 사용 가이드

### 이럴 때 사용하세요
- 알림 / 리마인더 시각 설정
- 복약 시간
- 상담 슬롯 시작 시간

### 이럴 때 사용하지 마세요
- 날짜 + 시간 → `DatePicker` + `TimePicker` 조합 또는 별도 datetime
- 슬롯 목록에서 선택 → `TimeSlotPicker`

## 사용법

```tsx
import { TimePicker } from "@nudge-design/react";

<TimePicker
  label="알림 시각"
  value={time}
  onValueChange={setTime}
  step={60 * 5}
  min="08:00"
  max="22:00"
/>
```

## Props

| Prop            | 타입                        | 기본값  | 설명                              |
| --------------- | --------------------------- | ------- | --------------------------------- |
| `value`         | `string`                    | —       | `HH:mm`                           |
| `onValueChange` | `(v: string) => void`       | —       | 변경 콜백                         |
| `step`          | `number`                    | `300`   | 초 단위 (300 = 5분)               |
| `label`         | `ReactNode`                 | —       | 라벨                              |
| `helperText`    | `ReactNode`                 | —       | 헬퍼 / 에러 텍스트                |
| `error`         | `boolean`                   | `false` | 에러                              |
| `fullWidth`     | `boolean`                   | `false` | 가로 가득                         |
| `disabled`      | `boolean`                   | `false` | 비활성화                          |
| `min`           | `string`                    | —       | 최소 시각 `HH:mm`                 |
| `max`           | `string`                    | —       | 최대 시각 `HH:mm`                 |
