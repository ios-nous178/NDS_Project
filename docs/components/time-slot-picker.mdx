# TimeSlotPicker

예약 가능한 시간대를 슬롯 그리드로 선택하는 컴포넌트입니다. 그룹화(오전/오후/저녁)와 개별 슬롯 비활성을 지원합니다.

```tsx
import { TimeSlotPicker } from "@nudge-design/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 상담/예약처럼 **정해진 시간 슬롯 중 하나**를 선택할 때
- 오전/오후/저녁 등 그룹으로 묶어 보여주고 싶을 때
- 이미 예약된 슬롯을 비활성(취소선) 처리해 함께 보여주고 싶을 때

### 이럴 때 사용하지 마세요

- 자유 시간 입력 → TimePicker를 사용하세요
- 날짜만 선택 → [Calendar](/components/calendar) / [DatePicker](/components/date-picker)를 사용하세요

## 기본 사용

### 평면(slots) 모드

```tsx
const [slot, setSlot] = useState<string>();

<TimeSlotPicker
  value={slot}
  onValueChange={setSlot}
  columns={3}
  slots={[
    { value: "13:00" },
    { value: "13:30", unavailable: true },
    { value: "14:00" },
    { value: "14:30" },
  ]}
/>;
```

### 그룹(groups) 모드

```tsx
<TimeSlotPicker
  value={slot}
  onValueChange={setSlot}
  groups={[
    { key: "am", label: "오전", slots: [{ value: "10:00" }, { value: "11:00" }] },
    { key: "pm", label: "오후", slots: [{ value: "14:00" }, { value: "15:00", unavailable: true }] },
  ]}
/>
```

`slots` / `groups`가 모두 비어 있으면 `emptyText`가 표시됩니다.

`<div>` HTML 속성을 모두 지원합니다 (`onChange` 제외).

### TimeSlot

| 속성          | 타입        | 설명                              |
| ------------- | ----------- | --------------------------------- |
| `value`       | `string`    | 슬롯 키 (보통 `"HH:MM"`)          |
| `label`       | `ReactNode` | 표시 라벨 (없으면 value 그대로)   |
| `unavailable` | `boolean`   | 예약 불가 (취소선 + 비활성)       |

### TimeSlotGroup

| 속성    | 타입          | 설명                              |
| ------- | ------------- | --------------------------------- |
| `key`   | `string`      | 그룹 키                           |
| `label` | `ReactNode`   | 그룹 라벨 (예: "오전")            |
| `slots` | `TimeSlot[]`  | 그룹 내 슬롯들                    |

## data-slot 구조

```
div [data-slot="root"]                              (role="radiogroup")
├── div [data-slot="group"]
│   ├── span [data-slot="group-label"]              (group.label이 있을 때)
│   └── div [data-slot="grid"]
│       └── button [data-slot="slot"] [data-selected] [data-unavailable]  (role="radio")
└── div [data-slot="empty"]                          (slots/groups가 비어있을 때)
```

## 접근성

- 루트 `role="radiogroup"`, 각 슬롯 `role="radio"` + `aria-checked`
- `unavailable` 슬롯은 `<button disabled>` + 시각적 취소선
- `:focus-visible`로 키보드 포커스 인디케이터 표시
