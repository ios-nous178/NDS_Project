# TimeSlotPicker

예약 가능한 시간대를 슬롯 그리드로 선택. 그룹화(오전/오후/저녁) 지원. 자유 시간 입력은 `TimePicker`.

## 사용법

```tsx
import { TimeSlotPicker } from "@nudge-eap/react";

const [slot, setSlot] = useState<string>();

<TimeSlotPicker
  value={slot}
  onValueChange={setSlot}
  columns={3}
  slots={[
    { value: "13:00" },
    { value: "13:30", unavailable: true },
    { value: "14:00" },
    { value: "14:30" },
  ]}
/>
```


> TimePicker(자유 입력) / Calendar(날짜만)와 분리. 정해진 슬롯 리스트 전용.
