# Tooltip

hover · focus 시 표시되는 짧은 보조 설명. 모바일 트리거가 제한적이므로 중요한 정보를 Tooltip에만 두지 않습니다.

## 사용법

```tsx
import { Tooltip, IconButton } from "@nudge-eap/react";

<Tooltip content="이 검사는 약 5분 소요됩니다" placement="top">
  <IconButton aria-label="안내" icon={<InfoIcon />} />
</Tooltip>
```


> 모바일에서는 트리거가 제한적이므로 중요 정보를 Tooltip에만 두지 않는다. 상세 안내는 CoachMark.

## 인벤토리

전체 컴포넌트 카탈로그는 [컴포넌트 인벤토리](/docs/components/inventory)에서 확인하세요. Storybook에서는 [Components/Tooltip](http://localhost:6006/?path=/docs/components-tooltip--docs)로 진입할 수 있습니다.
