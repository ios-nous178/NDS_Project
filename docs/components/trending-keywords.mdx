# TrendingKeywords

실시간 인기 검색어 리스트. 순위 + 키워드 + trend(new/up/down/same) 화살표.

## 사용법

```tsx
import { TrendingKeywords } from "@nudge-eap/react";

<TrendingKeywords
  title="인기 검색어"
  timestamp="09:00 기준"
  items={[
    { rank: 1, trend: "up", keyword: "직장 스트레스" },
    { rank: 2, trend: "same", keyword: "불안" },
    { rank: 3, trend: "new", keyword: "수면" },
  ]}
  onKeywordClick={(item) => navigate(`/search?q=${item.keyword}`)}
/>
```


> value/onChange 패턴 아님 — 클릭 시 검색 트리거하도록 외부에서 라우팅.
