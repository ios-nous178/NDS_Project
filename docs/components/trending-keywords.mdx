# TrendingKeywords

실시간 인기 검색어를 한 줄 슬라이더로 노출하고, 클릭 시 전체 목록 드롭다운을 펼치는 컴포넌트입니다.

```tsx
import { TrendingKeywords } from "@nudge-eap/react";
```

---

## 사용 가이드

### 이럴 때 사용하세요

- 검색바 또는 헤더 영역에 **실시간 인기 검색어**를 노출할 때
- 한 줄 슬라이더로 자동 순환 + 클릭 시 전체 목록을 열어야 할 때
- 키워드 클릭 시 검색 페이지로 이동시키고 싶을 때

### 이럴 때 사용하지 마세요

- value/onChange 패턴의 선택 UI가 필요 → SegmentedControl / [Tabs](/components/tabs)를 사용하세요
- 일반 리스트 표현 → [List](/components/list)를 사용하세요

---

## 기본 사용

```tsx
<TrendingKeywords
  title="인기 검색어"
  timestamp="09:00 기준"
  items={[
    { rank: 1, trend: "up", keyword: "직장 스트레스" },
    { rank: 2, trend: "same", keyword: "불안" },
    { rank: 3, trend: "new", keyword: "수면" },
    { rank: 4, trend: "down", keyword: "번아웃" },
  ]}
  onKeywordClick={(item) => navigate(`/search?q=${item.keyword}`)}
/>
```

> 동작:
> - 닫혀 있을 때는 `autoplayDelay` 간격으로 슬라이더가 자동 순환
> - 슬라이더 클릭 시 드롭다운 열림(자동 순환 일시정지)
> - 드롭다운 외부 클릭 또는 닫기 버튼으로 닫힘

## Props

| Prop             | 타입                                       | 기본값         | 설명                              |
| ---------------- | ------------------------------------------ | -------------- | --------------------------------- |
| `items`          | `TrendingKeywordItem[]`                    | —              | 인기 검색어 목록                  |
| `title`          | `string`                                   | `"인기 검색어"` | 드롭다운 제목                     |
| `timestamp`      | `string`                                   | —              | 기준 시간 텍스트 (예: "09:00 기준") |
| `autoplayDelay`  | `number`                                   | `2000`         | 자동 슬라이드 간격 (ms)           |
| `onKeywordClick` | `(item: TrendingKeywordItem) => void`      | —              | 키워드 클릭 콜백                  |
| `className`      | `string`                                   | —              | 루트 className                    |

### TrendingKeywordItem

| 속성      | 타입                                       | 설명                          |
| --------- | ------------------------------------------ | ----------------------------- |
| `rank`    | `number`                                   | 순위 (1부터 시작)             |
| `trend`   | `"new" \| "up" \| "down" \| "same"`        | 순위 변동 트렌드              |
| `keyword` | `string`                                   | 검색어 텍스트                 |

### Trend 표시

| trend  | 표시          | 색상              |
| ------ | ------------- | ----------------- |
| `new`  | `NEW` 텍스트  | `error.main` (빨강) |
| `up`   | 위쪽 삼각형   | `error.main` (빨강) |
| `down` | 아래쪽 삼각형 | `primary.main`    |
| `same` | 가로 막대     | `text.disabled`   |

## data-slot 구조

```
div.nds-trending-keywords
├── div.nds-trending-keywords__slider                 (자동 순환 슬라이더, 클릭 시 드롭다운 열기)
│   ├── div.nds-trending-keywords__slide-item
│   │   ├── span.nds-trending-keywords__rank
│   │   ├── span.nds-trending-keywords__trend [data-trend]
│   │   └── span.nds-trending-keywords__keyword
│   └── span.nds-trending-keywords__chevron [data-open]
└── div.nds-trending-keywords__dropdown [data-open]
    ├── div.__dropdown-header
    │   ├── span.__dropdown-title
    │   ├── span.__dropdown-time                       (timestamp가 있을 때)
    │   └── button.__dropdown-close                    (aria-label="닫기")
    └── div.__dropdown-list
        └── div.__dropdown-item                        (role="button", tabIndex=0)
```

> 이 컴포넌트는 다른 컴포넌트와 달리 `data-slot` 속성 대신 BEM 형식의 className만 노출합니다.

---

## CSS 변수

이 컴포넌트는 CSS 변수를 노출하지 않습니다. 슬라이더 너비(200px), 높이(30px) 등은 내부 고정값입니다.

---

## 접근성

- 드롭다운 외부 클릭 시 자동 닫힘 (`mousedown` 리스너)
- 드롭다운 항목 `role="button"`, `tabIndex={0}` — Enter 키로 활성화 가능
- 닫기 버튼 `aria-label="닫기"`
- 자동 순환은 드롭다운이 열리면 일시정지
- `items`가 비어 있으면 컴포넌트 자체가 렌더링되지 않음 (`return null`)
