# UserCard

사용자 프로필을 작은 카드로 표시합니다. CounselorCard가 EAP 특화라면 UserCard는 범용 — 팔로우 리스트, 그룹 멤버, 추천 친구 등.

## 사용 가이드

### 이럴 때 사용하세요
- 팔로우/팔로워 리스트
- 그룹 멤버
- 추천 친구/동료
- 작성자 프로필 미니카드

### 자주 하는 실수
- onClick(카드 전체)과 action(버튼)을 동시에 두면 버튼 클릭은 stopPropagation 처리됨 (의도)
- `bio`는 자동 2줄 클램프. 더 많이 보여줘야 하면 디테일 화면으로

## 사용법

```tsx
import { UserCard, Avatar, Button } from "@nudge-eap/react";

<UserCard
  avatar={<Avatar name="김민지" />}
  name="김민지"
  handle="@minji"
  bio="마음챙김 가이드 작성자"
  meta={<>팔로워 1.2K · 게시물 48</>}
  verified
  action={<Button size="sm">팔로우</Button>}
  onClick={openProfile}
/>

// 세로 (프로필 모달)
<UserCard layout="stacked" ... />
```

## Props

| Prop       | 타입                        | 기본값  | 설명                              |
| ---------- | --------------------------- | ------- | --------------------------------- |
| `avatar`   | `ReactNode`                 | —       | 아바타                            |
| `name`     | `ReactNode`                 | —       | 사용자 이름                       |
| `handle`   | `ReactNode`                 | —       | 핸들/직책 (이름 아래 작은 글씨)   |
| `bio`      | `ReactNode`                 | —       | 한 줄 소개 (2줄 클램프)           |
| `meta`     | `ReactNode`                 | —       | 메타 (팔로워/게시물 수 등)        |
| `action`   | `ReactNode`                 | —       | 우측 액션 슬롯 (버튼)             |
| `verified` | `boolean`                   | `false` | ✓ 인증 마크                       |
| `layout`   | `"row" \| "stacked"`        | `"row"` | 레이아웃                          |
| `onClick`  | `() => void`                | —       | 카드 전체 클릭                    |
