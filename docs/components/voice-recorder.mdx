# VoiceRecorder

녹음 시작/정지 버튼과 진행 시간을 보여주는 UI 컴포넌트입니다. 실제 마이크 접근 / 파일 저장은 외부에서 처리하고, 이 컴포넌트는 **시각적 인디케이터**만 담당합니다.

## 사용 가이드

### 이럴 때 사용하세요
- 감정 일기 음성 모드
- 챗 음성 메시지
- 상담 자료 음성 메모

### 자주 하는 실수
- `seconds`는 외부 timer로 갱신해야 함. 컴포넌트는 시간 자체를 측정하지 않음 (테스트 가능성/정확도)
- `maxSeconds` 도달 시 자동으로 `onStateChange("idle")` + `onComplete(sec)` 호출

## 사용법

```tsx
import { VoiceRecorder, type VoiceRecorderState } from "@nudge-eap/react";

const [state, setState] = useState<VoiceRecorderState>("idle");
const [seconds, setSeconds] = useState(0);

useEffect(() => {
  if (state !== "recording") return;
  const t = setInterval(() => setSeconds((s) => s + 1), 1000);
  return () => clearInterval(t);
}, [state]);

<VoiceRecorder
  state={state}
  onStateChange={(s) => {
    if (s === "idle") setSeconds(0);
    setState(s);
    // 외부에서 navigator.mediaDevices.getUserMedia 등 처리
  }}
  seconds={seconds}
  maxSeconds={60}
  onComplete={(sec) => saveRecording(sec)}
/>
```

## Props

| Prop             | 타입                                          | 기본값                       | 설명                          |
| ---------------- | --------------------------------------------- | ---------------------------- | ----------------------------- |
| `state`          | `"idle" \| "recording" \| "paused"`           | —                            | 녹음 상태 (controlled)        |
| `onStateChange`  | `(s: VoiceRecorderState) => void`             | —                            | 상태 변경 콜백                |
| `seconds`        | `number`                                      | —                            | 진행 시간 (초)                |
| `maxSeconds`     | `number`                                      | —                            | 최대 녹음 시간                |
| `onComplete`     | `(sec: number) => void`                       | —                            | 녹음 종료 시 콜백             |
| `idleLabel`      | `string`                                      | `"버튼을 눌러 녹음을 시작하세요"` | 대기 라벨               |
| `recordingLabel` | `string`                                      | `"녹음 중"`                  | 녹음 중 라벨                  |
