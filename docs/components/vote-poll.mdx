# VotePoll

질문 + 옵션 + 결과 바를 하나로 묶은 가벼운 투표 카드입니다. 사용자가 투표하면 자동으로 결과(퍼센트 바)가 노출됩니다.

## 사용 가이드

### 이럴 때 사용하세요
- 커뮤니티/콘텐츠 짧은 투표
- 챌린지 주제 정하기
- 가벼운 의견 수렴

### 이럴 때 사용하지 마세요
- 본격 설문(여러 문항) → 별도 검사/설문 컴포넌트
- 익명성이 중요한 PHQ-9 같은 임상 척도 → `LikertScale`

### 자주 하는 실수
- `votedKey`가 있으면 자동으로 결과 노출. 결과만 보여주려면 `showResults=true`
- count는 외부 state(서버 응답 등)와 동기화 — 컴포넌트가 자체로 카운트 안 함

## 사용법

```tsx
import { VotePoll } from "@nudge-design/react";

<VotePoll
  question="명상하기 가장 좋은 시간은?"
  options={[
    { key: "morning", label: "아침", count: 42 },
    { key: "evening", label: "저녁", count: 67 },
  ]}
  votedKey={voted}
  onVote={(key) => {
    setVoted(key);
    api.vote(pollId, key);
  }}
  footer={`총 ${total}명 투표`}
/>

// 결과만 (마감)
<VotePoll question="..." options={...} showResults disabled />
```
