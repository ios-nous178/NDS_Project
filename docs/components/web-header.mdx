# WebHeader

데스크톱·태블릿용 웹 헤더. 로고 + 메뉴 + 우측 액션 슬롯. 모바일 앱은 `AppBar` 사용.

## 사용법

```tsx
import { WebHeader } from "@nudge-eap/react";

<WebHeader position="sticky" maxWidth={1200}>
  <WebHeader.Logo alt="Brand" src="/logo.svg" />
  <WebHeader.Menu>
    <WebHeader.MenuItem href="/" active>홈</WebHeader.MenuItem>
    <WebHeader.MenuItem href="/programs">콘텐츠</WebHeader.MenuItem>
    <WebHeader.MenuItem href="/counselors">상담</WebHeader.MenuItem>
  </WebHeader.Menu>
  <WebHeader.Actions>
    <WebHeader.AuthButton href="/signin">로그인</WebHeader.AuthButton>
  </WebHeader.Actions>
</WebHeader>
```


> 모바일 앱은 AppBar 사용. WebHeader는 큰 화면 가로 메뉴 전용.
