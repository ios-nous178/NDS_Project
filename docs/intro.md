# NUDGE Design

Nudge Design System은 여러 플랫폼에서 일관된 사용자 경험을 만들기 위한 **공통 인터페이스 시스템**입니다.<br />
Web, App, WebView, Admin 등 각 플랫폼은 동일한 원칙과 구조를 기반으로 동작합니다.

불필요한 복잡성과 과한 규칙은 줄이고,<br />
팀이 더 적은 의사결정으로 핵심 문제 해결에 집중할 수 있는 구조를 지향합니다.

모든 토큰과 컴포넌트는 공통 스펙을 기준으로 관리되며,<br />
플랫폼별 구현체는 이를 기반으로 제공됩니다.
