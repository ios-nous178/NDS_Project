## 색상 팔레트 상세

### Neutral

UI의 기본 배경, 텍스트, 테두리에 사용됩니다.

| 토큰 | 값 | 용도 |
|---|---|---|
| `neutral["00"]` | `#FFFFFF` | 배경, 카드 |
| `neutral[50]` | `#FAFAFA` | 연한 배경 |
| `neutral[100]` | `#F5F5F5` | 구분 배경, Badge 기본 |
| `neutral[200]` | `#ECECEC` | 구분선 (연한) |
| `neutral[300]` | `#D8D8D8` | 테두리 기본 |
| `neutral[400]` | `#C7C7C7` | 비활성 아이콘 |
| `neutral[500]` | `#999999` | Placeholder, 비활성 텍스트 |
| `neutral[600]` | `#777777` | 보조 텍스트 |
| `neutral[700]` | `#666666` | 본문 텍스트 (서브) |
| `neutral[800]` | `#383838` | 본문 텍스트 (기본) |
| `neutral[900]` | `#111111` | 강조 텍스트 |
| `neutral[1000]` | `#000000` | 최대 강조 |

## 시멘틱 토큰

의미 기반으로 색상을 사용할 때 시멘틱 토큰을 권장합니다.

```tsx
import { colors } from "@nudge-eap/tokens";

colors.semantic.primary.main;    // Blue 500 — CTA, 주요 액션
colors.semantic.error.main;      // Red 500 — 에러
colors.semantic.text.default;    // Neutral 800 — 본문 텍스트
colors.semantic.border.default;  // Neutral 300 — 기본 테두리
```

### Primary

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.primary.main` | Blue 500 | CTA, 버튼, 링크 |
| `semantic.primary.hover` | Blue 600 | 호버 상태 |
| `semantic.primary.lighter` | Blue 200 | 연한 강조 |
| `semantic.primary.bg` | Blue 100 | 배경 |
| `semantic.primary.bgLighter` | Blue 50 | 더 연한 배경 |

### Secondary

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.secondary.sub` | Magenta 500 | 보조 강조 |
| `semantic.secondary.lighter` | Magenta 200 | 연한 강조 |
| `semantic.secondary.bg` | Magenta 100 | 배경 |
| `semantic.secondary.bgLighter` | Magenta 50 | 더 연한 배경 |

### 상태

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.error.main` | Red 500 | 에러 텍스트, 테두리 |
| `semantic.error.bg` | Red 50 | 에러 배경 |
| `semantic.caution.main` | Yellow 500 | 경고 |
| `semantic.caution.text` | Yellow 600 | 경고 텍스트 |
| `semantic.caution.bg` | Yellow 50 | 경고 배경 |
| `semantic.success.main` | Green 300 | 성공 |
| `semantic.success.bg` | Green 50 | 성공 배경 |

### 텍스트

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.text.default` | Neutral 800 | 기본 본문 |
| `semantic.text.subtle` | Neutral 700 | 보조 텍스트 |
| `semantic.text.disabled` | Neutral 500 | 비활성 텍스트 |
| `semantic.text.placeholder` | Neutral 500 | Placeholder |

### 배경

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.bg.white` | White | 기본 배경 |
| `semantic.bg.light` | Neutral 50 | 연한 배경 |
| `semantic.bg.coolGray` | Cool Gray 100 | 구분 배경 |
| `semantic.bg.coolGrayLighter` | Cool Gray 50 | 더 연한 구분 배경 |

### 테두리

| 토큰 | 값 | 용도 |
|---|---|---|
| `semantic.border.default` | Neutral 300 | 기본 테두리 |
| `semantic.border.light` | Neutral 200 | 연한 테두리 |
| `semantic.border.focus` | Blue 500 | 포커스 테두리 |

## 사용 예시

```tsx
import { colors } from "@nudge-eap/tokens";

// 직접 참조
const errorStyle = { color: colors.red[500] };

// 시멘틱 사용 (권장)
const textStyle = { color: colors.semantic.text.default };
const borderStyle = { borderColor: colors.semantic.border.default };
```
