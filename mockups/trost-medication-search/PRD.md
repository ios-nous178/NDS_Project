# 복용약 찾기 페이지 PRD

## 개요

트로스트 사용자가 복용 중인 약물의 정보를 쉽게 검색하고, 심리 상담과 자연스럽게 연계할 수 있는 페이지.

## 목표

- **SEO**: "복용약 검색", "항우울제 종류", "약 부작용 확인" 등 롱테일 키워드 유입
- **체류시간**: 검색 → 카테고리 탐색 → 상세 정보 → 상담 연계로 이어지는 플로우
- **전환**: 약물 정보 탐색 후 전문 상담사 연결 CTA

## 타겟 사용자

- 정신건강 관련 약물을 복용 중이거나 복용을 고려하는 사용자
- 복용 중인 약의 부작용/상호작용이 궁금한 사용자
- 약물 상담이 필요한 사용자

## 페이지 스코프

- 검색 + 목록 페이지 (상세 페이지는 별도)

## 데이터

- 더미 데이터 기반 (추후 식약처 API 연동 가능)

## 섹션 구조

### 1. AppBar (헤더)

- 트로스트 로고 + GNB (심리검사, 상담사 찾기, 복용약 찾기, 커뮤니티)
- AppBar.SearchBar 포함

### 2. Hero + 검색

- H1: "복용약 검색 — 내가 먹는 약, 제대로 알기"
- 서브카피: 부작용, 상호작용, 복용 주의사항까지 한눈에
- 중앙 SearchInput (큰 사이즈)
- 인기 검색어 태그 (Chip)

### 3. 카테고리별 필터

- Chip 필터: 전체 / 항우울제 / 항불안제 / 수면제 / 기분안정제 / ADHD 치료제
- 선택 시 목록 필터링

### 4. 약물 목록 (Card 그리드)

- Card: 약물명, 성분명, 카테고리 Badge, 주요 효능, 복용자 수, 주의등급
- 1열(모바일) / 2열(데스크탑) 그리드

### 5. 복용 주의사항 가이드

- 정보성 콘텐츠 블록 (SEO 롱테일)
- 약물 복용 시 주의할 점 3가지 (Card 3열)

### 6. FAQ

- Accordion (임시 구현)
- FAQ Schema 마크업 대응

### 7. 상담 연계 CTA

- "약에 대해 더 궁금한 점이 있으신가요?"
- 전문 상담사 찾기 / 복약 상담 신청 Card

### 8. 관련 검색 태그

- 내부 순환 링크 (Chip)

### 9. AppFooter (푸터)

- 트로스트 약관 링크 + 회사 정보

## 사용 DS 컴포넌트

- `AppBar` (Logo, SearchBar, GNB, AuthMenu)
- `AppFooter.Info` (Links, CompanyInfo)
- `SearchInput`
- `Button`
- `Card` (Header, Body, Footer, Title, Subtitle)
- `Badge`
- `Chip`

## 임시 구현 컴포넌트

- `Accordion` — 트로스트 스타일 (border #e5e5e5, rounded-lg, 333 텍스트)
- `StickyBottomBar` — 모바일 고정 CTA
