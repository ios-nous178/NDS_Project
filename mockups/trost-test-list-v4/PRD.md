## 1. 배경 & 목표

### 현재 문제

- `/test` 페이지가 **"서비스 메뉴"**로만 읽혀, 구글에서 broad 키워드(예: "무료 심리검사", "자가진단") 유입이 잡히지 않음
- 에디토리얼 컨텍스트 부재 → 크롤러가 "정보 제공 의도"를 감지 못 함
- 다크 라우팅 박스 → 시각적 분리로 키워드 밀도 약화
- FAQ 수 부족 (4개) + 스키마 미적용

### 개선 목표

1. **SEO 유입 확대**: "무료 심리검사 종류", "우울증 자가진단" 등 롱테일 키워드 타깃팅
2. **E-E-A-T 신호 강화**: 트로스트 차별점을 독립 섹션화 (전문성·신뢰도)
3. **FAQ Rich Result**: 구조화 데이터(`FAQPage` JSON-LD)로 구글 리치 스니펫 확보
4. **실제 서비스와 목록 정합성**: 제공하지 않는 검사(불안/수면 등) 제거, 실제 검사(코로나 우울증, MBTI 우울증) 추가

## 3. 섹션별 세부 스펙

### 3.1 Hero

- **H1**: "무료 심리검사 종류 - 나에게 맞는 자가진단 찾기"
- **서브카피**: "우울증 · 자존감 · 직무 스트레스까지. 3분만 투자하면 내 마음 상태를 객관적으로 확인할 수 있어요."
- **Primary CTA**: [3분만에 내 상태 확인하기] → `#test-list` 앵커 스크롤
- **감정 버튼 4종** (가로 배치, 모바일은 2×2 그리드):
  - `우울` → `/test/depression-type`
  - `스트레스` (← **불안에서 변경**) → `/test/job-stress`
  - `자존감` → `/test/self-esteem`
  - `모르겠다면 종합 검사` → `/test/general-psych`
- **삭제 항목**: "검사 전체 보기" 버튼(중복 기능)

### 3.2 정보형 허브 블록 (NEW)

- **형태**: 설명 카피 + 4개 유형 카드
- **유형 카드** (각 유형은 §3.3 검사 카드의 키워드로 1:1 매핑):
  - **증상형** — 우울, 공황, 코로나 우울증
  - **자기 이해형** — 자존감, MBTI 우울증
  - **상황형** — 직무 스트레스, 취업 스트레스
  - **종합형** — 종합 심리, 종합 성격
- 링크/버튼 없음 (순수 정보 영역)

### 3.3 무료 온라인 심리검사 목록

  - 타이틀 (headline4 18px/700)

  - 소요 시간 (caption1 13px)

### 3.4 감정/상황 맞춤 추천 (라우팅)

- **H2**: "지금 내 감정 · 상황에 맞는 심리검사"
- **라이트 카드 그리드** (기존 다크박스 → 라이트)
- **카드 5종**:
  - 우울 → `/test/depression-type`
  - 자존감 → `/test/self-esteem`
  - 성격/기질 → `/test/personality`
  - 스트레스 → `/test/job-stress`
  - (불안은 **제거**)
- **반응형**: 모바일 2열 Grid. 카드 홀수일 경우 마지막 카드는 full-width (1열)

### 3.5 심리검사 선택 가이드

- **H2**: "심리검사, 어떤 걸 먼저 해야 할까?"
- Long-form 형식 (3~4개 단락, 각 150~250자)
- 소제목 H3: "증상 기준으로 선택하기", "상황 기준으로 선택하기"
- **반응형**: PC에서만 "증상/상황 기준" 진입 카드 노출. 모바일은 텍스트만

### 3.6 트로스트가 다른 이유 (NEW)

- **H2**: "무료 온라인 심리 검사, 트로스트가 다른 이유"
- 3~4개 특징 카드 (아이콘 + 타이틀 + 설명):
  1. **임상심리 전문가 검수** — 모든 검사는 임상심리사 자격자가 문항을 검수
  2. **100% 무료 · 회원가입 불필요** — 즉시 시작
  3. **검사 후 전문 상담 연결** — 결과에 따른 상담사 매칭
  4. **누적 200만+ 참여** — 검증된 신뢰도

### 3.7 FAQ (8개 + FAQ schema)

- **H2**: "무료 심리 검사 FAQ"
- 아코디언 UI (첫 항목 기본 열림)
- **JSON-LD** `FAQPage` 스키마 동시 삽입
- **Q&A 8개**:
  1. 심리검사는 무료로 할 수 있나요?
     - **수정됨**: "트로스트의 심리검사는 모두 무료로 제공되며, 언제든지 부담 없이 바로 시작할 수 있습니다."
  2. 검사 결과는 얼마나 정확한가요?
  3. 검사 시간은 얼마나 걸리나요?
  4. 결과를 다른 사람이 볼 수 있나요?
  5. 여러 번 검사해도 되나요?
  6. 검사 결과가 이상하면 어떻게 해야 하나요?
  7. MBTI와 심리검사는 다른가요?
  8. 상담은 어떻게 연결되나요?
- **제거**: 기존 Q "우울증 검사와 불안 검사는 어떻게 다른가요?" (불안 검사 없음)
