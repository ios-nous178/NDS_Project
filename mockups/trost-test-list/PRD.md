# [TROST-TEST-LIST] 트로스트 심리검사 목록 페이지 개선

## 화면 목표

트로스트 심리검사 목록 페이지(`/test`)의 SEO 성능 강화 및 사용자 경험 개선.
기존 "서비스 메뉴" 성격의 페이지를 **정보 제공 + 검사 안내 허브**로 전환하여
"무료 심리검사", "자가진단", "우울증 검사" 등 broad 키워드에서 검색 유입을 확보한다.

## 대상 플랫폼

- 캐시워크앱
- 별도앱
- PC웹
- 모바일웹
- **반응형 필수** (PC ↔ 모바일 레이아웃 분기)

## 페이지 구조 (섹션별)

### 섹션 1: 헤더 / Hero 영역

- **변경 유형**: CHANGED
- **설명**: H1 키워드 재구성으로 SEO 강화. 감정별 빠른 진입 버튼 제공.
- **구성 요소**:
  - H1 타이틀: "무료 심리검사 종류" + "자가진단" 키워드 포함
  - 서브카피: "우울증", "불안장애", "자존감" 등 검색 빈도 높은 키워드
  - CTA 버튼: [3분만에 내 상태 확인하기] → 섹션 3으로 스크롤
  - 감정별 버튼 그룹: 우울 / 스트레스 / 자존감 / 모르겠다면 종합검사
- **인터랙션**:
  - [3분만에 내 상태 확인하기] → 섹션 3(심리검사 목록)으로 smooth scroll
  - 감정별 버튼 → 각 검사 시작 페이지로 랜딩
    - 우울 → 우울 유형 검사 시작
    - 스트레스 → 직무 스트레스 검사 시작
    - 자존감 → 자존감 검사 시작
    - 종합 검사 → 종합 심리 검사 시작
- **반응형 차이**: PC/모바일 동일 구조, 버튼 크기만 조정
- **변경 이유**: 기존 [검사 전체 보기] 버튼 제거 (CTA와 기능 중복), 불안→스트레스 변경 (불안 검사 없음)

### 섹션 2: 정보형 허브 블록

- **변경 유형**: NEW
- **설명**: Hero 직후 800~1,000자 에디토리얼 콘텐츠. 구글이 페이지 상단에서 "정보 제공 의도"를 감지하도록 함.
- **구성 요소**:
  - 에디토리얼 텍스트 블록
  - 심리검사 유형 분류 소개: 증상형 / 자기 이해형 / 상황형 / 종합형
- **인터랙션**: 없음 (읽기 전용)
- **반응형 차이**: PC/모바일 동일, 텍스트 리플로우만
- **참고**: 여기서 제시한 4가지 유형은 섹션 3의 검사 카드에 키워드로 매핑됨

### 섹션 3: 무료로 할 수 있는 온라인 심리검사 (검사 목록)

- **변경 유형**: CHANGED
- **설명**: 검사 카드 목록을 페이지 상단으로 이동 (기존: 라우팅 박스 뒤). SEO 가중치 확보.
- **구성 요소**:
  - 섹션 타이틀 H2
  - 검사 카드 리스트 (순위 정렬)
  - 각 카드: 검사명 / 유형 키워드(Badge) / 간단 설명 / 참여인원 / 소요시간 / 순위 표시
- **검사 목록 (10개)**:

  | 순서 | 검사명             | 유형 키워드 |
  | ---- | ------------------ | ----------- |
  | 1    | 종합 심리검사      | 종합형      |
  | 2    | 종합 성격검사      | 종합형      |
  | 3    | 우울증 검사        | 증상형      |
  | 4    | 자존감 검사        | 자기 이해형 |
  | 5    | MBTI 우울증 검사   | 자기 이해형 |
  | 6    | 직무 스트레스 검사 | 상황형      |
  | 7    | 취업 스트레스 검사 | 상황형      |
  | 8    | 코로나 우울증 검사 | 증상형      |
  | 9    | 우울 유형 검사     | 증상형      |
  | 10   | 공황 증상 자가점검 | 증상형      |

- **인터랙션**:
  - 카드 클릭 → 해당 검사 시작 페이지로 이동
  - 순위 기준: 최근 7일 검사 횟수 많은 순 정렬
- **반응형 차이**:
  - PC: 카드 그리드 (2~3열)
  - 모바일: 1열 리스트, 폴딩 시에도 유형 키워드 노출

### 섹션 4: 지금 내 감정/상황에 맞는 심리검사 추천 (라우팅)

- **변경 유형**: CHANGED
- **설명**: 기존 다크 배경 박스를 라이트 카드 그리드로 교체. 롱테일 H2 적용.
- **구성 요소**:
  - H2: "지금 내 감정/상황에 맞는 심리검사 추천"
  - 감정/상황 카드: 우울 / 자존감 / 성격·기질 / 스트레스
  - 각 카드: 아이콘 + 레이블 + 간단 설명
- **인터랙션**:
  - 우울 → 우울 유형 검사 시작
  - 자존감 → 자존감 검사 시작
  - 성격·기질 → 종합 성격 검사 시작
  - 스트레스 → 직무 스트레스 검사 시작
- **반응형 차이**:
  - PC: 4열 카드 그리드
  - 모바일: 2열 그리드, 홀수일 경우 마지막 카드 full-width

### 섹션 5: 심리검사 선택 가이드

- **변경 유형**: NEW
- **설명**: 기존 2x2 박스를 long-form 가이드로 확장. SEO 롱테일 콘텐츠.
- **구성 요소**:
  - H2: "심리검사 선택 가이드"
  - 가이드 텍스트 블록 (증상 기준, 상황 기준 등)
  - 관련 검사 추천 링크
- **인터랙션**: 텍스트 내 검사 링크 클릭 → 해당 검사 시작 페이지
- **반응형 차이**:
  - PC: "증상 기준으로", "상황 기준으로" 진입 카드 노출
  - 모바일: 진입 카드 미노출, 텍스트만

### 섹션 6: 트로스트가 다른 이유 (차별점)

- **변경 유형**: NEW
- **설명**: 기존 가이드 카드 안에 묻혀있던 차별점을 독립 섹션으로 분리. E-E-A-T 시그널 강화.
- **구성 요소**:
  - H2: "무료 온라인 심리 검사, 트로스트가 다른 이유"
  - 차별점 카드/리스트 (전문 심리검사 도구, 결과 해석, 후속 상담 연계 등)
- **인터랙션**: 없음 (읽기 전용)
- **반응형 차이**: PC/모바일 동일

### 섹션 7: 무료 심리 검사 FAQ

- **변경 유형**: CHANGED
- **설명**: 4개 → 8개로 확장. FAQ Schema(application/ld+json) 추가로 구글 리치 결과 노출.
- **구성 요소**:
  - H2: "무료 심리 검사 FAQ"
  - 아코디언 Q&A 리스트
  - FAQ 항목 예시:
    - Q: 심리검사는 무료로 할 수 있나요?
    - A: 트로스트의 심리검사는 모두 무료로 제공되며, 언제든지 부담 없이 바로 시작할 수 있습니다.
- **인터랙션**: 아코디언 토글 (클릭 시 답변 펼침/접기)
- **반응형 차이**: PC/모바일 동일
- **변경 사항**: 불안 검사 관련 FAQ 제거 (서비스에 없음), 유료 언급 문구 변경

### 섹션 8: 이런 검사도 찾고 있으신가요? (관련 검색 태그)

- **변경 유형**: CHANGED
- **설명**: SEO 관련 검색어를 태그 칩으로 노출.
- **구성 요소**:
  - H2: "이런 검사도 찾고 있으신가요?"
  - 태그 칩 리스트

  | 태그칩명                 | 연결 검사          |
  | ------------------------ | ------------------ |
  | 우울증 자가진단 테스트   | 우울증 검사        |
  | 공황장애 자가점검        | 공황 증상 자가점검 |
  | 자존감 테스트 무료       | 자존감 검사        |
  | 직장인 번아웃 검사       | 직무 스트레스 검사 |
  | 취준생 스트레스 자가진단 | 취업 스트레스 검사 |
  | 온라인 심리검사 무료     | 종합 심리 검사     |
  | 성격유형 검사            | 종합 성격 검사     |
  | 우울감 체크리스트        | 우울 유형 검사     |
  | 정신건강 자가진단        | 종합 심리 검사     |

- **인터랙션**: 칩 클릭 → 해당 검사 시작 페이지 이동
- **반응형 차이**: 칩 wrap 처리

### 섹션 9: 검사 후에는 다음 행동으로 + 푸터

- **변경 유형**: AS-IS (일부 수정)
- **설명**: 검사 후 다음 행동 안내 카드 + 사이트 푸터.
- **구성 요소**:
  - 행동 안내 카드 3개:
    - [전문 상담사 찾기] → 상담사 찾기 화면
    - [커뮤니티 보러가기] → 커뮤니티 화면
    - [콘텐츠 보러가기] → 명언/성경 목록 화면
  - 사이트 푸터
- **인터랙션**: 각 카드 클릭 → 해당 페이지 이동
- **반응형 차이**: PC 3열 / 모바일 1열

### 섹션 10: 하단 고정 배너

- **변경 유형**: NEW
- **설명**: 스크롤해도 항상 하단에 고정 노출되는 CTA 배너.
- **구성 요소**:
  - 고정 배너 바
  - [무료 심리검사 시작하기] 버튼
- **인터랙션**: 버튼 클릭 → 종합 심리검사 시작 페이지 랜딩
- **반응형 차이**: PC/모바일 동일 (하단 고정)

## 데이터 요구사항

- 심리검사 목록 (10개) — 검사명, 설명, 소요시간, 문항수, 누적 참여인원
- 최근 7일 검사별 참여 횟수 (순위 산정용)
- FAQ 콘텐츠 (8개 Q&A)
- 정보형 허브 에디토리얼 텍스트
- 심리검사 선택 가이드 텍스트

## 제약사항

- 현재 서비스에 없는 검사는 노출하지 않음 (불안 검사, 수면 검사 등 제거 완료)
- 모든 검사가 무료 — "무료" 키워드를 개별 검사에 붙이지 않음
- FAQ Schema(application/ld+json) 별도 구현 필요
- SEO meta tag 별도 작업 필요

## 미정/추가 확인 필요

- 순위 산정 로직 상세 (최근 7일 기준이지만, 동률 처리 등)
- 정보형 허브 블록 실제 텍스트 원고
- FAQ 8개 항목 최종 확정 (현재 2개만 확정)
- 심리검사 선택 가이드 텍스트 원고
- 트로스트 차별점 섹션 구체적 콘텐츠
- 하단 고정 배너 디자인 상세 (높이, 배경색 등)
